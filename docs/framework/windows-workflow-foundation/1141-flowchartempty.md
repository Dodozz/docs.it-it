# <a name="1141---flowchartempty"></a>1141 - FlowchartEmpty
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|Id|1141|  
|Parole chiave|WFActivities|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Indica che un diagramma di flusso è stato eseguito senza nodi.  
  
## <a name="message"></a>Messaggio  
 Diagramma di flusso '%1' eseguito senza nodi.  
  
## <a name="details"></a>Dettagli  
  
|Nome elemento dati|Tipo elemento dati|Descrizione|  
|--------------------|--------------------|-----------------|  
|Diagramma di flusso|xs:string|Nome visualizzato del diagramma di flusso.|  
|AppDomain|xs:string|Stringa restituita da AppDomain.CurrentDomain.FriendlyName.|
