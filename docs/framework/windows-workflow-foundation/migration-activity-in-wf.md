# <a name="migration-activity-in-wf"></a>Attività Migration in WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] fornisce il <xref:System.Activities.Statements.Interop> attività per l'esecuzione di attività che derivano dall'impegno all'interno di un flusso di lavoro basato su <xref:System.Activities.Activity>. Per altre informazioni, vedere la [materiale sussidiario di migrazione](migration-guidance.md) sezione.  
  
> [!NOTE]
>  Il <xref:System.Activities.Statements.Interop> attività non viene visualizzato nella casella degli strumenti della finestra di progettazione del flusso di lavoro a meno che non dispone di progetto del flusso di lavoro relativi **Framework di destinazione** impostazione **.Net Framework 4** o versione successiva.
