-   <span data-ttu-id="9ab35-145">In uno scenario a nodo singolo, l'istanza in questione è stata interrotta a un certo punto e viene creata una nuova istanza del provider di persistenza con un ID host differente.</span><span class="sxs-lookup"><span data-stu-id="9ab35-145">In a single-node scenario and the instance in question was aborted at some point and a new persistence provider instance is created which has a different host ID.</span></span>
