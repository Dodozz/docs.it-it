# <a name="39460---trackingvaluenotserializable"></a>39460 - TrackingValueNotSerializable
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|Id|39460|  
|Parole chiave|WFTracking|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Indica che la variabile o l'argomento estratto in un record di rilevamento non è serializzabile.  
  
## <a name="message"></a>Messaggio  
 La variabile o l'argomento estratto '%1' non è serializzabile.
