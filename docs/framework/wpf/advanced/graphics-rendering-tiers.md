## <a name="graphics-hardware"></a><span data-ttu-id="034e5-104">Hardware grafico</span><span class="sxs-lookup"><span data-stu-id="034e5-104">Graphics Hardware</span></span>

-   <span data-ttu-id="034e5-106">**RAM video** La quantità di memoria video sull'hardware grafico determina le dimensioni e il numero di buffer che possono essere usati per la composizione della grafica.</span><span class="sxs-lookup"><span data-stu-id="034e5-106">**Video RAM** The amount of video memory on the graphics hardware determines the size and number of buffers that can be used for compositing graphics.</span></span>
