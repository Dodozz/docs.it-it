# <a name="flowdocumentscrollviewer"></a>FlowDocumentScrollViewer
Il <xref:System.Windows.Controls.FlowDocumentScrollViewer> controllo consente di visualizzare <xref:System.Windows.Documents.FlowDocument> contenuto in un contenitore scorrevole. Si differenzia <xref:System.Windows.Controls.FlowDocumentPageViewer>, che visualizza il contenuto in base a una per ogni pagina.
