# <a name="wrappanel"></a>WrapPanel
Il <xref:System.Windows.Controls.WrapPanel> elemento posiziona gli elementi figlio in sequenza da sinistra a destra, interrompendo il contenuto nella riga successiva sul bordo della casella che contiene.
