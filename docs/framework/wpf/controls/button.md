# <a name="button"></a>Button
Oggetto <xref:System.Windows.Controls.Button> controllo risponde all'input dell'utente da un mouse, tastiera, dello stilo, o altro dispositivo di input e genera un <xref:System.Windows.Controls.Primitives.ButtonBase.Click> evento. Oggetto <xref:System.Windows.Controls.Button> è una semplice [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] componente che può contenere contenuto semplice, ad esempio testo e può anche contenere contenuto complesso, ad esempio immagini e <xref:System.Windows.Controls.Panel> controlli.
