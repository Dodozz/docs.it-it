# <a name="visual-layer-programming"></a>Programmazione a livello visivo
Il <xref:System.Windows.Media.Visual> oggetto è dei principali [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] oggetto, il cui ruolo principale consiste nel fornire supporto per il rendering. Controlli dell'interfaccia utente, ad esempio <xref:System.Windows.Controls.Button> e <xref:System.Windows.Controls.TextBox>, derivano dal <xref:System.Windows.Media.Visual> e come utilizzarla per rendere persistenti i dati di rendering.
