2.  <span data-ttu-id="4ccc4-110">Alla fine del quarto secondo, viene utilizzata un'istanza di <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> classe per spostare rapidamente il rettangolo alla posizione successiva.</span><span class="sxs-lookup"><span data-stu-id="4ccc4-110">At the end of the fourth second, uses an instance of the <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> class to suddenly move the rectangle to the next position.</span></span> <span data-ttu-id="4ccc4-111">Fotogrammi chiave discreti come <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> creano salti improvvisi tra valori.</span><span class="sxs-lookup"><span data-stu-id="4ccc4-111">Discrete key frames like <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> create sudden jumps between values.</span></span> <span data-ttu-id="4ccc4-112">In questo esempio, il rettangolo si trova in corrispondenza della posizione iniziale e improvvisamente appare nella posizione 500.</span><span class="sxs-lookup"><span data-stu-id="4ccc4-112">In this example, the rectangle is at the starting position and then suddenly appears at the 500 position.</span></span>  
  
3.  <span data-ttu-id="4ccc4-113">Nei due secondi finali, viene utilizzata un'istanza di <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame> classe per riportare il rettangolo alla posizione iniziale.</span><span class="sxs-lookup"><span data-stu-id="4ccc4-113">In the final two seconds, uses an instance of the <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame> class to move the rectangle back to its starting position.</span></span> <span data-ttu-id="4ccc4-114">Ad esempio i fotogrammi chiave spline <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame> creano una transizione variabile tra i valori a seconda del valore del <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame.KeySpline%2A> proprietà.</span><span class="sxs-lookup"><span data-stu-id="4ccc4-114">Spline key frames like <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame> create a variable transition between values according to the value of the <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame.KeySpline%2A> property.</span></span> <span data-ttu-id="4ccc4-115">In questo esempio, il rettangolo inizia a spostarsi lentamente e quindi accelera in modo esponenziale verso la fine del segmento temporale.</span><span class="sxs-lookup"><span data-stu-id="4ccc4-115">In this example, the rectangle begins by moving slowly and then speeds up exponentially toward the end of the time segment.</span></span>
