     <span data-ttu-id="1d8d9-108">WCF Data Services supporta inoltre i modelli di dati basati su oggetti common language runtime (CLR) che restituiscono un'istanza di <xref:System.Linq.IQueryable%601> interfaccia.</span><span class="sxs-lookup"><span data-stu-id="1d8d9-108">WCF Data Services also supports data models that are based on common language runtime (CLR) objects that return an instance of the <xref:System.Linq.IQueryable%601> interface.</span></span> <span data-ttu-id="1d8d9-109">In questo modo è possibile distribuire servizi dati basati su elenchi, matrici e raccolte di .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="1d8d9-109">This enables you to deploy data services that are based on lists, arrays, and collections in the .NET Framework.</span></span> <span data-ttu-id="1d8d9-110">Per consentire l'esecuzione di operazioni di creazione, aggiornamento ed eliminazione su queste strutture di dati, è inoltre necessario implementare l'interfaccia <xref:System.Data.Services.IUpdatable>.</span><span class="sxs-lookup"><span data-stu-id="1d8d9-110">To enable create, update, and delete operations over these data structures, you must also implement the <xref:System.Data.Services.IUpdatable> interface.</span></span> <span data-ttu-id="1d8d9-111">Per altre informazioni, vedere [Procedura: Creare un servizio dati tramite il Provider di Reflection](../../../../docs/framework/data/wcf/create-a-data-service-using-rp-wcf-data-services.md).</span><span class="sxs-lookup"><span data-stu-id="1d8d9-111">For more information, see [How to: Create a Data Service Using the Reflection Provider](../../../../docs/framework/data/wcf/create-a-data-service-using-rp-wcf-data-services.md).</span></span>
