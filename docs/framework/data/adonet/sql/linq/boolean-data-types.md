# <a name="boolean-data-types"></a>Tipi di dati Boolean
Gli operatori booleani funzionano correttamente in Common Language Runtime (CLR), con l'eccezione del comportamento noto come short-circuit che non viene convertito. Ad esempio, l'operatore `AndAlso` Visual Basic si comporta come l'operatore `And`. L'operatore `&&` C# si comporta come l'operatore `&`.  
  
 In [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] sono supportati gli operatori riportati di seguito.  
  
|Visual Basic|C#|  
|------------------|---------|  
|[Operatore And](~/docs/visual-basic/language-reference/operators/and-operator.md)|[Operatore &](~/docs/csharp/language-reference/operators/and-operator.md)|  
|[Operatore AndAlso](~/docs/visual-basic/language-reference/operators/andalso-operator.md)|[Operatore &&](~/docs/csharp/language-reference/operators/conditional-and-operator.md)|  
|[Operatore Or](~/docs/visual-basic/language-reference/operators/or-operator.md)|[&#124;Operatore](~/docs/csharp/language-reference/operators/or-operator.md)|  
|[Operatore OrElse](~/docs/visual-basic/language-reference/operators/orelse-operator.md)|[&#124;&#124; Operator](~/docs/csharp/language-reference/operators/conditional-or-operator.md)|  
|[Operatore Xor](~/docs/visual-basic/language-reference/operators/xor-operator.md)|[Operatore ^](~/docs/csharp/language-reference/operators/xor-operator.md)|  
|[Operatore Not](~/docs/visual-basic/language-reference/operators/not-operator.md)|[Operatore \!](~/docs/csharp/language-reference/operators/logical-negation-operator.md)|
