# <a name="boolean-data-types"></a>Tipi di dati Boolean
Gli operatori booleani funzionano correttamente in Common Language Runtime (CLR), con l'eccezione del comportamento noto come short-circuit che non viene convertito. Ad esempio, l'operatore `AndAlso` Visual Basic si comporta come l'operatore `And`. L'operatore `&&` C# si comporta come l'operatore `&`.  
  
 [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] supporta gli operatori seguenti.  
  
|Visual Basic|C#|  
|------------------|---------|  
|[Operatore And](~/docs/visual-basic/language-reference/operators/and-operator.md)|[Operatore &](~/docs/csharp/language-reference/operators/boolean-logical-operators.md#logical-and-operator-)|  
|[Operatore AndAlso](~/docs/visual-basic/language-reference/operators/andalso-operator.md)|[Operatore &&](~/docs/csharp/language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-)|  
|[Operatore Or](~/docs/visual-basic/language-reference/operators/or-operator.md)|[&#124;Operatore](~/docs/csharp/language-reference/operators/boolean-logical-operators.md#logical-or-operator-)|  
|[Operatore OrElse](~/docs/visual-basic/language-reference/operators/orelse-operator.md)|[&#124;&#124; Operator](~/docs/csharp/language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-)|  
|[Operatore Xor](~/docs/visual-basic/language-reference/operators/xor-operator.md)|[Operatore ^](~/docs/csharp/language-reference/operators/boolean-logical-operators.md#logical-exclusive-or-operator-)|  
|[Operatore Not](~/docs/visual-basic/language-reference/operators/not-operator.md)|[\! Operatore](~/docs/csharp/language-reference/operators/boolean-logical-operators.md#logical-negation-operator-)|
