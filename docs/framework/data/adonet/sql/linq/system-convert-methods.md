- Versioni con un parametro <xref:System.IFormatProvider>.

- Metodi che interessano matrici di caratteri o matrici di byte:

  - <xref:System.Convert.FromBase64CharArray%2A>

  - <xref:System.Convert.ToBase64CharArray%2A>
