## <a name="user-instances"></a><span data-ttu-id="fd9d2-114">Istanze utente</span><span class="sxs-lookup"><span data-stu-id="fd9d2-114">User Instances</span></span>

 <span data-ttu-id="fd9d2-118">Un'istanza utente viene generata da un'istanza primaria di SQL Server o SQL Server Express per conto di un utente.</span><span class="sxs-lookup"><span data-stu-id="fd9d2-118">A user instance is generated from a primary instance of SQL Server or SQL Server Express on behalf of a user.</span></span> <span data-ttu-id="fd9d2-119">Viene eseguita come processo utente nel contesto di sicurezza Windows dell'utente e non come servizio.</span><span class="sxs-lookup"><span data-stu-id="fd9d2-119">It runs as a user process under the Windows security context of the user, not as a service.</span></span> <span data-ttu-id="fd9d2-120">Gli account di accesso di SQL Server non sono consentiti. Sono infatti supportati solo quelli di Windows.</span><span class="sxs-lookup"><span data-stu-id="fd9d2-120">SQL Server logins are disallowed; only Windows logins are supported.</span></span> <span data-ttu-id="fd9d2-121">In tal modo si impedisce a software in esecuzione in un'istanza utente di apportare modifiche a livello di sistema per le quali l'utente non dispone delle autorizzazioni.</span><span class="sxs-lookup"><span data-stu-id="fd9d2-121">This prevents software executing on a user instance from making system-wide changes that the user would not have permissions to make.</span></span> <span data-ttu-id="fd9d2-122">A un'istanza utente, nota anche come istanza figlio o client, viene talvolta fatto riferimento usando l'acronimo RAN (Run As Normal User).</span><span class="sxs-lookup"><span data-stu-id="fd9d2-122">A user instance is also known as a child or client instance, and is sometimes referred to by using the RANU acronym ("run as normal user").</span></span>
