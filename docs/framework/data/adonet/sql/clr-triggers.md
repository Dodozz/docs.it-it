# <a name="clr-triggers"></a>Trigger CLR
Un trigger è un tipo particolare di stored procedure che viene avviato automaticamente quando si esegue un evento relativo al linguaggio. L'integrazione di Microsoft SQL Server con CLR (Common Language Runtime) di .NET Framework consente di usare qualsiasi linguaggio di .NET Framework per creare trigger CLR.  
  
 Per informazioni più dettagliate, vedere la [documentazione di SQL Server](/sql) per la versione di SQL Server in uso.
