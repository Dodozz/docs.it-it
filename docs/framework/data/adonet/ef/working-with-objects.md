# <a name="working-with-objects"></a>Utilizzo di oggetti
[!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] consente di eseguire query sui dati e di inserire, aggiornare ed eliminare dati, espressi come oggetti CLR (Common Language Runtime) tipizzati che sono istanze dei tipi di entità. I tipi di entità rappresentano le entità definite nel modello concettuale. [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] esegue il mapping di entità e relazioni definite in un modello concettuale a un'origine dati. Il [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] offre funzionalità per eseguire le operazioni seguenti: materializzare i dati restituiti dall'origine dati come oggetti; rilevare le modifiche apportate agli oggetti; gestire la concorrenza, propagare le modifiche di oggetti nuovamente all'origine dati; e associare oggetti ai controlli.  
  
 Per altre informazioni sull'utilizzo di oggetti nella versione più recente di Entity Framework, vedere, [utilizzo di oggetti](https://go.microsoft.com/fwlink/?LinkId=235289).
