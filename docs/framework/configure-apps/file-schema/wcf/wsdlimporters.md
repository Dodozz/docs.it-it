# <a name="wsdlimporters"></a>\<wsdlImporters>
Questo elemento di configurazione specifica tutte le unità di importazione WSDL che importano metadati WSDL 1.1 con allegati di WS-Policy. Ogni elemento figlio è un <`wsdlImporter`> che specifica il modo per importare i metadati, nonché di convertire tali informazioni in diverse classi che rappresentano informazioni di contratto e dell'endpoint. Può importare selettivamente informazioni di contratto e di endpoint e proprietà che espongono qualsiasi errore di importazione e accettano informazioni sul tipo relative al processo di importazione e di conversione. Supporta inoltre l'importazione di informazioni dell'associazione e proprietà che forniscono accesso a qualsiasi documento di criteri, documento WSDL, estensione WSDL e documento di XML Schema.
