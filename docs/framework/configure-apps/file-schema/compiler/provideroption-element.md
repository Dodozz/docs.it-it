## <a name="example"></a><span data-ttu-id="c56f5-143">Esempio</span><span class="sxs-lookup"><span data-stu-id="c56f5-143">Example</span></span>  
 <span data-ttu-id="c56f5-144">Nell'esempio seguente viene illustrato come specificare la versione 3.5 del provider di codice c# da usare.</span><span class="sxs-lookup"><span data-stu-id="c56f5-144">The following example demonstrates how to specify that version 3.5 of the C# code provider should be used.</span></span>  
  
```xml  
<configuration>  
  <system.codedom>  
    <compilers>  
      <!-- zero or more compiler elements -->  
      <compiler  
        language="c#;cs;csharp"  
        extension=".cs"  
        type="Microsoft.CSharp.CSharpCodeProvider, System,   
          Version=2.0.3600.0, Culture=neutral,   
          PublicKeyToken=b77a5c561934e089"  
        compilerOptions="/optimize"  
        warningLevel="1" >  
          <providerOption  
            name="CompilerVersion"  
            value="v3.5" />  
      </compiler>  
    </compilers>  
  </system.codedom>  
</configuration>  
```
