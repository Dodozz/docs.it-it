## <a name="to-create-a-key-pair"></a><span data-ttu-id="83f08-110">Per creare una coppia di chiavi</span><span class="sxs-lookup"><span data-stu-id="83f08-110">To create a key pair</span></span>

1.  <span data-ttu-id="83f08-111">Al prompt dei comandi digitare il seguente comando:</span><span class="sxs-lookup"><span data-stu-id="83f08-111">At the command prompt, type the following command:</span></span>

     <span data-ttu-id="83f08-112">**sn –k** \<*nome file*></span><span class="sxs-lookup"><span data-stu-id="83f08-112">**sn –k** \<*file name*></span></span>
