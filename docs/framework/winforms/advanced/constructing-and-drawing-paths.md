# <a name="constructing-and-drawing-paths"></a>Costruzione e creazione di percorsi
Un percorso è una sequenza di primitive grafiche (linee, rettangoli, curve, testo e così via) che può essere modificato e disegnato come una singola unità. Un percorso può essere suddiviso *figure* che sono aperti o chiusi. Una figura può contenere diverse primitive.  
  
 È possibile disegnare un tracciato chiamando il <xref:System.Drawing.Graphics.DrawPath%2A> metodo del <xref:System.Drawing.Graphics> classe, è possibile inserire un percorso tramite una chiamata il <xref:System.Drawing.Graphics.FillPath%2A> metodo del <xref:System.Drawing.Graphics> classe.
