# <a name="using-regions"></a>Utilizzo delle regioni
Il [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Region> classe consente di definire una forma personalizzata. La forma può essere composto da poligoni, linee e curve.  
  
 Due utilizzi comuni per le aree sono hit testing e di ritaglio. Hit test consiste nel determinare se il mouse è stato fatto clic in una determinata area dello schermo. Ritaglio consiste nel limito a una determinata area di disegno.

 Viene illustrato come utilizzare un <xref:System.Drawing.Region> per eseguire un hit test.

 Viene spiegato come impostare l'area di ritaglio per un <xref:System.Drawing.Graphics> oggetto.
