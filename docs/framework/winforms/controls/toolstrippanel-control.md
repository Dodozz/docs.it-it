# <a name="toolstrippanel-control"></a>Controllo ToolStripPanel
`ToolStripPanel` controllo consente di condividere lo spazio orizzontale o verticale nell'area strumenti, quando è ancorato e di disporre <xref:System.Windows.Forms.ToolStrip> controlla quando non è necessario i quattro pannelli e un pannello centrale di un <xref:System.Windows.Forms.ToolStripContainer>.  
  
 Gli argomenti di questa sezione descrivono i concetti e le tecniche che è possibile usare per compilare funzionalità `ToolStripPanel` nelle applicazioni.
