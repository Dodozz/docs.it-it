# <a name="how-to-resize-controls-on-windows-forms"></a><span data-ttu-id="a2bbd-102">Procedura: Ridimensionare i controlli Windows Form</span><span class="sxs-lookup"><span data-stu-id="a2bbd-102">How to: Resize Controls on Windows Forms</span></span>
<span data-ttu-id="a2bbd-103">È possibile ridimensionare i singoli controlli ed è possibile ridimensionare più controlli di tipo uguale o diverso, ad esempio <xref:System.Windows.Forms.Button> e <xref:System.Windows.Forms.GroupBox> controlli.</span><span class="sxs-lookup"><span data-stu-id="a2bbd-103">You can resize individual controls, and you can resize multiple controls of the same or different kind, such as <xref:System.Windows.Forms.Button> and <xref:System.Windows.Forms.GroupBox> controls.</span></span>  
  
### <a name="to-resize-a-control"></a><span data-ttu-id="a2bbd-104">Ridimensionare un controllo</span><span class="sxs-lookup"><span data-stu-id="a2bbd-104">To resize a control</span></span>  
  
1.  <span data-ttu-id="a2bbd-105">Fare clic sul controllo da ridimensionare e trascinare uno dei quadratini di otto ridimensionamento.</span><span class="sxs-lookup"><span data-stu-id="a2bbd-105">Click the control to be resized and drag one of the eight sizing handles.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="a2bbd-106">Selezionare il controllo e premere i tasti di direzione e tenendo premuto il tasto MAIUSC per ridimensionare il controllo di un pixel in una fase.</span><span class="sxs-lookup"><span data-stu-id="a2bbd-106">Select the control and press the ARROW keys while holding down the SHIFT key to resize the control one pixel at a time.</span></span> <span data-ttu-id="a2bbd-107">Premere i tasti di direzione a destra o verso il basso mentre vengono premuti i tasti MAIUSC + CTRL per ridimensionare il controllo in incrementi di grandi dimensioni.</span><span class="sxs-lookup"><span data-stu-id="a2bbd-107">Press the DOWN or RIGHT arrow keys while holding down the SHIFT and CTRL keys to resize the control in large increments.</span></span>  
  
### <a name="to-resize-multiple-controls-on-a-form"></a><span data-ttu-id="a2bbd-108">Per ridimensionare più controlli in un form</span><span class="sxs-lookup"><span data-stu-id="a2bbd-108">To resize multiple controls on a form</span></span>  
  
1.  <span data-ttu-id="a2bbd-109">Tenere premuto il tasto CTRL o MAIUSC e selezionare i controlli che si intende ridimensionare.</span><span class="sxs-lookup"><span data-stu-id="a2bbd-109">Hold down the CTRL or SHIFT key and select the controls you want to resize.</span></span> <span data-ttu-id="a2bbd-110">Viene utilizzata la dimensione del primo controllo che scelto per gli altri controlli.</span><span class="sxs-lookup"><span data-stu-id="a2bbd-110">The size of the first control you select is used for the other controls.</span></span>  
  
2.  <span data-ttu-id="a2bbd-111">Nel **formato** menu, scegliere **Rendi uguali**, selezionare una delle quattro opzioni.</span><span class="sxs-lookup"><span data-stu-id="a2bbd-111">On the **Format** menu, choose **Make Same Size**, and select one of the four options.</span></span> <span data-ttu-id="a2bbd-112">I primi tre comandi modificano le dimensioni dei controlli in modo che corrisponda il primo controllo selezionato.</span><span class="sxs-lookup"><span data-stu-id="a2bbd-112">The first three commands change the dimensions of the controls to match the first-selected control.</span></span>
