# <a name="data-entry-in-the-windows-forms-datagridview-control"></a>Immissione di dati nel controllo DataGridView Windows Form
Il `DataGridView` controllo offre diverse funzionalità che consentono modificare la modalità di aggiungere o modificare i dati nel controllo. Ad esempio, è possibile rendere immissione di dati più efficiente fornendo i valori predefiniti per le nuove righe e dagli avvisi degli utenti quando si verificano errori.
