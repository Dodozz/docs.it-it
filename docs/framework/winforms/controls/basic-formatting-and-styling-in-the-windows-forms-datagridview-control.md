# <a name="basic-formatting-and-styling-in-the-windows-forms-datagridview-control"></a>Formattazione e stile di base nel controllo DataGridView Windows Form
Il `DataGridView` controllo rende più semplice definire l'aspetto di base di celle e la formattazione dei valori delle celle. È possibile definire l'aspetto e impostando le proprietà di formattazione stili per le singole celle, per le celle delle righe e colonne specifiche o per tutte le celle nel controllo di `DataGridViewCellStyle` oggetti accessibili tramite diversi `DataGridView` proprietà del controllo. Inoltre, è possibile modificare questi stili dinamicamente in base ai fattori quali il valore di cella tramite la gestione di `CellFormatting` evento.
