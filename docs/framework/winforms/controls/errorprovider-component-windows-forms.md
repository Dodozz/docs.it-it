# <a name="errorprovider-component-windows-forms"></a>Componente ErrorProvider (Windows Form)
I moduli di Windows `ErrorProvider` componente viene utilizzato per indicare all'utente in modo non invadente che qualcosa non vada. Viene in genere usato in combinazione con la convalida dell'input utente in un form o visualizzare gli errori all'interno di un set di dati.

 Spiega che cos'è il componente le funzionalità principali e i relativi delle proprietà.  
  
 [Procedura: Visualizzare le icone di errore per la convalida dei Form con il componente ErrorProvider di Windows Form](display-error-icons-for-form-validation-with-wf-errorprovider.md)  
 Vengono fornite indicazioni per la convalida dell'input dell'utente con un componente di provider di errore.  
  
 [Procedura: Visualizzare gli errori all'interno di un set di dati con il componente ErrorProvider di Windows Form](view-errors-within-a-dataset-with-wf-errorprovider-component.md)  
 Vengono fornite indicazioni per l'uso di un componente di provider di errore per visualizzare gli errori di dati.
