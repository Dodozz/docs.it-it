# <a name="statusstrip-control"></a>Controllo StatusStrip
Il controllo `StatusStrip` di Windows Form viene usato nei form come un'area solitamente rappresentata nella parte inferiore di una finestra, in cui possono essere visualizzati diversi tipi di informazioni sullo stato di un'applicazione. `StatusStrip` controlli hanno in genere `ToolStripStatusLabel` su di essi i controlli che visualizzano testo o icone per indicare lo stato o una <xref:System.Windows.Forms.ToolStripProgressBar> che graficamente lo stato di completamento di un processo.
