### <a name="to-add-a-table-to-the-datagrid-control-in-the-designer"></a><span data-ttu-id="18c2c-116">Per aggiungere una tabella al controllo DataGrid nella finestra di progettazione</span><span class="sxs-lookup"><span data-stu-id="18c2c-116">To add a table to the DataGrid control in the designer</span></span>  
  
1.  <span data-ttu-id="18c2c-117">Per visualizzare i dati nella tabella, è innanzitutto necessario associare il <xref:System.Windows.Forms.DataGrid> controllo a un set di dati.</span><span class="sxs-lookup"><span data-stu-id="18c2c-117">In order to display data in the table, you must first bind the <xref:System.Windows.Forms.DataGrid> control to a dataset.</span></span> <span data-ttu-id="18c2c-118">Per altre informazioni, vedere [Procedura: Associare il controllo DataGrid di Windows Form a un'origine dati tramite la finestra di progettazione](../../../../docs/framework/winforms/controls/bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span><span class="sxs-lookup"><span data-stu-id="18c2c-118">For more information, see [How to: Bind the Windows Forms DataGrid Control to a Data Source Using the Designer](../../../../docs/framework/winforms/controls/bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span></span>  
  
2.  <span data-ttu-id="18c2c-119">Selezionare il <xref:System.Windows.Forms.DataGrid> del controllo <xref:System.Windows.Forms.DataGrid.TableStyles%2A> proprietà nella finestra Proprietà, quindi fare clic sul pulsante con puntini di sospensione (![schermata di VisualStudioEllipsesButton](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")) accanto a la proprietà da visualizzare il **raccolta DataGridTableStyle**.</span><span class="sxs-lookup"><span data-stu-id="18c2c-119">Select the <xref:System.Windows.Forms.DataGrid> control's <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property in the Properties window, and then click the ellipsis button (![VisualStudioEllipsesButton screenshot](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")) next to the property to display the **DataGridTableStyle Collection Editor**.</span></span>  
  
3.  <span data-ttu-id="18c2c-120">Nell'editor della raccolta, fare clic su **Add** per inserire uno stile di tabella.</span><span class="sxs-lookup"><span data-stu-id="18c2c-120">In the collection editor, click **Add** to insert a table style.</span></span>  
  
4.  <span data-ttu-id="18c2c-121">Fare clic su **OK** per chiudere l'editor della raccolta e quindi riaprirlo facendo clic sul pulsante con puntini di sospensione accanto al <xref:System.Windows.Forms.DataGrid.TableStyles%2A> proprietà.</span><span class="sxs-lookup"><span data-stu-id="18c2c-121">Click **OK** to close the collection editor, and then reopen it by clicking the ellipsis button next to the <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property.</span></span>  
  
     <span data-ttu-id="18c2c-122">Quando si riapre l'editor della raccolta, tutte le tabelle di dati associate al controllo verranno visualizzato nell'elenco a discesa per il <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> proprietà dello stile di tabella.</span><span class="sxs-lookup"><span data-stu-id="18c2c-122">When you reopen the collection editor, any data tables bound to the control will appear in the drop-down list for the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> property of the table style.</span></span>  
  
5.  <span data-ttu-id="18c2c-123">Nel **membri** finestra dell'editor della raccolta, scegliere lo stile di tabella.</span><span class="sxs-lookup"><span data-stu-id="18c2c-123">In the **Members** box of the collection editor, click the table style.</span></span>  
  
6.  <span data-ttu-id="18c2c-124">Nel **delle proprietà** finestra dell'editor della raccolta, selezionare il <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> valore per la tabella che si desidera visualizzare.</span><span class="sxs-lookup"><span data-stu-id="18c2c-124">In the **Properties** box of the collection editor, select the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> value for the table you want to display.</span></span>
