# <a name="toolstripstatuslabel-control-overview"></a>Cenni preliminari sul controllo ToolStripStatusLabel
Il <xref:System.Windows.Forms.ToolStripStatusLabel> è un'etichetta per un <xref:System.Windows.Forms.StatusStrip>. Ad esempio la <xref:System.Windows.Forms.Label> o <xref:System.Windows.Forms.ToolStripLabel>, il <xref:System.Windows.Forms.ToolStripStatusLabel> fornisce un'area di visualizzazione non selezionabile per testo, immagini o entrambi. Il <xref:System.Windows.Forms.ToolStripStatusLabel> è ospitato da un <xref:System.Windows.Forms.StatusStrip>.  
  
### <a name="important-toolstripstatuslabel-members"></a>Membri importanti ToolStripStatusLabel  
  
|nome|Descrizione|  
|----------|-----------------|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.Spring%2A>|Ottiene o imposta un valore che indica se il <xref:System.Windows.Forms.ToolStripStatusLabel> riempie automaticamente lo spazio disponibile sul <xref:System.Windows.Forms.StatusStrip> quando il form viene ridimensionato|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderSides%2A>|Ottiene o imposta un valore che indica quali lati del <xref:System.Windows.Forms.ToolStripStatusLabel> Mostra bordi.|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderStyle%2A>|Ottiene o imposta lo stile del bordo di <xref:System.Windows.Forms.ToolStripStatusLabel>.|
