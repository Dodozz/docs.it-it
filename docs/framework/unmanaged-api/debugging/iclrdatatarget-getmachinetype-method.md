# <a name="iclrdatatargetgetmachinetype-method"></a>Metodo ICLRDataTarget::GetMachineType

## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetMachineType (  
    [out] ULONG32     *machineType  
);  
```
