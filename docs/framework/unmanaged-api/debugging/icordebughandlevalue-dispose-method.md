# <a name="icordebughandlevaluedispose-method"></a>Metodo ICorDebugHandleValue::Dispose
Rilascia l'handle fa riferimento questo oggetto ICorDebugHandleValue senza rilasciare in modo esplicito il puntatore di interfaccia.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT Dispose ();  
```
