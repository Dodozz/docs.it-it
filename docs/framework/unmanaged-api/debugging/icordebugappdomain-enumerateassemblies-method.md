# <a name="icordebugappdomainenumerateassemblies-method"></a>Metodo ICorDebugAppDomain::EnumerateAssemblies

## <a name="syntax"></a>Sintassi  
  
```  
HRESULT EnumerateAssemblies (  
    [out] ICorDebugAssemblyEnum  **ppAssemblies  
);  
```
