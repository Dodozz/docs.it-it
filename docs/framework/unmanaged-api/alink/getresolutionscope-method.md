# <a name="getresolutionscope-method"></a>Metodo GetResolutionScope
Recupera l'ambito di un determinato tipo.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetResolutionScope(  
    mdAssembly  AssemblyID,  
    mdToken     FileToken,  
    mdToken     TargetFile,  
    mdToken*    pScope  
) PURE;  
```  
  
## <a name="parameters"></a>Parametri  
 `AssemblyID`  
 ID dell'assembly.  
  
 `FileToken`  
 File che richiede un riferimento.  
  
 `TargetFile`  
 Token del file di tipo Ã¨ definito, in genere recuperato mediante [metodo ImportFile](../../../../docs/framework/unmanaged-api/alink/importfile-method.md).  
  
 `pScope`  
 Riceve il riferimento all'assembly o modulo.  
  
## <a name="return-value"></a>Valore restituito  
 Restituisce S_OK se il metodo ha esito positivo.  
  
## <a name="requirements"></a>Requisiti  
 Richiede alink.h.
