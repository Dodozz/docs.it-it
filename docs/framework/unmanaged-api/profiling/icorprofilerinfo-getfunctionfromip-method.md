# <a name="icorprofilerinfogetfunctionfromip-method"></a>Metodo ICorProfilerInfo::GetFunctionFromIP
Esegue il mapping di un puntatore all'istruzione di codice gestito per una `FunctionID`.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetFunctionFromIP(  
    [in]  LPCBYTE    ip,  
    [out] FunctionID *pFunctionId);  
```  
  
## <a name="parameters"></a>Parametri  
 `ip`  
 [in] Il puntatore dell'istruzione nel codice gestito.  
  
 `pFunctionId`  
 [out] L'ID restituito di funzione.
