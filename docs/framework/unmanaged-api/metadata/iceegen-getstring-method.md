# <a name="iceegengetstring-method"></a>Metodo ICeeGen::GetString

## <a name="syntax"></a>Sintassi  
  
```  
HRESULT GetString (  
    [in]  ULONG      RVA,   
    [out] LPWSTR     *lpString  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `RVA`  
 [in] L'indirizzo virtuale relativo della stringa da restituire.  
  
 `lpString`
