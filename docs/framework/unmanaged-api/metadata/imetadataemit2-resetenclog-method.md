# <a name="imetadataemit2resetenclog-method"></a>Metodo IMetaDataEmit2::ResetENCLog
Reimposta il log di modifica e continuazione e avvia una nuova sessione.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT ResetENCLog ();  
```  
  
## <a name="parameters"></a>Parametri  
 `ResetENCLog` non accetta parametri.
