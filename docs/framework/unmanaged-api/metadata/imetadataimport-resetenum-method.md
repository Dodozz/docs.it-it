# <a name="imetadataimportresetenum-method"></a>Metodo IMetaDataImport::ResetEnum
Reimposta l'enumeratore specificato nella posizione specificata.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,   
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `hEnum`  
 [in] L'enumeratore reimpostare.  
  
 `ulPos`  
 [in] La nuova posizione in corrispondenza del quale inserire l'enumeratore.
