# <a name="hosting-coclasses"></a>Coclassi di hosting
Questa sezione descrive le coclassi API non gestite usate dall'API di hosting.  
  
## <a name="in-this-section"></a>In questa sezione  
 [Coclasse CLRRuntimeHost](../../../../docs/framework/unmanaged-api/hosting/clrruntimehost-coclass.md)  
 Fornisce interfacce per gestire l'esecuzione di codice dal runtime.  
  
 [Coclasse ComCallUnmarshal](../../../../docs/framework/unmanaged-api/hosting/comcallunmarshal-coclass.md)  
 Fornisce interfacce per gestire il marshalling di puntatori a interfaccia.  
  
 [Coclasse CorRuntimeHost](../../../../docs/framework/unmanaged-api/hosting/corruntimehost-coclass.md)  
 Fornisce interfacce per la gestione delle applicazioni che vengono eseguite da common language runtime.  
  
 [Coclasse TypeNameFactory](../../../../docs/framework/unmanaged-api/hosting/typenamefactory-coclass.md)  
 Fornisce un'interfaccia per gestire la Decostruzione di un nome di tipo.
