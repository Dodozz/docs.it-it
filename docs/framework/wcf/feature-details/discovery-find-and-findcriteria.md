## <a name="findresponse"></a><span data-ttu-id="cb306-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="cb306-140">FindResponse</span></span>  
 <span data-ttu-id="cb306-141"><xref:System.ServiceModel.Discovery.FindResponse> dispone di una proprietà della raccolta <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> che contiene qualsiasi replica inviata da servizi corrispondenti sulla rete.</span><span class="sxs-lookup"><span data-stu-id="cb306-141"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="cb306-142">Se nessun servizio invia una risposta, la raccolta è vuota.</span><span class="sxs-lookup"><span data-stu-id="cb306-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="cb306-143">Se uno o più servizi inviano una risposta, ogni replica viene archiviata in un oggetto <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> che contiene l'indirizzo, il contratto e alcune informazioni aggiuntive sul servizio.</span><span class="sxs-lookup"><span data-stu-id="cb306-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>  
  
 <span data-ttu-id="cb306-144">Nell'esempio seguente viene mostrato come eseguire un'operazione di ricerca nel codice.</span><span class="sxs-lookup"><span data-stu-id="cb306-144">The following example shows how to perform a find operation in code.</span></span>  
  
```  
// Create DiscoveryClient  
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());  
  
// Create FindCriteria  
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));  
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));  
findCriteria.Duration = TimeSpan.FromSeconds(10);   
  
// Find ICalculatorService endpoints              
FindResponse findResponse = discoveryClient.Find(findCriteria);  
  
Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)  
```
