 <span data-ttu-id="5a0e9-106">L'implementazione di WS-AT fornisce Windows Communication Foundation (WCF) include un servizio di protocollo incorporato nel gestore di transazioni Microsoft Distributed Transaction Coordinator (MSDTC).</span><span class="sxs-lookup"><span data-stu-id="5a0e9-106">The WS-AT implementation Windows Communication Foundation (WCF) provides includes a protocol service built into the Microsoft Distributed Transaction Coordinator (MSDTC) transaction manager.</span></span> <span data-ttu-id="5a0e9-107">Usa WS-AT, le applicazioni WCF possono propagare transazioni alle altre applicazioni, tra cui servizi Web interoperativi creati mediante la tecnologia di terze parti.</span><span class="sxs-lookup"><span data-stu-id="5a0e9-107">Using WS-AT, WCF applications can flow transactions to other applications, including interoperable Web services built using third-party technology.</span></span>

 <span data-ttu-id="5a0e9-111">La scelta del protocollo influisce sugli elementi seguenti:</span><span class="sxs-lookup"><span data-stu-id="5a0e9-111">The choice of protocol influences:</span></span>

 <span data-ttu-id="5a0e9-114">Se il server e client vengono scritti tramite WCF, non devi usare WS-AT.</span><span class="sxs-lookup"><span data-stu-id="5a0e9-114">If the server and client are written using WCF, you do not need to use WS-AT.</span></span> <span data-ttu-id="5a0e9-115">È possibile invece utilizzare le impostazioni predefinite di `NetTcpBinding` con l'attributo `TransactionFlow` abilitato, in modo da utilizzare il protocollo `OleTransactions`.</span><span class="sxs-lookup"><span data-stu-id="5a0e9-115">Instead, you can use the default settings of `NetTcpBinding` with the `TransactionFlow` attribute enabled, which will use the `OleTransactions` protocol instead.</span></span> <span data-ttu-id="5a0e9-116">Per altre informazioni, vedere [ \<netTcpBinding >](../../../../docs/framework/configure-apps/file-schema/wcf/nettcpbinding.md).</span><span class="sxs-lookup"><span data-stu-id="5a0e9-116">For more information, see [\<netTcpBinding>](../../../../docs/framework/configure-apps/file-schema/wcf/nettcpbinding.md).</span></span> <span data-ttu-id="5a0e9-117">In caso contrario, se le transazioni vengono propagate a servizi Web basati su tecnologie di terze parti, sarà necessario utilizzare WS-AT.</span><span class="sxs-lookup"><span data-stu-id="5a0e9-117">Otherwise, if you are flowing transactions to Web services built on third-party technologies, you must use WS-AT.</span></span>
