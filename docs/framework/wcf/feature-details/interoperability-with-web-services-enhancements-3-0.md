# <a name="interoperability-with-web-services-enhancements-30"></a>Interoperabilità con Web Services Enhancements 3.0
I client Windows Communication Foundation (WCF) e i servizi sono compatibili a livello di transito con Web Services Enhancements 3.0 per servizi e i client di Microsoft .NET (WSE) quando i client WCF e servizi sono configurati per usare la versione dell'agosto 2004 delle Specifica di WS-Addressing.

 Descrive come configurare un servizio WCF, in modo che una Web Services Enhancements 3.0 per Microsoft .NET (WSE) client possono comunicare con esso.  
  
 [Procedura: Configurare un client WCF per interagire con servizi WSE 3.0](../../../../docs/framework/wcf/feature-details/how-to-configure-a-wcf-client-to-interoperate-with-wse3-0-services.md)  
 Illustra in dettaglio come configurare un client WCF, in modo che comunica con un Web Services Enhancements 3.0 per il servizio di Microsoft .NET (WSE).
