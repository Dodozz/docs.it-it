# <a name="partial-trust"></a>Attendibilità parziale
A partire da [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)], i chiamanti parzialmente attendibili possono accedere a tipi e metodi pubblici implementati in <xref:System.ServiceModel>, <xref:System.Runtime.Serialization>e <xref:System.ServiceModel.Web>. In questa sezione vengono descritti gli scenari supportati per l'uso di Windows Communication Foundation (WCF) all'interno di un'applicazione parzialmente attendibile, nonché il sottoinsieme limitato delle funzionalità WCF disponibile alle applicazioni in esecuzione con sicurezza dall'accesso di codice (CAS) autorizzazioni.

 [Compatibilità con la funzionalità di trust parziale](../../../../docs/framework/wcf/feature-details/partial-trust-feature-compatibility.md)  
 Descrive le funzionalità WCF non possono essere usati con attendibilità parziale.  
  
 [Procedure consigliate per l'attendibilità parziale](../../../../docs/framework/wcf/feature-details/partial-trust-best-practices.md)  
 Contiene le procedure consigliate per l'utilizzo di WCF in applicazioni parzialmente attendibili.
