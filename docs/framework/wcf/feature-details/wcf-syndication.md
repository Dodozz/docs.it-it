# <a name="wcf-syndication"></a>Diffusione WCF
Windows Communication Foundation (WCF) fornisce il supporto usare facilmente feed di diffusione in Atom, RSS o in altri formati personalizzati, che consente di leggere e crearle nonché di esporli in un endpoint del servizio. Negli argomenti di questa sezione viene descritto dettagliatamente tale modello di programmazione per la diffusione.
