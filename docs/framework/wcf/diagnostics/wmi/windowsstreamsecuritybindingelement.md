# <a name="windowsstreamsecuritybindingelement"></a>WindowsStreamSecurityBindingElement
WindowsStreamSecurityBindingElement  
  
## <a name="syntax"></a>Sintassi  
  
```csharp
class WindowsStreamSecurityBindingElement : BindingElement  
{  
  string ProtectionLevel;  
};  
```  
  
## <a name="methods"></a>Metodi  
 La classe WindowsStreamSecurityBindingElement non definisce metodi.  
  
## <a name="properties"></a>Proprietà  
 La classe WindowsStreamSecurityBindingElement dispone della proprietà seguente:  
  
### <a name="protectionlevel"></a>ProtectionLevel  
 Tipo di dati: stringa  
  
 Tipo di accesso: Sola lettura  
  
 Livello di protezione per il flusso TCP.
