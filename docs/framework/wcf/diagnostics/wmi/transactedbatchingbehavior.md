# <a name="transactedbatchingbehavior"></a>TransactedBatchingBehavior
TransactedBatchingBehavior  
  
## <a name="syntax"></a>Sintassi  
  
```csharp
class TransactedBatchingBehavior : Behavior  
{  
  sint32 MaxBatchSize;  
};  
```  
  
## <a name="methods"></a>Metodi  
 La classe TransactedBatchingBehavior non definisce metodi.  
  
## <a name="properties"></a>Proprietà  
 La classe TransactedBatchingBehavior dispone della proprietà seguente:  
  
### <a name="maxbatchsize"></a>MaxBatchSize  
 Tipo di dati: sint32  
  
 Tipo di accesso: Sola lettura  
  
 Dimensione massima del batch transazionale.
