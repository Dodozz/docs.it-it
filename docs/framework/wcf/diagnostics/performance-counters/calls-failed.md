# <a name="calls-failed"></a>Chiamate non riuscite
Nome contatore: Chiamate non riuscite  
  
## <a name="description"></a>Descrizione  
 Numero di chiamate con eccezioni non gestite presenti in questa operazione. Questo contatore viene incrementato ogni volta che si verifica un'eccezione non gestita in questa operazione.
