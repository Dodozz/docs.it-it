# <a name="transactions-flowed"></a>Transazioni propagate
Nome contatore: Transazioni propagate  
  
## <a name="description"></a>Descrizione  
 Numero di transazioni propagate a questa operazione. Il contatore viene incrementato per ogni messaggio che viene inviato all'operazione e che contiene un ID transazione.
