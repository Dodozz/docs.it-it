## <a name="description"></a>Descrizione  
 Questo evento viene generato quando una channel factory è stata rimossa dalla cache perché è stato raggiunto il limite della cache.  
  
## <a name="message"></a>Messaggio  
 Una channel factory è stata rimossa dalla cache perché è stato raggiunto il limite di '%1' della cache.
