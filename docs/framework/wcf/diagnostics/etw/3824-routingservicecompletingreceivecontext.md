## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|Id|3824|  
|Parole chiave|RoutingServices|  
|Livello|Dettagliato|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando il servizio di routing chiama ReceiveContext.Complete per un messaggio con l'ID specificato.  
  
## <a name="message"></a>Messaggio  
 Servizio di routing: è in corso la chiamata a ReceiveContext.Complete per il messaggio con ID '%1'.
