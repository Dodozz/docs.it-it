# <a name="3403---wrappedkeydecryptionstart"></a>3403 - WrappedKeyDecryptionStart
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|Id|3403|  
|Parole chiave|Sicurezza|  
|Livello|Dettagliato|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando la decrittografia della chiave con wrapping viene avviata.  
  
## <a name="message"></a>Messaggio  
 Decrittografia della chiave di cui è stato eseguito il wrapping completata.
