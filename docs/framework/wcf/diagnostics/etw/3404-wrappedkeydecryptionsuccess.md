# <a name="3404---wrappedkeydecryptionsuccess"></a>3404 - WrappedKeyDecryptionSuccess
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|Id|3404|  
|Parole chiave|Sicurezza|  
|Livello|Dettagliato|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando la decrittografia della chiave con wrapping viene completata.  
  
## <a name="message"></a>Messaggio  
 Decrittografia chiave di cui è stato eseguito il wrapping completata.
