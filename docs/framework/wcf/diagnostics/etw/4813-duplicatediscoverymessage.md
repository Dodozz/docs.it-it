# <a name="4813---duplicatediscoverymessage"></a>4813 - DuplicateDiscoveryMessage
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|Id|4813|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando il messaggio viene eliminato perché duplicato.  
  
## <a name="message"></a>Messaggio  
 Un messaggio %1 con messageId='%2' è stato eliminato poiché era un duplicato.
