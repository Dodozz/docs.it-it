# <a name="4811---discoverymessagewithnullrelatesto"></a>4811 - DiscoveryMessageWithNullRelatesTo
## <a name="properties"></a>Proprietà  
  
|||  
|-|-|  
|Id|4811|  
|Parole chiave|Individuazione|  
|Livello|Avviso|  
|Canale|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descrizione  
 Questo evento viene generato quando il messaggio di individuazione viene eliminato da DiscoveryClient poiché l'intestazione del messaggio non conteneva la proprietà RelatesTo richiesta.  
  
## <a name="message"></a>Messaggio  
 Un messaggio %1 con messageId='%2' è stato eliminato da DiscoveryClient poiché l'intestazione del messaggio non conteneva la proprietà RelatesTo richiesta.
