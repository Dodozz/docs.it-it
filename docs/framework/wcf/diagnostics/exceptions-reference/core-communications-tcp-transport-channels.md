# <a name="core-communications-tcp-transport-channels"></a>Comunicazioni di base: canali di trasporto TCP
In questo argomento vengono elencate tutte le eccezioni generate dai canali di trasporto TCP.  
  
## <a name="exception-list"></a>Elenco delle eccezioni  
  
|Codice risorsa|Stringa di risorsa|  
|-------------------|---------------------|  
|SocketCloseReadTimeout|L'endpoint remoto del socket specificato non ha risposto a una richiesta di chiusura entro il timeout assegnato specificato. È possibile che l'endpoint remoto non stia chiamando Close dopo avere ricevuto il segnale EOF (null) dall'operazione di ricezione. È possibile che la durata consentita per l'operazione fosse una porzione di un timeout più lungo.|
