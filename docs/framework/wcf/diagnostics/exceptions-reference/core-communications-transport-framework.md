# <a name="core-communications-transport-framework"></a>Comunicazioni di base: framework di trasporto
In questo argomento vengono elencate tutte le eccezioni generate dal framework di trasporto.  
  
## <a name="exception-list"></a>Elenco delle eccezioni  
  
|Codice risorsa|Stringa di risorsa|  
|-------------------|---------------------|  
|Hosting_TransportBindingNotFound|Nessuna associazione di protocollo corrisponde all'indirizzo specificato. Le associazioni di protocollo sono configurate a livello di sito nella configurazione di IIS o WAS.|
