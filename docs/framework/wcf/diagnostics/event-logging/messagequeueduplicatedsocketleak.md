# <a name="messagequeueduplicatedsocketleak"></a>MessageQueueDuplicatedSocketLeak
Id: 165  
  
 Gravità: Error  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che si è verificato un errore durante l'invio di un socket duplicato. L'handle è perso nel processo. Vengono indicati l'origine, l'eccezione, il nome e l'ID del processo.
