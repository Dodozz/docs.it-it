# <a name="wasdisconnected"></a>WasDisconnected
Id: 162  
  
 Gravità: Error  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrizione  
 Questo evento segnala la disconnessione del servizio di attivazione dei processi di Windows (WAS, Windows Process Activation Service). Vengono elencati il valore HRESULT nonché il nome e l'ID del processo.
