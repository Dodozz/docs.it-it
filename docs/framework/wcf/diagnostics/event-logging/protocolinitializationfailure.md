# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure
Id: 142  
  
 Gravità: Error  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che è stato impossibile inizializzare il servizio del protocollo WS-Atomic Transaction. La funzionalità WS-Atomic Transaction viene pertanto disattivata. Vengono indicati l'ID e il nome del protocollo, l'eccezione, il nome e l'ID del processo. .
