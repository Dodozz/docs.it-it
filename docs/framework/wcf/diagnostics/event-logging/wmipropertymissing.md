# <a name="wmipropertymissing"></a>WmiPropertyMissing
Id: 129  
  
 Gravità: Error  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che si è verificata un'errata corrispondenza del tipo. Per la classe CIM specificata non è disponibile la proprietà specificata del tipo specificato. Vengono indicati il nome e l'ID del processo.
