# <a name="failedtocreatemessageloggingtracesource"></a>FailedToCreateMessageLoggingTraceSource
Id: 115  
  
 Gravità: Error  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che la funzionalità di registrazione dei messaggi non è stata in grado di creare l'origine di traccia. Vengono indicati l'eccezione nonché il nome e l'ID del processo.
