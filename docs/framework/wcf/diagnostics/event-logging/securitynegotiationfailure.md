# <a name="securitynegotiationfailure"></a>SecurityNegotiationFailure
Id: 172  
  
 Gravità: Error  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che la negoziazione di sicurezza non è riuscita. Vengono indicati il servizio, l'azione, l'identità client, l'ID attività e la negoziazione.
