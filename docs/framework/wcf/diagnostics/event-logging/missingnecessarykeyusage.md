# <a name="missingnecessarykeyusage"></a>MissingNecessaryKeyUsage
Id: 156  
  
 Gravità: Error  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che il certificato di identità con il nome del soggetto e l'identificazione personale specificati non fornisce alcuni dati specifici tra i valori KeyUsage. Vengono inoltre indicati il nome e l'ID del processo.
