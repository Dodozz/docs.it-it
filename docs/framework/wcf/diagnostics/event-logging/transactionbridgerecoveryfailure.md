# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
Id: 146  
  
 Gravità: Error  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che si sono verificati errori di MSDTC TransactionBridge durante il ripristino. È una condizione irreversibile, con conseguente chiusura del servizio MSDTC. Nell'evento vengono specificati l'eccezione, il nome del processo e l'ID del processo. .
