# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed
Id: 126  
  
 Gravità: Error  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che la registrazione del provider WMI non è stata eseguita. Vengono indicati l'oggetto WMI, l'errore, il nome e l'ID del processo.
