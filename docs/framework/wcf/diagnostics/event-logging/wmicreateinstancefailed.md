# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed
Id: 123  
  
 Gravità: Error  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che la richiesta WMI CreateInstance non è stata elaborata. Nell'evento vengono indicati il nome della classe, l'eccezione, nonché il nome e l'ID del processo.
