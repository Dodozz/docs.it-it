# <a name="failedtoloadperformancecounter"></a>FailedToLoadPerformanceCounter
Id: 118  
  
 Gravità: Error  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che un contatore delle prestazioni non è stato caricato. L'evento elenca il nome della categoria, il nome del contatore, il nome dell'istanza, l'eccezione, il nome e l'ID del processo.
