# <a name="performancecounterinitializationfailure"></a>PerformanceCounterInitializationFailure
Id: 149  
  
 Gravità: Error  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che è stato impossibile inizializzare un contatore delle prestazioni. Vengono indicati il nome del contatore, l'eccezione, il nome e l'ID del processo.
