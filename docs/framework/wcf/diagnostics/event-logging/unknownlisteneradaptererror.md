# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
Id: 161  
  
 Gravità: Error  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che si è verificato un errore imprevisto nell'adattatore del listener durante la gestione di una notifica WAS. Il processo viene chiuso. Nell'evento vengono indicati il protocollo, l'eccezione, nonché il nome e l'ID del processo.
