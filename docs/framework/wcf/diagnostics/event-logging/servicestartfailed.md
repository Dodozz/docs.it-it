# <a name="servicestartfailed"></a>ServiceStartFailed
Id: 164  
  
 Gravità: Error  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che una richiesta di avvio del servizio non è riuscita. Nell'evento vengono specificati il codice di errore, il nome del processo e l'ID del processo.
