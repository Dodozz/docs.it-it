# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
Id: 130  
  
 Gravità: Error  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che si è verificato un errore durante l'avvio del servizio COM+. Viene elencato l'AppID, il ClsID, il surrogato, l'eccezione, il nome e l'ID del processo.
