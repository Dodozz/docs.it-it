# <a name="participantrecoverylogentrycorrupt"></a>ParticipantRecoveryLogEntryCorrupt
Id: 138  
  
 Gravità: Error  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che una voce del registro di ripristino del partecipante è danneggiata e non può essere deserializzata. L'errore può causare una perdita di dati. L'evento elenca l'ID transazione, i dati di ripristino (codifica Base64), l'eccezione, il nome processo e l'ID processo.
