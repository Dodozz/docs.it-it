# <a name="protocolstopfailure"></a>ProtocolStopFailure
Id: 147  
  
 Gravità: Error  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che è stato impossibile arrestare il servizio del protocollo WS-Atomic Transaction. Vengono indicati l'ID e il nome del protocollo, l'eccezione, il nome e l'ID del processo.
