# <a name="systemservicemodelserviceoperationmissingreplycontext"></a>System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## <a name="description"></a>Descrizione  
 ReplyContext è Null per un'operazione bidirezionale. I dati di traccia viene visualizzato il messaggio seguente: "Operazione di risposta non è disponibile alcun IRequestContext da utilizzare per la risposta".  
  
 Questa traccia è utile per l'esecuzione del debug durante lo sviluppo.
