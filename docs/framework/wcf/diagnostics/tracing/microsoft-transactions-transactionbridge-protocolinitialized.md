# <a name="microsofttransactionstransactionbridgeprotocolinitialized"></a>Microsoft.Transactions.TransactionBridge.ProtocolInitialized
Il servizio del protocollo WS-Atomic Transaction è stato inizializzato.  
  
## <a name="description"></a>Descrizione  
 Viene tracciato durante l'avvio del gestore transazioni per indicare quale protocollo è stato inizializzato correttamente.
