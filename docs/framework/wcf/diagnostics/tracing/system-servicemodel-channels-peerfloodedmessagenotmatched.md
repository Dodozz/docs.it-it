# <a name="systemservicemodelchannelspeerfloodedmessagenotmatched"></a>System.ServiceModel.Channels.PeerFloodedMessageNotMatched
Un PeerNode ha ricevuto un messaggio non corrispondente a nessun canale locale.  
  
## <a name="description"></a>Descrizione  
 Si verifica durante l'elaborazione di un messaggio flood in arrivo.
