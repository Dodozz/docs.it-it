# <a name="systemservicemodelactivationwebhostserviceactivated"></a>System.ServiceModel.Activation.WebHostServiceActivated
System.ServiceModel.Activation.WebHostServiceActivated  
  
## <a name="description"></a>Descrizione  
 Si verifica quando un servizio ospitato in IIS (Internet Information Services) o in WAS (Windows Process Activation Service ) viene attivato completamente e inizia ad attendere messaggi.
