# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion
L'istanza del servizio è stata rilasciata al completamento della transazione '{0}' perché la classe ServiceBehaviorAttribute di ReleaseServiceInstanceOnTransactionComplete è stata impostata su true.  
  
## <a name="description"></a>Descrizione  
 Traccia eseguita quando l'istanza del servizio corrente viene rilasciata o eliminata a causa del completamento della transazione attiva corrente e dell'impostazione di ReleaseServiceInstanceOnTransactionComplete su `true`.
