# <a name="guide-to-the-documentation"></a>Guida alla documentazione
Fornite di seguito sono riportate indicazioni relative alla documentazione di Windows Communication Foundation (WCF). I documenti collegati sono punti di partenza consigliati, raggruppati secondo specifici interessi e livelli di esperienza.  
  
 Per installare una versione autonoma della documentazione e un visualizzatore della Guida, scaricare il [Microsoft Windows SDK v 7.1](https://go.microsoft.com/fwlink/?LinkID=194146&clcid=0x409).  
  
## <a name="new-to-windows-communication-foundation-programming"></a>Novità della programmazione in Windows Communication Foundation  
  
-   Se si ha familiarità con la programmazione con WCF e si desidera semplicemente esaminare esempi di applicazioni funzionanti, vedere gli argomenti elencati [degli esempi Windows Communication Foundation](../../../docs/framework/wcf/samples/index.md).  
  
-   Per un'esercitazione che illustra i passaggi di base della creazione di un servizio WCF e un client, vedere [esercitazione introduttiva](../../../docs/framework/wcf/getting-started-tutorial.md).  
  
-   Se è interessati ai concetti alla base di WCF, vedere gli argomenti in di [panoramica concettuale](../../../docs/framework/wcf/conceptual-overview.md) sezione.  
  
-   Per vedere esempi grafici delle configurazioni di sicurezza client/server, vedere [scenari di sicurezza comuni](../../../docs/framework/wcf/feature-details/common-security-scenarios.md).  
  
## <a name="programming-in-depth"></a>Informazioni dettagliate sulla programmazione  
  
-   Se si è pronti per iniziare a sviluppare un'applicazione, vedere [programmazione WCF di base](../../../docs/framework/wcf/basic-wcf-programming.md).  
  
-   Se si sta cercando informazioni aggiuntive relative a una determinata funzionalità o funzionalità di WCF, vedere gli argomenti in [dettagli delle funzionalità di WCF](../../../docs/framework/wcf/feature-details/index.md).  
  
-   Se si desidera estendere o personalizzare WCF per soddisfare esigenze specifiche, vedere [estensione di WCF](../../../docs/framework/wcf/extending/index.md).  
  
-   Per informazioni sugli strumenti che consentono di creare ed eseguire il debug di applicazioni WCF, vedere [strumenti di Windows Communication Foundation](../../../docs/framework/wcf/tools.md).  
  
-   Utilizzo di file XML di configurazione è un modo primario di programmazione client e servizi WCF. Per documentazione di riferimento per gli elementi XML utilizzati nei file di configurazione, vedere [Schema di configurazione WCF](../../../docs/framework/configure-apps/file-schema/wcf/index.md).

## <a name="using-windows-communication-foundation-with-other-technologies"></a>Utilizzo di Windows Communication Foundation con altre tecnologie  
  
-   Per creare un servizio che comunica con i client ASP.NET, vedere [come: Configurare un servizio WCF per interagire con i client del servizio Web ASP.NET](../../../docs/framework/wcf/feature-details/config-wcf-service-with-aspnet-web-service.md).  
  
-   Integrazione con [!INCLUDE[dnprdnshort](../../../includes/dnprdnshort-md.md)] remoting è spiegata in [eseguire la migrazione di applicazioni .NET Remoting a WCF](../../../docs/framework/wcf/feature-details/migrating-net-remoting-applications-to-wcf.md).  
  
-   Per integrare un'applicazione COM+ esistente con un servizio WCF o un client, vedere [l'integrazione con panoramica delle applicazioni COM+](../../../docs/framework/wcf/feature-details/integrating-with-com-plus-applications-overview.md).  
  
-   Per integrare un'applicazione COM+ esistente con un servizio WCF o un client, vedere [integrazione con applicazioni COM](../../../docs/framework/wcf/feature-details/integrating-with-com-applications.md).  
  
-   Per integrare un'applicazione MSMQ esistente con un servizio WCF o un client, vedere [come: Scambiare messaggi con endpoint WCF in coda](../../../docs/framework/wcf/feature-details/how-to-exchange-queued-messages-with-wcf-endpoints.md) e [come: Scambiare messaggi con endpoint WCF e applicazioni di Accodamento messaggi](../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md).  
  
-   Per usare Internet Information Services (IIS) per ospitare un servizio, vedere [servizi di Hosting](../../../docs/framework/wcf/hosting-services.md).  
  
-   Per usare WCF per utilizzare un servizio Web Services Extensions (WSE) 3.0, vedere [come: Accedere a WSE 3.0 Service](../../../docs/framework/wcf/feature-details/how-to-access-a-wse-3-0-service-with-a-wcf-client.md).
