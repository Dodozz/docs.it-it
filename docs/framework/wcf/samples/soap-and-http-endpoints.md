 <span data-ttu-id="8d2bc-109">Questo esempio è costituito da due componenti: un progetto di applicazione Web (servizio) che contiene un servizio WCF e un'applicazione console (Client) che richiama le operazioni di servizio usando associazioni SOAP e HTTP.</span><span class="sxs-lookup"><span data-stu-id="8d2bc-109">This sample consists of two components: a Web Application project (Service) that contains a WCF service and a console application (Client) that invokes service operations using SOAP and HTTP bindings.</span></span>

 <span data-ttu-id="8d2bc-115">Il file Web. config configura il servizio WCF con 3 endpoint:</span><span class="sxs-lookup"><span data-stu-id="8d2bc-115">The Web.config file configures the WCF service with 3 endpoints:</span></span>

- <span data-ttu-id="8d2bc-118">L'endpoint ~/service.svc/soap che consente ai client di accedere al servizio usando SOAP sull'associazione HTTP.</span><span class="sxs-lookup"><span data-stu-id="8d2bc-118">The ~/service.svc/soap endpoint that allows the clients to access the service using the SOAP over HTTP binding.</span></span>

 <span data-ttu-id="8d2bc-121">Il progetto client viene illustrato l'accesso al servizio tramite un proxy SOAP (generato tramite **Aggiungi riferimento al servizio**) e l'accesso del servizio utilizzando <xref:System.Net.WebClient>.</span><span class="sxs-lookup"><span data-stu-id="8d2bc-121">The client project demonstrates accessing the service using a SOAP proxy (generated through **Add Service Reference**) and accessing the service using <xref:System.Net.WebClient>.</span></span>  
  
 <span data-ttu-id="8d2bc-122">L'esempio è costituito da un servizio ospitato sul Web e da un'applicazione console.</span><span class="sxs-lookup"><span data-stu-id="8d2bc-122">The sample consists of a Web-hosted service and a console application.</span></span> <span data-ttu-id="8d2bc-123">Quando l'applicazione console è in esecuzione, il client invia richieste al servizio e scrive nella finestra della console le informazioni pertinenti incluse nelle risposte.</span><span class="sxs-lookup"><span data-stu-id="8d2bc-123">As the console application runs, the client makes requests to the service and writes the pertinent information from the responses to the console window.</span></span>
