# <a name="ltpropertynamegt-implicitly-defines-ltmembernamegt-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-lttypenamegt"></a>«&lt;propertyname&gt;'definisce in modo implicito'&lt;membername&gt;', che è in conflitto con un membro con lo stesso nome in &lt;tipo&gt; '&lt;typename&gt;»
Il nome di un membro di tipo è in conflitto con il nome di un membro creato in modo implicito per una proprietà. Le proprietà creano in modo implicito diverse variabili implicite che possono essere in conflitto con i membri definiti.  
  
 **ID errore:** BC31060  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rinominare il membro dichiarato in modo esplicito per rimuovere il conflitto di denominazione.
