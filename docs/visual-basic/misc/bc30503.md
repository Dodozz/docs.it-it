# <a name="properties-in-a-module-cannot-be-declared-specifier"></a>Le proprietà in un modulo non possono essere dichiarate '\<identificatore >'
Si è provato a usare un identificatore, ad esempio `MustOverride`, `NotOverridable`o `Overridable`, in una proprietà all'interno di un modulo.  
  
 **ID errore:** BC30503  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere l'identificatore.
