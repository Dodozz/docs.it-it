# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Le istruzioni di una dichiarazione di variabile non sono valide nella finestra di controllo immediato
Le dichiarazioni di variabili non possono essere usate nella finestra di controllo **immediato** .  
  
 **ID errore:** BC30709  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Non dichiarare variabili nella finestra di controllo **immediato** .
