# <a name="-expected"></a><span data-ttu-id="17b6e-102">È previsto '{'</span><span class="sxs-lookup"><span data-stu-id="17b6e-102">'{' expected</span></span>
<span data-ttu-id="17b6e-103">Per dichiarare un'istanza di un tipo denominato o anonimo usando un inizializzatore di oggetto, è necessario racchiudere l'elenco dei campi o delle proprietà e i rispettivi valori iniziali tra parentesi graffe ({e}).</span><span class="sxs-lookup"><span data-stu-id="17b6e-103">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>  
  
```  
Dim client As New Customer() With {.Name = "Microsoft", .City = "Seattle"}  
Dim emp = New Employee() With {.Name = "Rob Young", .ID = 55555}  
Dim anon = New With {.ID = 123456}  
```
