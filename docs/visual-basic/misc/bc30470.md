# <a name="myclass-cannot-be-used-outside-of-a-class"></a>'MyClass' non può essere utilizzata all'esterno di una classe
È stata usata la parola chiave `MyClass` all'esterno di una struttura di classe.  
  
 **ID errore:** BC30470  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Spostare la dichiarazione `MyClass` all'interno della struttura di classe.
