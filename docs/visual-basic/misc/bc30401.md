# <a name="ltinterfacename1gt-cannot-implement-ltmethodnamegt-because-there-is-no-matching-ltmethodgt-on-interface-ltinterfacename2gt"></a>«&lt;interfacename1&gt;'non può implementare'&lt;NomeMetodo&gt;' perché non corrisponde ad alcuna &lt;metodo&gt; nell'interfaccia '&lt;interfacename2&gt;»
Si è provato a implementare un metodo in un'interfaccia, ma non esiste un metodo corrispondente nell'interfaccia da cui deriva la prima interfaccia.  
  
 **ID errore:** BC30401  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che il nome del metodo sia stato digitato in modo corretto oppure fornire il metodo mancante.
