# <a name="interfacename1-cannot-implement-methodname-because-there-is-no-matching-method-on-interface-interfacename2"></a>'\<interfacename1 >' non può implementare '\<NomeMetodo >' perché non corrisponde ad alcuna \<metodo > nell'interfaccia '\<interfacename2 >'
Si è provato a implementare un metodo in un'interfaccia, ma non esiste un metodo corrispondente nell'interfaccia da cui deriva la prima interfaccia.  
  
 **ID errore:** BC30401  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che il nome del metodo sia stato digitato in modo corretto oppure fornire il metodo mancante.
