# <a name="ltnamegt-is-ambiguous-between-declarations-in-modules-ltmodulename1gt-and-ltmodulename2gt"></a>«&lt;name&gt;'è ambiguo tra le dichiarazioni nei moduli'&lt;modulename1&gt;"e"&lt;modulename2&gt;»
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il compilatore Visual Basic non dispone di regole di risoluzione dei conflitti. è necessario evitare ambiguità tra nomi manualmente.  
  
 **ID errore:** BC30562  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare un nome completo.
