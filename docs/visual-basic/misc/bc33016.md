# <a name="operator-ltoperatorgt-must-have-either-one-or-two-parameters"></a>Operatore '&lt;operatore&gt;' deve avere uno o due parametri
Un operatore che può essere unario o binario è definito senza parametro oppure con più di due parametri.  
  
 Un operatore unario/binario deve avere uno o due parametri.  
  
 **ID errore:** BC33016  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare la definizione per specificare uno o due parametri.  
  
-   Se è necessario senza parametri o con più di due, è necessario usare il [istruzione Function](../../visual-basic/language-reference/statements/function-statement.md) per definire un `Function` routine invece di un operatore.
