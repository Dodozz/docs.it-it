# <a name="namespace-or-type-name-has-already-been-imported"></a>Namespace o di tipo '\<nome >' è già stato importato
Si è provato a importare più volte uno spazio dei nomi.  
  
 **ID errore:** BC31051  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere le istruzioni `Imports` ridondanti.
