# <a name="ltnamegt-necessary-for-compiling-this-construct-cannot-be-found"></a>«&lt;nome&gt;', necessario per compilare il costrutto non è stata trovata
Un elemento necessario per la compilazione non è stato trovato.  
  
 **ID errore:** BC30930  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Specificare l'elemento helper necessario.  
  
2.  Verificare che il codice sia codice gestito valido.
