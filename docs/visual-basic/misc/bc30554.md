# <a name="namespacename-is-ambiguous"></a>'\<NomeSpazioDeiNomi >' è ambiguo
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il compilatore Visual Basic è privo di eventuali regole di risoluzione dei conflitti. è necessario evitare ambiguità tra nomi manualmente.  
  
 **ID errore:** BC30554  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Specificare un nome completo.
