# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;type1&gt; '&lt;typename&gt;' e &lt;type2&gt; '&lt;typename&gt;' sono in conflitto nel &lt;spazio dei nomi&gt; '&lt;NomeSpazioDeiNomi &gt;'
Due elementi dichiarati, ad esempio una classe e un modulo, hanno lo stesso nome nello stesso spazio dei nomi.  
  
 **ID errore:** BC30179  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Spostare la dichiarazione di uno degli elementi in uno spazio dei nomi diverso.  
  
-   Modificare il nome di uno degli elementi.
