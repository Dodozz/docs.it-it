# <a name="type1-typename-and-type2-typename-conflict-in-namespace-namespacename"></a>\<type1 > '\<nomeTipo >' e \<type2 > '\<nomeTipo >' sono in conflitto nel \<dello spazio dei nomi > '\<NomeSpazioDeiNomi >'
Due elementi dichiarati, ad esempio una classe e un modulo, hanno lo stesso nome nello stesso spazio dei nomi.  
  
 **ID errore:** BC30179  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Spostare la dichiarazione di uno degli elementi in uno spazio dei nomi diverso.  
  
- Modificare il nome di uno degli elementi.
