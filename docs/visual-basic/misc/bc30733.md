# <a name="ltmembernamegt-is-already-declared-by-ltprocedurenamegt-which-was-generated-for-this-ltcontainernamegt"></a>'&lt;nomemembro&gt;'è già dichiarato'&lt;nomeroutine&gt;', che è stato generato per questo &lt;containername&gt;
È stato dichiarato un membro che ha lo stesso nome di un membro dichiarato da Common Language Runtime.  
  
 **ID errore:** BC30733  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il nome del membro per rimuovere il conflitto.
