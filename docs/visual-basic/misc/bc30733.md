# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<nomeMembro >' è già dichiarato '\<nomeroutine >', che è stato generato per questo \<nomecontenitore >

È stato dichiarato un membro che ha lo stesso nome di un membro dichiarato da common language runtime.

**ID errore:** BC30733

## <a name="to-correct-this-error"></a>Per correggere l'errore

- Modificare il nome del membro per rimuovere il conflitto.
