# <a name="project-projectname-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-classname"></a>Progetto '\<NomeProgetto >' contiene un riferimento indiretto all'assembly '\<assemblyname >' che contiene '\<NomeClasse >'
Progetto '\<NomeProgetto >' contiene un riferimento indiretto all'assembly '\<assemblyname >' che contiene '\<NomeClasse >'. Aggiungere un riferimento a '\<assemblyname >' al progetto.  
  
 Il progetto contiene membri che dipendono da un assembly non incluso nei riferimenti del progetto.  
  
 **ID errore:** BC31515  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere un riferimento all'assembly facendo clic con il pulsante destro del mouse su **Riferimenti** in **Esplora soluzioni** e scegliendo **Aggiungi riferimento**.
