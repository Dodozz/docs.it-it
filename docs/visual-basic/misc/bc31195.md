# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>L'operatore 'operator' non è definito per i tipi '&lt;tipo1&gt;' e '&lt;tipo2&gt;'
L'operatore 'operator' non è definito per i tipi '<tipo1>' e '<tipo2>'. È possibile usare la proprietà 'Value' per ottenere il valore di stringa del primo elemento di '\<ElementoPadre >'.  
  
 È stato usato un valore letterale XML in un file binario o in un confronto `Boolean` .  
  
 **ID errore:** BC31195  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare la proprietà `Value` del valore letterale XML per il file binario o il confronto `Boolean` .
