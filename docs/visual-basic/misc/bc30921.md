# <a name="ltderivedtypenamegt-cannot-inherit-from-lttypegt-ltconstructedbasetypenamegt-because-it-expands-the-access-of-type-ltinternaltypenamegt-to-ltregiongt-ltregionnamegt"></a><span data-ttu-id="1becc-102">«&lt;nometipoderivato&gt;' non può ereditare da &lt;tipo&gt; '&lt;nometipobasecreato&gt;'perché espande l'accesso di tipo'&lt;nometipointerno&gt;' per &lt;regione&gt; '&lt;regionname&gt;'</span><span class="sxs-lookup"><span data-stu-id="1becc-102">'&lt;derivedtypename&gt;' cannot inherit from &lt;type&gt; '&lt;constructedbasetypename&gt;' because it expands the access of type '&lt;internaltypename&gt;' to &lt;region&gt; '&lt;regionname&gt;'</span></span>
<span data-ttu-id="1becc-103">Una classe o un'interfaccia derivata prova a espandere il livello di accesso di un tipo interno usandolo come argomento di tipo per una classe o un'interfaccia base.</span><span class="sxs-lookup"><span data-stu-id="1becc-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="1becc-104">Il codice seguente può generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="1becc-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="1becc-105">Al codice esterno a `containingClass` non è consentito l'accesso a `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="1becc-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="1becc-106">Tuttavia, è possibile accedere a `derivedClass` da qualsiasi codice nello stesso assembly.</span><span class="sxs-lookup"><span data-stu-id="1becc-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="1becc-107">Per questo motivo `derivedClass` non può ereditare `baseClass` se usa `internalStructure` come argomento di tipo, perché ciò potrebbe esporre `internalStructure` in tutta la regione del codice di definizione.</span><span class="sxs-lookup"><span data-stu-id="1becc-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="1becc-108">**ID errore:** BC30921</span><span class="sxs-lookup"><span data-stu-id="1becc-108">**Error ID:** BC30921</span></span>
