# <a name="project-has-been-closed"></a>Il progetto è stato chiuso
Si è verificato un errore di accesso del modello di codice nel compilatore Visual Basic.  
  
 **ID errore:** BC32301  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Ripetere la compilazione del programma, per controllare se l'errore si verifica di nuovo.  
  
2.  Se l'errore si ripete, reinstallare il compilatore Visual Basic.  
  
3.  Se l'errore persiste dopo la reinstallazione, raccogliere informazioni sulla situazione contingente e informare il Servizio supporto tecnico Microsoft.
