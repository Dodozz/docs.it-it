# <a name="name-is-ambiguous-in-the-application-objects-list"></a>«\<nome >' è ambiguo negli oggetti applicazione\<elenco >»
È stato specificato un nome che è ambiguo ed è quindi in conflitto con un altro nome. Il compilatore Visual Basic non dispone di regole di risoluzione dei conflitti. è necessario evitare ambiguità tra nomi manualmente. L'elenco fornito nei dettagli del messaggio di errore indica gli oggetti dell'applicazione in cui il nome è ambiguo.  
  
 **ID errore:** BC30563  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare un nome completo.
