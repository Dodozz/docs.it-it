# <a name="named-arguments-cannot-match-paramarray-parameters"></a>Gli argomenti denominati non possono corrispondere ai parametri ParamArray
Le matrici di parametro devono essere passate per valore.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Passare gli argomenti `ByVal`invece di denominarli.
