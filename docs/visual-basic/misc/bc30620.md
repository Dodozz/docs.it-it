# <a name="option-strict-can-be-followed-only-by-on-or-off"></a>'Option Strict' può essere seguito solo da 'On' o 'Off'
Le istruzioni`Option Strict` devono specificare `On` o `Off`. Tutti gli altri valori non sono validi.  
  
 **ID errore:** BC30620  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare `On` o `Off` per l'istruzione `Option Strict` .
