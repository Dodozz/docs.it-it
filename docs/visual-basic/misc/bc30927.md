# <a name="mustoverride-cannot-be-specified-on-procedurename-because-it-is-in-a-partial-type-that-is-declared-notinheritable-in-another-partial-definition"></a><span data-ttu-id="73a54-102">'MustOverride' non può essere specificato in '\<nomeroutine >' perché si trova in un tipo parziale dichiarato 'NotInheritable' in un'altra definizione parziale</span><span class="sxs-lookup"><span data-stu-id="73a54-102">'MustOverride' cannot be specified on '\<procedurename>' because it is in a partial type that is declared 'NotInheritable' in another partial definition</span></span>
<span data-ttu-id="73a54-103">Una routine o una proprietà è dichiarata come `MustOverride` all'interno di una classe definita in più dichiarazioni parziali, ma una delle definizioni parziali specifica `NotInheritable` per la classe.</span><span class="sxs-lookup"><span data-stu-id="73a54-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="73a54-104">Quando si divide la definizione di una classe in diverse dichiarazioni parziali, il compilatore considera la classe come l'unione di tutte le sue dichiarazioni parziali.</span><span class="sxs-lookup"><span data-stu-id="73a54-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="73a54-105">Questo riguarda non soltanto i membri, ma anche l'implementazione, l'ereditarietà e il livello di accesso.</span><span class="sxs-lookup"><span data-stu-id="73a54-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="73a54-106">Per eseguire l'override di una routine o una proprietà, una classe deve ereditarla da una classe base.</span><span class="sxs-lookup"><span data-stu-id="73a54-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="73a54-107">Per specificare `MustOverride` per una routine o una proprietà in una classe base, di conseguenza, è necessario specificare `MustInherit` per la classe.</span><span class="sxs-lookup"><span data-stu-id="73a54-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="73a54-108">Dal momento che `MustInherit` e `NotInheritable` sono in contraddizione fra loro, non è possibile specificarli entrambi per la medesima classe.</span><span class="sxs-lookup"><span data-stu-id="73a54-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>  
  
 <span data-ttu-id="73a54-109">**ID errore:** BC30927</span><span class="sxs-lookup"><span data-stu-id="73a54-109">**Error ID:** BC30927</span></span>
