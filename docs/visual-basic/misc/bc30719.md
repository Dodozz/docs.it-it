# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>Le istruzioni 'Case' non sono valide nella finestra di controllo immediato
Le istruzioni`Case` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30719  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Case` dalla finestra di controllo **immediato** .
