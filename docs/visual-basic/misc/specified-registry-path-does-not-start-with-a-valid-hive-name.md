# <a name="specified-registry-path-does-not-start-with-a-valid-hive-name"></a>Il percorso del Registro di sistema specificato non inizia con un nome di hive valido
Il percorso del Registro di sistema specificato non inizia con un nome di hive valido. I nomi di hive validi sono:  
  
-   `HKEY_CLASSES_ROOT`  
  
-   `HKEY_CURRENT_CONFIG`  
  
-   `HKEY_CURRENT_USER`  
  
-   `HKEY_DYNDATA`  
  
-   `HKEY_LOCAL_MACHINE`  
  
-   `HKEY_PERFORMANCE_DATA`  
  
-   `HKEY_USERS`  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che sia stato fornito un nome di hive valido.
