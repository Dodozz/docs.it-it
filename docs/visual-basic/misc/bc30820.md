# <a name="ltlsetstmtgt-is-not-declared"></a>«&lt;lsetstmt&gt;' non è dichiarato
'\<lsetstmt >' non è dichiarato. Le istruzioni LSet non sono più supportate. Usare Microsoft.VisualBasic.LSet.  
  
 Numerose funzioni che erano intrinseche a Visual Basic nelle versioni precedenti sono state spostate i <xref:Microsoft.VisualBasic?displayProperty=nameWithType> dello spazio dei nomi. In questo modo le relative funzionalità sono rese più generalmente disponibili per tutti i linguaggi di programmazione.  
  
 **ID errore:** BC30820  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare la funzione `LSet` nello spazio dei nomi `Microsoft.VisualBasic` .
