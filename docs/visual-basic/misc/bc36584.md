# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>Parametro '\<nomeparametro >' nel metodo di estensione '\<NomeMetodo >' definito in '\<nomeTipo >' ha già un argomento corrispondente
Un argomento viene passato in base al nome più di una volta nella stessa chiamata di routine.  
  
 **ID errore:** BC36584  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare se uno dei nomi degli argomenti sia stato digitato in modo errato.  
  
-   Rimuovere l'argomento duplicato dall'elenco di argomenti.
