# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>'End AddHandler' deve essere la prima istruzione in una riga
Un'istruzione `End AddHandler` segue un separatore di istruzioni due punti (:). `End AddHandler` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC31118
