# <a name="field-or-property-ltmembernamegt-is-not-found"></a>Campo o una proprietà '&lt;nomemembro&gt;' non è stato trovato
Viene fatto riferimento a un campo o a una proprietà per l'attributo.  
  
 **ID errore:** BC30661  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare l'ortografia del campo o della proprietà.  
  
2.  Verificare che il progetto possa accedere alla classe Attribute corretta e aggiungerla ai riferimenti al progetto, se necessario.
