# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>Non è possibile usare i caratteri tipo nelle dichiarazioni della variabile di intervallo
Un carattere tipo è stato incluso alla fine del nome di una dichiarazione della variabile di intervallo in una query LINQ.  
  
 **ID errore:** BC36601  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Rimuovere il carattere tipo dalla dichiarazione della variabile di intervallo.
