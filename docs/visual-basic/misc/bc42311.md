# <a name="unable-to-create-xml-documentation-file-name-message"></a>Impossibile creare il file di documentazione XML '\<nome >': \<messaggio >
Il file di documentazione XML non può essere creato a causa di un errore.  
  
 **ID errore:** BC42311  
  
 I messaggi seguenti vengono visualizzati con questo errore:  
  
- [Il blocco di commento XML non può essere associato a un elemento del linguaggio che supporta l'applicazione di commenti relativi alla documentazione XML](../../visual-basic/misc/bc42312.md)  
  
- [Il tag 'returns' del commento XML non è consentito in una proprietà 'WriteOnly'](../../visual-basic/misc/bc42313.md)  
  
- [Il commento XML non può essere applicato più volte in un elemento parziale \<tipo >](../../visual-basic/misc/bc42314.md).  
  
- [Non è possibile applicare il commento XML più di una volta in un elemento ](../../visual-basic/misc/bc42315.md) parziale  
  
- [Errore di analisi di documentazione XML: Tag di inizio '\<tag >' non esiste un corrispondente tag di fine](../../visual-basic/misc/bc42316.md).  
  
- [Parametro di tipo del commento XML \<parametro > non corrisponde a un parametro di tipo corrispondente \<parola chiave > istruzione](../../visual-basic/misc/bc42317.md).  
  
- [Il parametro di tipo del commento XML deve avere un attributo 'name'](../../visual-basic/misc/bc42318.md)  
  
- [Creare commenti relativi alla documentazione XML](../../visual-basic/language-reference/error-messages/xml-comment-exception-must-have-a-cref-attribute.md)  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Per indicazioni, consultare il messaggio specificato.
