# <a name="name-cannot-expose-type-type1-used-in-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type2"></a>'\<nome >' non può esporre il tipo '\<type1 >' usato nel tipo delegato sottostante '\<tipodelegato >' dell'evento che sta implementando all'esterno del progetto mediante \<identificatore > '\<type2 >'
Il codice espone un tipo privato all'esterno della classe pubblica.  
  
 **ID errore:** BC30924  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare il tipo `Public` o usare un altro tipo.
