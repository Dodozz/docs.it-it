# <a name="ltnamegt-cannot-expose-type-lttype1gt-used-in-the-underlying-delegate-type-ltdelegatetypegt-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-lttype2gt"></a>«&lt;name&gt;'non può esporre il tipo'&lt;type1&gt;'usato nel tipo delegato sottostante'&lt;tipodelegato&gt;' dell'evento che sta implementando all'esterno del progetto mediante &lt;specifier&gt; '&lt;type2&gt;'
Il codice espone un tipo privato all'esterno della classe pubblica.  
  
 **ID errore:** BC30924  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare il tipo `Public` o usare un altro tipo.
