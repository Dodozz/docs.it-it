# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>La dichiarazione 'RemoveHandler' deve terminare con un oggetto 'End RemoveHandler' corrispondente
Una dichiarazione `RemoveHandler` deve terminare con un'istruzione `End RemoveHandler` .  
  
 **ID errore:** BC31116  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Controllare che la dichiarazione `RemoveHandler` termini con un'istruzione `End RemoveHandler` .
