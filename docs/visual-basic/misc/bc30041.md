# <a name="maximum-number-of-errors-has-been-exceeded"></a>È stato superato il numero massimo di errori consentiti
Il numero massimo di errori di compilazione è 100. Il compilatore impone questo limite per ridurre le dimensioni dei report di errore.  
  
 **ID errore:** BC30041  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Esaminare gli altri errori generati dalla compilazione e risolverne il maggior numero possibile. Se il codice sorgente contiene più di 100 errori, è possibile che nella compilazione successiva vengano visualizzati più errori.
