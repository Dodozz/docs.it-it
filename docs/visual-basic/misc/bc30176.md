# <a name="only-one-of-public-private-protected-friend-or-protected-friend-can-be-specified"></a>È possibile specificare solo un elemento tra 'Public', 'Private', 'Protected', 'Friend' e 'Protected Friend'
Esiste un conflitto di accessibilità in un elemento dichiarato. È possibile specificare solo un'accessibilità.  
  
 **ID errore:** BC30176  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Scegliere un'accessibilità e rimuovere le altre parole chiave dalla dichiarazione.
