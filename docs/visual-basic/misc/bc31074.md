# <a name="delegate-constructor-for-type-typename-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Costruttore di delegato per il tipo '\<nomeTipo >', necessario per l'implementazione dei delegati, non può essere trovato
L'assembly che contiene il tipo potrebbe essere danneggiato o non valido.  
  
 **ID errore:** BC31074  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Verificare che l'assembly contenente il tipo sia valido.  
  
2. Ripetere la compilazione del programma, per controllare se l'errore si verifica di nuovo.  
  
3. Se l'errore si ripresenta, salvare il lavoro e riavviare Visual Studio.
