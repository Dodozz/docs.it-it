# <a name="raiseevent-declaration-must-end-with-a-matching-end-raiseevent"></a>La dichiarazione 'RaiseEvent' deve terminare con un oggetto 'End RaiseEvent' corrispondente
Una dichiarazione `RaiseEvent` deve terminare con un'istruzione `End RaiseEvent` .  
  
 **ID errore:** BC31117  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare che la dichiarazione `RaiseEvent` termini con un'istruzione `End RaiseEvent` .
