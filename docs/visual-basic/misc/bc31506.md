# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>'\<nomeTipo >' non può essere usato come attributo perché è dichiarato 'come MustInherit'
Le classi Attribute personalizzate non possono essere dichiarate come `MustInherit`.  
  
 **ID errore:** BC31506  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere il modificatore `MustInherit` dagli attributi personalizzati.
