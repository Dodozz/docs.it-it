# <a name="option-ltspecifiergt-statement-can-only-appear-once-per-file"></a>' Opzione &lt;identificatore&gt;' istruzione può apparire una sola volta per ogni file
Le istruzioni `Option Compare`, `Option Explicit`e `Option Strict` possono essere usate una sola volta nello stesso file di origine.  
  
 **ID errore:** BC30225  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione `Option` duplicata.
