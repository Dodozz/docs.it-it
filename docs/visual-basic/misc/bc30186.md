# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>Le istruzioni 'Using' non sono valide nella finestra di controllo immediato
Le istruzioni`Using` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30186  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere l'istruzione `Using` dalla finestra di controllo **immediato** .
