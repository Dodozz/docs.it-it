# <a name="project-level-conditional-compilation-constant-constantname-is-not-valid-error"></a>Costante di compilazione condizionale a livello di progetto '\<nomecostante >' non è valido: \<errore >
È stata specificata un'opzione di compilazione condizionale non valida per il progetto.  
  
 **ID errore:** BC31030  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare le costanti di compilazione condizionale usate con questo progetto.
