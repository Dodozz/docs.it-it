# <a name="goto-ltlabelnamegt-is-not-valid-because-ltlabelnamegt-is-inside-a-synclock-statement-that-does-not-contain-this-statement"></a>' GoTo &lt;NomeEtichetta&gt;' non è valido perché '&lt;NomeEtichetta&gt;' si trova all'interno di un'istruzione 'SyncLock' che non contiene questa istruzione
Non è consentito creare rami in un blocco `SyncLock` .  
  
 **ID errore:** BC30755  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Ristrutturare il codice inserendo l'etichetta prima del blocco `SyncLock` .
