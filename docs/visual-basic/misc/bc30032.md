# <a name="events-cannot-have-a-return-type"></a>Gli eventi non possono avere un tipo restituito
Gli eventi possono accettare argomenti ma non possono restituire direttamente i valori.  
  
 **ID errore:** BC30032  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il tipo restituito dall'istruzione `Event` .
