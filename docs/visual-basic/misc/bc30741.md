# <a name="lttypename1gt-cannot-be-converted-to-lttypename2gt"></a>«&lt;NomeTipo1&gt;'non può essere convertito in'&lt;in NomeTipo2&gt;»
È stata tentata una conversione del tipo tra due tipi incompatibili.  
  
 **ID errore:** BC30741  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare l'esecuzione del cast dei tipi.
