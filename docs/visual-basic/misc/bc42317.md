# <a name="xml-comment-type-parameter-ltparametergt-does-not-match-a-type-parameter-on-the-corresponding-ltkeywordgt-statement"></a>Parametro di tipo del commento XML &lt;parametri&gt; non corrisponde a un parametro di tipo corrispondente &lt;parola chiave&gt; istruzione
Il parametro del commento XML \<parametro > non corrisponde a un parametro di tipo corrispondente \<parola chiave > istruzione. Il commento XML verrà ignorato.  
  
 Un tag di commenti XML è stato specificato per alcuni, ma non per tutti, i parametri di tipi in un metodo.  
  
 **ID errore:** BC42317  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare i tag di commento XML per tutti i parametri di tipo nel metodo.
