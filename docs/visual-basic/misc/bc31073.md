# <a name="ltname1gt-conflicts-with-a-lttypegt-by-the-same-name-defined-in-ltname2gt"></a>'&lt;name1&gt;' è in conflitto con un &lt;tipo&gt; con lo stesso nome definito in '&lt;nome2&gt;'
È stato rilevato un nome in conflitto con un tipo con lo stesso nome.  
  
 **ID errore:** BC31073  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Dichiarare un nome univoco.  
  
2.  Racchiudere il nome del membro in conflitto tra parentesi quadre.
