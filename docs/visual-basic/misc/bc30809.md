# <a name="wend-statements-are-no-longer-supported-use-end-while-statements-instead"></a>Le istruzioni 'Wend' non sono più supportate. Usare le istruzioni 'End While'
Il costrutto `End While` sostituisce la parola chiave `Wend` .  
  
 **ID errore:** BC30809  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare `End While` per terminare un blocco `While` .
