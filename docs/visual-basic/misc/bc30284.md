# <a name="lttype1gt-lttypenamegt-cannot-be-declared-overrides-because-it-does-not-override-a-lttype1gt-in-a-base-lttype2gt"></a>&lt;type1&gt; »&lt;typename&gt;' non può essere dichiarato come 'Overrides' perché non l'override di un &lt;type1&gt; in una base &lt;type2&gt;
Un'istruzione `Sub`, `Function`o `Property` specifica `Overrides` quando non è presente alcun tipo con lo stesso nome in una classe base.  
  
 **ID errore:** BC30284  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare che il nome del tipo sia stato digitato correttamente.  
  
2.  Rimuovere la parola chiave `Overrides` superflua.
