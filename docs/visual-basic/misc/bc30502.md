# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-property-declaration"></a>Impossibile combinare 'Shared' con '&lt;identificatore&gt;' in una dichiarazione di proprietà
Si è provato a combinare il modificatore `Shared` con un identificatore che non è valido in tale combinazione in una dichiarazione di proprietà.  
  
 **ID errore:** BC30502  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'identificatore.
