# <a name="method-in-a-structure-cannot-be-declared-protected-or-protected-friend"></a>Il metodo in una struttura non può essere dichiarato come 'Protected' o 'Protected Friend'
Un metodo in una struttura è stato dichiarato come `Protected` o `Protected Friend`.  
  
 **ID errore:** BC31067  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare i metodi nelle strutture come `Public` o `Private`.
