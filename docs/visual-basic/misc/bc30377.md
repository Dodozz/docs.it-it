# <a name="method-ltmethodnamegt-is-already-declared-in-interface-ltinterfacenamegt"></a>Metodo '&lt;NomeMetodo&gt;'è già dichiarato nell'interfaccia'&lt;interfacename&gt;'
Un metodo di interfaccia è stato dichiarato una seconda volta e quindi duplicato.  
  
 **ID errore:** BC30377  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la dichiarazione duplicata.
