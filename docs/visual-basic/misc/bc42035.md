# <a name="bad-guid-format"></a>Il formato del GUID non è valido
Un GUID è stato fornito con un formato non corretto.  
  
 **ID errore:** BC42035  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Controllare il formato del GUID.
