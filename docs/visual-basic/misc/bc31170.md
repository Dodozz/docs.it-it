# <a name="character-character-0xlthexadecimalgt-is-not-allowed-in-an-xml-name"></a>Il 'carattere character' (0x&lt;esadecimale&gt;) non è consentito in un nome XML
Un carattere non supportato è incluso nel nome di un valore letterale XML o di una proprietà axis XML.  
  
 **ID errore:** BC31170  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il carattere non supportato dal nome del valore letterale XML o della proprietà axis XML.
