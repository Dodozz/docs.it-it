# <a name="an-embedded-expression-may-not-be-used-here"></a>In questo punto non è possibile usare un'espressione incorporata
Un'espressione incorporata è stata inclusa in un percorso non supportato di un valore letterale XML. Tenere presente che le espressioni incorporate non possono essere usate per specificare un valore dello spazio dei nomi XML.  
  
 **ID errore:** BC31172  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'espressione incorporata dal percorso non supportato.
