# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>Argomento parametro corrispondente '\<nomeparametro >' viene convertito in '\<nomeTipo >'
È stata effettuata una chiamata a un metodo di overload, ma il compilatore non riesce a trovare un metodo che può essere chiamato senza una conversione verso un tipo di dati più piccolo. Una *conversione verso un tipo di dati più piccolo* imposta un valore su un tipo di dati che potrebbe non contenere alcuni dei possibili valori.  
  
 **ID errore:** BC36711
