# <a name="instance-of-restricted-type-lttypenamegt-cannot-be-used-in-a-query-expression"></a>Istanza di tipo limitato '&lt;typename&gt;' non può essere utilizzato in un'espressione di query
Le istanze di tipo limitato, ad esempio <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>e <xref:System.TypedReference>, non possono essere usate in un'espressione lambda o di query.  
  
 **ID errore:** BC36598
