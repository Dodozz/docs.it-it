# <a name="wrong-number-of-type-arguments-passed-to-method-ltgenericprocedurenamegt"></a>Numero errato di argomenti tipo passati al metodo '&lt;nomeroutinegenerica&gt;'
Una routine generica viene chiamata con un numero di argomenti di tipo che non corrisponde al numero di parametri di tipo con cui Ã¨ definita.  
  
 **ID errore:** BC30951  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Fornire un argomento di tipo per ogni parametro di tipo definito per la routine generica.  
  
     -oppure-  
  
-   Chiamare la routine generica senza argomenti di tipo e lasciare che il compilatore tenti di dedurli.
