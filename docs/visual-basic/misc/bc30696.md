# <a name="ltmembername1gt-and-ltmembername2gt-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>«&lt;membername1&gt;'e'&lt;membername2&gt;' non è in rapporto di overload perché si differenziano solo per i tipi di parametri facoltativi
I membri di overload devono presentare tipi di dati diversi per uno o più parametri obbligatori.  
  
 **ID errore:** BC30696  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Definire almeno un parametro obbligatorio come tipo di dati diverso.
