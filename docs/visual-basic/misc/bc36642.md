  
 <span data-ttu-id="56cd3-104">La dichiarazione precedente è valida se il tipo di `n` può essere dedotto.</span><span class="sxs-lookup"><span data-stu-id="56cd3-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="56cd3-105">Se ad esempio si assegna l'espressione lambda precedente a un delegato della funzione, `Del`:</span><span class="sxs-lookup"><span data-stu-id="56cd3-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>

 <span data-ttu-id="56cd3-106">A questo punto il tipo di `n` può essere dedotto dal parametro `p`:</span><span class="sxs-lookup"><span data-stu-id="56cd3-106">Now the type of `n` can be inferred from parameter `p`:</span></span>
