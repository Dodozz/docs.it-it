# <a name="ltspecifier1gt-lttypegt-cannot-inherit-from-a-ltspecifier2gt-lttypegt-because-it-expands-the-access-of-the-base-lttypegt"></a>&lt;specifier1&gt; &lt;tipo&gt; non può ereditare da una &lt;specifier2&gt; &lt;tipo&gt; perché espande l'accesso della base &lt;tipo&gt;
Si è provato a far ereditare una classe pubblica da una classe base contrassegnata come `Private` o `Friend`.  
  
 **ID errore:** BC30509  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare la classe base `Public` o dichiarare la classe che eredita `Private` o `Friend`.
