# <a name="comma-or--expected-procedure-parameter-list"></a>Prevista virgola o ')' (elenco dei parametri di routine)
Un elenco di parametri contiene un elemento imprevisto. Usare le virgole (`,`) per separare i parametri in un elenco di questo tipo e usare una parentesi di chiusura (`)`) per concludere l'elenco.  
  
 **ID errore:** BC30213  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Aggiungere una virgola e proseguire l'elenco dei parametri di tipo oppure aggiungere una parentesi di chiusura per terminare l'elenco.
