# <a name="named-argument-argumentname-specified-multiple-times"></a>L'argomento denominato '\<NomeArgomento >' è specificato più volte
Un argomento denominato è stato specificato più volte.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere tutte le specifiche tranne una.
