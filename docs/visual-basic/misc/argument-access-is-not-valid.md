# <a name="argument-access-is-not-valid-valid-values-for-output-mode-are-openaccesswrite-and-openaccessdefault"></a>L'argomento 'Access' non è valido. I valori validi per la modalità Output sono 'OpenAccess.Write' e 'OpenAccess.Default'
Un valore access non è valido in modalità Output.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Modificare il valore in `OpenAccess.Write` o `OpenAccess.Default`.
