# <a name="too-few-type-arguments"></a>Argomenti di tipo insufficienti
È stato richiamato un metodo generico con meno argomenti di tipo rispetto ai parametri di tipo.  
  
 Quando si richiama un metodo generico, è necessario fornire un argomento di tipo per ogni parametro di tipo definito dal metodo.  
  
 **ID errore:** BC36578  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Aggiungere gli argomenti di tipo all'elenco di argomenti di tipo in modo che ne esista uno per ogni parametro di tipo del metodo generico che si sta chiamando.
