# <a name="value-ltvaluename1gt-cannot-be-converted-to-ltvaluename2gt"></a>Valore '&lt;valuename1&gt;'non può essere convertito in'&lt;valuename2&gt;'
È stata tentata una conversione del tipo tra due tipi incompatibili.  
  
 **ID errore:** BC30742  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare l'esecuzione del cast dei tipi.
