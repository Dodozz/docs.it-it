# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>Valore '\<valuename1 >' non può essere convertito in '\<valuename2 >'
È stata tentata una conversione del tipo tra due tipi incompatibili.  
  
 **ID errore:** BC30742  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare l'esecuzione del cast dei tipi.
