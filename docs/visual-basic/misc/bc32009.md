# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Le istruzioni della dichiarazione di metodo devono essere le prime in una riga logica
Un'istruzione `Function` o `Sub` segue un separatore di istruzioni due punti (:). `Function` o `Sub` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC32009
