# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Se uno o più parametri sono tipizzati, allora devono esserlo tutti i parametri
In una funzione sono stati usati più tipi diversi di dichiarazioni di tipo.  
  
 **ID errore:** BC30529  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che tutti i parametri siano tipizzati in modo esplicito.
