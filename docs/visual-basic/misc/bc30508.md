# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<nome >' non può esporre il tipo \<type1 > nelle \<identificatore > \<type2 > '\<nomeTipo >' tramite \<nome >
Si è provato a esporre un tipo privato al di fuori della classe pubblica `MyClass`.  
  
 **ID errore:** BC30508  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Dichiarare il tipo `Public`.
