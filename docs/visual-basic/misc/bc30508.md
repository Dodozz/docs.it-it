# <a name="ltnamegt-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-lttypenamegt-through-ltnamegt"></a>'&lt;name&gt;' non può esporre il tipo &lt;type1&gt; nel &lt;identificatore&gt; &lt;type2&gt; '&lt;typename&gt;' tramite &lt;nome&gt;
Si è provato a esporre un tipo privato al di fuori della classe pubblica `MyClass`.  
  
 **ID errore:** BC30508  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare il tipo `Public`.
