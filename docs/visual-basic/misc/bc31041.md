# <a name="statement-is-not-valid-in-an-interface"></a>L'istruzione non è valida in un'interfaccia
Nell'interfaccia è stata rilevata un'istruzione non valida.  
  
 **ID errore:** BC31041  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che le istruzioni nell'interfaccia definiscano gli elementi, ma non forniscano l'implementazione.
