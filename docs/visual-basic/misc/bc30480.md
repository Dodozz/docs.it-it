# <a name="shared-sub-new-cannot-be-declared-ltspecifiergt"></a>Condiviso 'Sub New' non può essere dichiarato '&lt;identificatore&gt;'
Si è provato a usare un identificatore in una dichiarazione `Shared Sub New` . Questa operazione non è valida.  
  
 **ID errore:** BC30480  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'identificatore.
