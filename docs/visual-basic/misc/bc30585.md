# <a name="event-eventname-cannot-be-handled-because-it-is-not-accessible-from-name"></a>Evento '\<NomeEvento >' non può essere gestita perché non è accessibile da '\<nome >'
Si è provato a gestire un evento che non è accessibile. Ad esempio, se una variabile `Handles` è condivisa, anche il metodo che gestisce gli eventi deve essere condiviso.  
  
 **ID errore:** BC30585  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare che l'evento sia accessibile.
