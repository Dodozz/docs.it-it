# <a name="get-not-supported-write-only-property"></a>Get non supportato (proprietà di sola scrittura)
Non è possibile leggere una proprietà perché è di sola scrittura.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare la proprietà e determinare in quali condizioni è possibile impostarla.  
  
2.  Rimuovere il riferimento alla proprietà.
