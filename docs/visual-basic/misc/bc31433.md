# <a name="method-ltmethodnamegt-cannot-be-declared-partial-because-only-one-method-ltmethodnamegt-can-be-marked-partial"></a>Metodo '&lt;NomeMetodo&gt;' non può essere dichiarato 'Partial' perché un solo metodo '&lt;NomeMetodo&gt;' può essere contrassegnato come 'Partial'
È possibile avere due dichiarazioni di metodo parziale con lo stesso nome e firma.  
  
 **ID errore:** BC31433  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Modificare il nome o la firma di una delle dichiarazioni di metodo parziale.
