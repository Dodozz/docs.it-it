# <a name="type-characters-cannot-be-used-in-anonymous-type-declarations"></a><span data-ttu-id="37d04-102">Non è possibile usare i caratteri tipo nelle dichiarazioni di tipo anonimo</span><span class="sxs-lookup"><span data-stu-id="37d04-102">Type characters cannot be used in anonymous type declarations</span></span>
<span data-ttu-id="37d04-103">Quando si dichiara un'istanza di un tipo anonimo, non è possibile usare un carattere tipo in un nome di proprietà.</span><span class="sxs-lookup"><span data-stu-id="37d04-103">You cannot use a type character on a property name when you declare an instance of an anonymous type.</span></span> <span data-ttu-id="37d04-104">Il tipo di dati della proprietà viene dedotto dal valore assegnato a esso.</span><span class="sxs-lookup"><span data-stu-id="37d04-104">The data type of the property is inferred from the value assigned to it.</span></span> <span data-ttu-id="37d04-105">Ad esempio, le dichiarazioni seguenti non sono valide.</span><span class="sxs-lookup"><span data-stu-id="37d04-105">For example, the following declarations are not valid.</span></span>  
  
```vb  
'' Not valid.  
'Dim anon1 = New With {.ID$ = "abc"}  
'Dim anon2 = New With {.ID$ = 42}  
```  
  
 <span data-ttu-id="37d04-106">**ID errore:** BC36560</span><span class="sxs-lookup"><span data-stu-id="37d04-106">**Error ID:** BC36560</span></span>
