# <a name="xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>Tag del commento XML '\<tag >' presente più volte con attributi identici nello stesso blocco di commento XML
Tag del commento XML \<tag > presente più volte con attributi identici nello stesso blocco di commento XML. Il commento XML verrà ignorato.  
  
 Il commento XML contiene più tag con attributi identici.  
  
 **ID errore:** BC42305  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere i tag duplicati.
