# <a name="assembly-ltassemblynamegt-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Assembly '&lt;assemblyname&gt;' non può essere creata perché il percorso è più lungo di 259 caratteri
Il percorso per questo assembly supera la lunghezza massima di 259 caratteri.  
  
 **ID errore:** BC31518  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Abbreviare il percorso usando nomi più brevi o selezionando una gerarchia di file ridotta.
