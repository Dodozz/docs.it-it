# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>Le istruzioni di assegnazione 'Let' e 'Set' non sono più supportate
`Let` e `Set` non possono essere usati nelle assegnazioni.  
  
 **ID errore:** BC30807  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Omettere `Let` e `Set` dalle istruzioni di assegnazione, ad esempio:  
  
     `X = 2`
