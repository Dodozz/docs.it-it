# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>Le istruzioni 'Throw' non sono valide nella finestra di controllo immediato
Le istruzioni`Throw` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30725  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere le istruzioni `Throw` dal codice di debug.
