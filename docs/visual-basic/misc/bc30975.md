# <a name="for-loop-control-variable-already-in-use-by-an-enclosing-for-loop"></a>Variabile di controllo del ciclo 'For' già in uso da un ciclo 'For' di inclusione
Tutti i cicli annidati `For` devono usare variabili di controllo univoche.  
  
 **ID errore:** BC30975  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare uno dei nomi della variabile di controllo.
