# <a name="lttypenamegt-is-a-type-in-ltcontainernamegt-and-cannot-be-used-as-an-expression"></a>«&lt;nomeTipo&gt;'è un tipo in'&lt;containername&gt;' e non può essere usato come espressione
Uno dei nomi in uso è già stato dichiarato come nome di un tipo e non può essere usato in un'espressione.  
  
 **ID errore:** BC30691  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Assegnare un nome univoco al membro.  
  
-   Racchiudere il nome del membro tra parentesi quadre.
