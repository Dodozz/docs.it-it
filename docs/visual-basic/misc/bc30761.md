# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>Progetto '\<NomeProgetto >' non può fare riferimento a progetto '\<NomeProgetto >' perché '\<NomeProgetto >' direttamente o indirettamente fa riferimento a '\<NomeProgetto >'
Un progetto include un riferimento a un secondo progetto. Il secondo progetto include a sua volta un riferimento al progetto contenente un riferimento a esso.  
  
 **ID errore:** BC30761  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere uno dei riferimenti.
