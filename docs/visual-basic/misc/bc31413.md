# <a name="ltmembergt-implicitly-defined-for-lteventnamegt-cannot-shadow-a-mustoverride-method-in-the-base-ltclassgt-ltclassnamegt"></a>«&lt;membro&gt;', definita in modo implicito per'&lt;eventname&gt;', non può nascondere un metodo 'MustOverride' nella base &lt;classe&gt; '&lt;classname&gt;»
L'evento specificato dichiara in modo implicito un membro con lo stesso nome di un metodo dichiarato con il modificatore `MustOverride` .  
  
 **ID errore:** BC31413  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il modificatore `MustOverride` dal metodo nella classe base o assegnare un nome univoco alla proprietà o al metodo.
