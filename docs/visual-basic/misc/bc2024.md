# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>Il riferimento all'assembly predefinito '\<riferimento >' già ha stato automaticamente aggiunto; verrà ignorato
Si è provato ad aggiungere il riferimento all'assembly predefinito, ma l'assembly è già stato aggiunto, pertanto il tentativo viene ignorato.  
  
 **ID errore:** BC2024  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere il codice superfluo.
