# <a name="lambda-expression-cannot-be-converted-to-lttypenamegt-because-type-lttypenamegt-is-declared-mustinherit-and-cannot-be-created"></a>Espressione lambda non può essere convertita in '&lt;nomeTipo&gt;' in quanto tipo di '&lt;typename&gt;' è dichiarato 'come MustInherit' e non è possibile creare
Un'istruzione tenta di convertire un'espressione lambda in un tipo che può essere solo una classe base e non può essere usato per creare un'istanza.  
  
 **ID errore:** BC36626
