# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>Espressione lambda non può essere convertita in '\<nomeTipo >' perché tipo '\<nomeTipo >' è dichiarato 'come MustInherit' e non può essere creato
Un'istruzione tenta di convertire un'espressione lambda in un tipo che può essere solo una classe base e non può essere usato per creare un'istanza.  
  
 **ID errore:** BC36626
