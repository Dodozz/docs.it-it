# <a name="sub-main-is-declared-more-than-once-in-namespace-message"></a>'Sub Main' è dichiarato più volte '\<dello spazio dei nomi >': \<messaggio >
`Sub Main` può essere dichiarato solo all'interno di uno spazio dei nomi.  
  
 **ID errore:** BC30738  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che nel progetto sia presente una sola routine `Sub Main` .
