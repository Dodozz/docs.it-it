# <a name="propertyname1-cannot-override-propertyname2-because-they-differ-by-readonly-or-writeonly"></a>«\<nomeproprietà1 >' non è possibile eseguire l'override '\<nomeproprietà2 >' perché si differenziano per 'ReadOnly' o 'WriteOnly'
Si è provato a eseguire l'override di una proprietà con una seconda proprietà che è diversa dalla prima solo per `ReadOnly` o `WriteOnly`.  
  
 **ID errore:** BC30362  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che le proprietà si differenzino per altri elementi oltre a `ReadOnly` e `WriteOnly`.
