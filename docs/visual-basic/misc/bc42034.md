# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>Il nome file è già dichiarato con un valore diverso per checksum e GUID
Si è provato a dichiarare un file, ma il file esiste già con un valore GUID e checksum diverso.  
  
 **ID errore:** BC42034  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Controllare il nome e il percorso del file per assicurarsi che siano corretti.  
  
- Eliminare il file ridondante se non è necessario.
