# <a name="xml-comment-parameter-ltparametergt-does-not-match-a-parameter-on-the-corresponding-ltkeywordgt-statement"></a>Il parametro del commento XML '&lt;parametri&gt;' non corrisponde a un parametro corrispondente &lt;parola chiave&gt; istruzione
Il parametro del commento XML \<parametro > non corrisponde a un parametro corrispondente \<parola chiave > istruzione. Il commento XML verrà ignorato.  
  
 Un tag di commenti XML è stato specificato per alcuni, ma non per tutti, i parametri in un metodo.  
  
 **ID errore:** BC42307  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare il tag di commento XML per tutti i parametri nel metodo.
