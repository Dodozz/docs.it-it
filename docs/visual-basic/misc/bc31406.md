# <a name="ltmethodnamegt-cannot-override-a-method-that-has-been-shadowed"></a>«&lt;NomeMetodo&gt;' non è possibile eseguire l'override di un metodo nascosto
Un metodo ha tentato di eseguire l'override di un metodo che nasconde un altro membro.  
  
 **ID errore:** BC31406  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare il modificatore `Shadows` per eseguire l'override dei membri nascosti.
