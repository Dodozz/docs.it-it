# <a name="error-embedding-win32-manifest-manifest-file-file-cannot-be-found"></a>Errore durante l'incorporamento del manifesto Win32: File manifesto \<file > non è stata trovata
Il file specificato per l'opzione del compilatore `/win32manifest` non è stato trovato.  
  
 **ID errore:** BC31192  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Fornire un nome file valido e un percorso per l'opzione del compilatore `/win32manifest` .
