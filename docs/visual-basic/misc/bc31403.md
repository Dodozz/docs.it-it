# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>Degli alias di Imports&lt;name&gt;'è in conflitto con'&lt;nome2&gt;' dichiarato nello spazio dei nomi radice
Il nome dell'alias usato nell'istruzione `Imports` è in conflitto con un altro nome dichiarato nello spazio dei nomi radice.  
  
 **ID errore:** BC31403  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Scegliere un nome univoco per l'alias usato dall'istruzione `Imports` .
