# <a name="security-attribute-ltattributenamegt-cannot-be-applied-to-a-module"></a>Attributo di sicurezza '&lt;attributename&gt;' non può essere applicato a un modulo
Common Language Runtime non accetta l'attributo di sicurezza in un modulo.

**ID errore:** BC30131

## <a name="to-correct-this-error"></a>Per correggere l'errore

- Rimuovere l'attributo dalla dichiarazione.
