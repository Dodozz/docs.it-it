# <a name="xml-comment-tag-lttaggt-is-not-permitted-on-a-ltelementgt-language-element"></a>Tag del commento XML '&lt;tag&gt;' non è consentito in un '&lt;elemento&gt;' elemento di linguaggio
Tag del commento XML \<tag > non è consentito in un '\<elemento >' elemento. Il commento XML verrà ignorato.  
  
 Un tag di commento XML non è supportato dall'elemento di linguaggio al quale viene applicato.  
  
 **ID errore:** BC42306  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il tag di commento XML non supportato.
