# <a name="argument-argumentname-cannot-be-converted-to-a-numeric-value"></a>Argomento '\<NomeArgomento >' non può essere convertito in valore numerico
È stata tentata una conversione di una variabile, ad esempio una stringa, in un valore numerico.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Trovare una conversione significativa.
