# <a name="argument-ltargumentnamegt-cannot-be-converted-to-type-lttypenamegt"></a>Argomento '&lt;NomeArgomento&gt;'non può essere convertito nel tipo'&lt;nomeTipo&gt;'
Una conversione tenta di modificare un argomento a un tipo non valido.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il tipo in un tipo valido.
