# <a name="ltmodifiergt-is-not-valid-on-a-local-constant-declaration"></a>«&lt;modificatore&gt;' non è valido in una dichiarazione di costante locale
Un'istruzione `Const` all'interno di una routine contiene una parola chiave non valida, ad esempio `Overloads`.  
  
 **ID errore:** BC30246  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere la parola chiave non valida dall'istruzione `Const` .
