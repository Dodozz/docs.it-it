# <a name="class-classname-could-not-be-created-error"></a>Della classe\<NomeClasse >' non è stato possibile creare: \<errore >
Si è verificato un errore nell'inizializzazione di una classe. L'installazione di [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] potrebbe essere danneggiata.  
  
 **ID errore:** BC32400  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Esaminare il messaggio di errore tra virgolette e intraprendere l'azione appropriata.  
  
2. Reinstallare [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] e ricompilare il programma.  
  
3. Se l'errore si ripete, reinstallare il compilatore Visual Basic.  
  
4. Se l'errore persiste dopo la reinstallazione, raccogliere informazioni sulla situazione contingente e informare il Servizio supporto tecnico Microsoft.
