# <a name="module-statement-must-end-with-a-matching-end-module"></a>L'istruzione 'Module' deve terminare con un oggetto 'End Module' corrispondente
I blocchi del modulo devono terminare con un'istruzione `End Module` .  
  
 **ID errore:** BC30625  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere un'istruzione `End Module` alla fine del blocco `Module` .
