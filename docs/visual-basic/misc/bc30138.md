# <a name="unable-to-create-temp-file-in-path-ltfilenamegt-lterror-messagegt"></a>Non è possibile creare file temporaneo nel percorso '&lt;nomefile&gt;': &lt;messaggio di errore&gt;
Il compilatore Visual Basic chiama Assembly Linker (Al.exe, definito anche Alink) per generare un assembly con un manifesto. Il linker ha rilevato un errore durante la creazione di un file o durante la scrittura di una risorsa in memoria in un file. Potrebbe trattarsi di un problema di configurazione.  
  
 **ID errore:** BC30138  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Esaminare il messaggio di errore riportato e vedere l'argomento [Assembly Linker (Al.exe)](../../framework/tools/al-exe-assembly-linker.md) per spiegazioni e suggerimenti aggiuntivi.  
  
2.  Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
