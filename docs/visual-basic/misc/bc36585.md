# <a name="ltelementnamegt-is-not-a-parameter-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>«&lt;elementname&gt;'non è un parametro del metodo di estensione'&lt;NomeMetodo&gt;'definito '&lt;typename&gt;»
Una chiamata di routine fornisce un argomento aggiuntivo non definito dal metodo di estensione.  
  
 **ID errore:** BC36585  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'argomento superfluo dall'elenco di argomenti.
