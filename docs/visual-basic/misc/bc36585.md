# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>'\<NomeElemento >' non è un parametro del metodo di estensione '\<NomeMetodo >' definito in '\<nomeTipo >'
Una chiamata di routine fornisce un argomento aggiuntivo non definito dal metodo di estensione.  
  
 **ID errore:** BC36585  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere l'argomento superfluo dall'elenco di argomenti.
