# <a name="variable-cannot-be-initialized-with-non-array-type-ltelementnamegt"></a>Impossibile inizializzare la variabile con tipo non matrice '&lt;elementname&gt;'
Una variabile dichiarata come matrice deve essere inizializzata con un valore di matrice.  
  
```  
' Not valid.  
' The following line causes this error when executed with Option Strict off.  
' Dim arrayVar1() = 10  
```  
  
 **ID errore:** BC36536  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Inizializzare la variabile di matrice con un valore di matrice:  
  
    ```
    ' With Option Strict off.  
    Dim arrayVar2() = {1, 2, 3}  
    ' With Option Strict on.  
    Dim arrayVar3() As Integer = {1, 2, 3}  
    ```
