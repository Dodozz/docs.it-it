# <a name="paramarray-parameters-must-be-declared-byval"></a>I parametri ParamArray devono essere dichiarati come 'ByVal'
I parametri`ParamArray` non possono usare il modificatore `ByRef` .  
  
 **ID errore:** BC30667  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare i parametri `ParamArray` con il modificatore `ByVal` .
