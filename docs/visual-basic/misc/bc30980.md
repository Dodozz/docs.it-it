    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="cd634-106">Ciclo`For` </span><span class="sxs-lookup"><span data-stu-id="cd634-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
-   <span data-ttu-id="cd634-107">Ciclo`For Each` </span><span class="sxs-lookup"><span data-stu-id="cd634-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```
