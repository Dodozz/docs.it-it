# <a name="type-of-variablename-cannot-be-inferred-from-an-expression-containing-variablename"></a><span data-ttu-id="f26ba-102">Tipo di '\<nomevariabile >' non è possibile dedurre da un'espressione contenente '\<nomevariabile >'</span><span class="sxs-lookup"><span data-stu-id="f26ba-102">Type of '\<variablename>' cannot be inferred from an expression containing '\<variablename>'</span></span>
<span data-ttu-id="f26ba-103">Il compilatore non può dedurre il tipo di dati di una variabile se la variabile viene usata per stabilire il relativo valore iniziale nella dichiarazione.</span><span class="sxs-lookup"><span data-stu-id="f26ba-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="f26ba-104">Ad esempio, con `Option Infer` impostato su `On`, negli esempi seguenti non vengono compilati:</span><span class="sxs-lookup"><span data-stu-id="f26ba-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
- <span data-ttu-id="f26ba-105">Dichiarazioni</span><span class="sxs-lookup"><span data-stu-id="f26ba-105">Declarations</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
- <span data-ttu-id="f26ba-106">Ciclo`For` </span><span class="sxs-lookup"><span data-stu-id="f26ba-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
- <span data-ttu-id="f26ba-107">Ciclo`For Each` </span><span class="sxs-lookup"><span data-stu-id="f26ba-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="f26ba-108">**ID errore:** BC30980</span><span class="sxs-lookup"><span data-stu-id="f26ba-108">**Error ID:** BC30980</span></span>
