# <a name="argument-access-is-not-valid-valid-values-for-input-mode-are-openaccessread-and-openaccessdefault"></a>L'argomento 'Access' non è valido. I valori validi per la modalità Input sono 'OpenAccess.Read' e 'OpenAccess.Default'
Un valore access non è valido in modalità `Input` .  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Modificare il valore in `OpenAccess.Read` o `OpenAccess.Default`.
