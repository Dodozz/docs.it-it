# <a name="the-remote-server-machine-does-not-exist-or-is-unavailable-visual-basic"></a>Il computer server remoto non esiste o non è disponibile (Visual Basic)
Si sta provando a connettersi a un server remoto che non è disponibile o non esiste. Questo errore può essere causato dalla perdita della connessione di rete.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Ristabilire la connessione alla rete e riprovare.
