# <a name="ltmodifiergt-is-not-valid-on-a-delegate-declaration"></a>«&lt;modificatore&gt;' non è valido in una dichiarazione Delegate
Si è provato a usare un modificatore, ad esempio `Shared`, in una dichiarazione di delegato. I delegati sono oggetti usati per chiamare i metodi di altri oggetti tramite la definizione di un costruttore a cui viene passata la specifica di un metodo di oggetto. Non è consentito specificare un modificatore in una dichiarazione di delegato.  
  
 **ID errore:** BC30385  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere il modificatore.
