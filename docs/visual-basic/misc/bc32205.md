# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>La lunghezza del nome della variabile WithEvents non può superare i 1019 caratteri
Il nome o l'identificatore di tutte le variabili dichiarate con `WithEvents` è limitato a 1019 caratteri.  
  
 **ID errore:** BC32205  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Ridurre la lunghezza dell'identificatore.
