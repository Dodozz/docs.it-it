# <a name="partial-method-ltmethodnamegt-cannot-use-the-implements-keyword"></a>Metodo parziale '&lt;NomeMetodo&gt;' non è possibile usare la parola chiave 'Implements'
Una definizione di metodo parziale non può implementare un'interfaccia o un membro di interfaccia.  
  
 **ID errore:** BC31439  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere l'istruzione `Implements` dalla definizione del metodo parziale.
