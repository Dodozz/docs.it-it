# <a name="ltmodifiergt-is-not-valid-on-an-event-declaration"></a>«&lt;modificatore&gt;' non è valido in una dichiarazione di evento
Un'istruzione `Event` contiene una parola chiave non valida, ad esempio `ReadOnly`.  
  
 **ID errore:** BC30243  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere la parola chiave non valida dall'istruzione `Event` .
