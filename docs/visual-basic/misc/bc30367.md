# <a name="class-ltclassnamegt-cannot-be-indexed-because-it-has-no-default-property"></a>Della classe&lt;classname&gt;' non può essere indicizzata perché non contiene proprietà predefinite
Non è stato possibile specificare una proprietà predefinita per una classe.  
  
 **ID errore:** BC30367  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare una proprietà come proprietà predefinita iniziando la relativa dichiarazione con la parola chiave `Default` .
