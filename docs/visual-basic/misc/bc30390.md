# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>«\<metodoqualificato >' non è accessibile in questo contesto perché è '\<modificatore >»
Si è provato ad accedere a un metodo che non è accessibile in questo contesto a causa di un modificatore che è stato applicato a tale metodo.  
  
 **ID errore:** BC30390  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere il modificatore.
