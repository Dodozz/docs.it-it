# <a name="keyword-does-not-name-a-type"></a>La parola chiave non indica un tipo
La parola chiave viene visualizzata nel punto in cui Ã¨ richiesto il nome di un tipo. Il contesto chiama un tipo di dati elementare o un nome di elemento dichiarato che rappresenta un tipo.  
  
 **ID errore:** BC30180
