# <a name="ltmodifiergt-is-not-valid-on-a-method-declaration"></a>«&lt;modificatore&gt;' non è valido in una dichiarazione di metodo
Un'istruzione `Function` o `Sub` contiene una parola chiave non valida.  
  
 **ID errore:** BC30242  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la parola chiave non valida dall'istruzione `Function` o `Sub` .
