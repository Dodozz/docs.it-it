# <a name="string-length-exceeds-maximum-length-of-32767-characters-for-filesystem-apis"></a>La lunghezza della stringa supera il numero massimo di 32767 caratteri ammesso dalle API 'FileSystem'
La lunghezza di una stringa supera la lunghezza massima di 32767 caratteri.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Abbreviare la stringa.
