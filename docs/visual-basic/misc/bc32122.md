# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-from-which-the-interface-interfacename4-inherits-for-some-type-arguments"></a><span data-ttu-id="778ca-102">Non è possibile ereditare l'interfaccia '\<interfacename1 >' perché l'interfaccia '\<interfacename2 >' da cui eredita potrebbe essere identica all'interfaccia '\<interfacename3 >' da cui l'interfaccia '\< interfacename4 >' eredita per alcuni argomenti tipo</span><span class="sxs-lookup"><span data-stu-id="778ca-102">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' from which the interface '\<interfacename4>' inherits for some type arguments</span></span>
<span data-ttu-id="778ca-103">Un'interfaccia generica eredita da due o più interfacce generiche e due delle ereditarietà potrebbero essere in conflitto per determinati valori di argomenti di tipo.</span><span class="sxs-lookup"><span data-stu-id="778ca-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="778ca-104">Le istruzioni seguenti possono generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="778ca-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface interfaceY(Of w)  
    Inherits interfaceA(Of w)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceX(Of t1), interfaceY(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="778ca-105">Se l'interfaccia `derivedInterface` è costruita o implementata specificando lo stesso tipo sia in `t1` che in `t2`, deve ereditare due versioni di `interfaceA` con argomenti di tipo identici.</span><span class="sxs-lookup"><span data-stu-id="778ca-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="778ca-106">In questo modo si produrrebbe un'ambiguità sulla versione a cui accedere.</span><span class="sxs-lookup"><span data-stu-id="778ca-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="778ca-107">**ID errore:** BC32122</span><span class="sxs-lookup"><span data-stu-id="778ca-107">**Error ID:** BC32122</span></span>
