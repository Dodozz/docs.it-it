# <a name="mybase-cannot-be-used-with-method-ltmethodnamegt-because-it-is-declared-mustoverride"></a>'MyBase' non può essere usato con metodo '&lt;NomeMetodo&gt;' perché è dichiarato come 'MustOverride'
Si è provato a usare `MyBase` con un metodo dichiarato come `MustOverride`.  
  
 **ID errore:** BC30399  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la dichiarazione `MustOverride` .
