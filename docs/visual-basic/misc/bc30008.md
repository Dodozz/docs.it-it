# <a name="reference-required-to-module-modulename-containing-the-base-class-classname"></a>Necessario un riferimento al modulo '\<NomeModulo >' contenente la classe base\<NomeClasse >'
Necessario un riferimento al modulo '\<NomeModulo >' contenente la classe base\<NomeClasse >'. Aggiungerne uno al progetto.  
  
 La classe è definita in un modulo a cui non si fa riferimento direttamente nel progetto. Il compilatore Visual Basic richiede un riferimento per evitare ambiguità nel caso in cui la classe è definita in più di un modulo.  
  
 **ID errore:** BC30008  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Includere il nome del modulo senza riferimento nei riferimenti del progetto.
