# <a name="error-embedding-win32-manifest-manifest"></a>Errore durante l'incorporamento del manifesto Win32: \<manifesto >
Il file specificato per l'opzione del compilatore `/win32manifest` non può essere incorporato. Ciò può essere dovuto a privilegi di sicurezza non sufficienti per il file manifesto.  
  
 **ID errore:** BC31191  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Verificare che l'identità di esecuzione del comando del compilatore di Visual Basic abbia accesso al file manifesto Win32.
