# <a name="end-function-must-be-the-first-statement-on-a-line"></a>'End Function' deve essere la prima istruzione in una riga
Un'istruzione `End Function` segue un separatore di istruzioni due punti (:). `End Function` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC32032
