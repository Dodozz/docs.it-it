# <a name="end-set-must-be-the-first-statement-on-a-line"></a>'End Set' deve essere la prima istruzione in una riga
Un'istruzione `End Set` segue un separatore di istruzioni due punti (:). `End Set` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC32034
