# <a name="anonymous-type-member-property-ltpropertynamegt-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-ltpropertynamegt-is-not-yet-established"></a>Proprietà di membro di tipo anonimo '&lt;propertyname&gt;' non può essere usato per dedurre il tipo di un'altra proprietà membro perché il tipo di '&lt;propertyname&gt;' non è ancora stabilito
Fino a quando non viene stabilito il tipo di una proprietà di tipo anonimo, non può essere usata per stabilire il tipo di un'altra proprietà. Ad esempio, nella dichiarazione seguente `.IDName = .LastName` non è valido perché `.LastName` non è ancora stato inizializzato.  
  
```  
' Not valid.   
' Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}   
```  
  
 **ID errore:** BC36559  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Stabilire il tipo della proprietà prima di usarla per inizializzare un'altra proprietà.  
  
    ```  
    Dim anon2 = New With {Key .LastName = "Jones", Key .IDName = .LastName}  
    ```
