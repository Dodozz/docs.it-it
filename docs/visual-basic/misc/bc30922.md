# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="0903d-102">«\<nometipoderivato >' non può ereditare \<tipo > '\<nometipobasecreato >' perché espande l'accesso di tipo '\<nometipointerno >' all'esterno dell'assembly</span><span class="sxs-lookup"><span data-stu-id="0903d-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>
<span data-ttu-id="0903d-103">Una classe o un'interfaccia derivata prova a espandere il livello di accesso di un tipo ristretto usandolo come argomento di tipo per una classe o un'interfaccia base.</span><span class="sxs-lookup"><span data-stu-id="0903d-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="0903d-104">Il codice seguente può generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="0903d-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="0903d-105">Il codice all'esterno dell'assembly non può accedere a `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="0903d-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="0903d-106">L'accesso all'oggetto `derivedClass` , tuttavia, è consentito a qualsiasi codice che può farvi riferimento.</span><span class="sxs-lookup"><span data-stu-id="0903d-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="0903d-107">Di conseguenza, `derivedClass` non può ereditare da `baseClass` se usa `restrictedStructure` come argomento di tipo perché altrimenti `restrictedStructure` verrebbe esposta al codice in qualsiasi assembly.</span><span class="sxs-lookup"><span data-stu-id="0903d-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="0903d-108">**ID errore:** BC30922</span><span class="sxs-lookup"><span data-stu-id="0903d-108">**Error ID:** BC30922</span></span>

     <span data-ttu-id="0903d-111">-oppure-</span><span class="sxs-lookup"><span data-stu-id="0903d-111">-or-</span></span>
