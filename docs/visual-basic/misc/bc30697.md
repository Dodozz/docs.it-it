# <a name="ltmembername1gt-cannot-override-ltmembername2gt-because-they-differ-by-the-types-of-optional-parameters"></a>«&lt;membername1&gt;'non è possibile eseguire l'override'&lt;membername2&gt;' perché si differenziano per i tipi di parametri facoltativi
I membri di overload devono presentare tipi di dati diversi per uno o più parametri obbligatori.  
  
 **ID errore:** BC30697  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Definire almeno un parametro obbligatorio come tipo di dati diverso.
