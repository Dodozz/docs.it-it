# <a name="delegate-in-an-interface-cannot-be-declared-ltspecifiergt"></a>Delegato in un'interfaccia non può essere dichiarato come '&lt;identificatore&gt;'
Il modificatore specificato con un delegato dichiarato nell'interfaccia non è valido.  
  
 **ID errore:** BC31068  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere i modificatori come `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`e `Overridable`.
