# <a name="variant-is-no-longer-a-supported-type-use-the-object-type-instead"></a>Il tipo 'Variant' non è più supportato. Usare il tipo 'Object'
Il tipo di dati `Object` sostituisce `Variant`.  
  
 **ID errore:** BC30804  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Dichiarare l'elemento come `Object`.
