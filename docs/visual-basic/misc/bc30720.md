# <a name="on-error-statements-are-not-valid-in-the-immediate-window"></a>Le istruzioni 'On Error' non sono valide nella finestra di controllo immediato
Le istruzioni`On Error` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30720  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere le istruzioni `On Error` dal codice di debug.
