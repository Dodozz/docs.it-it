# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-without-a-widening-conversion-list"></a>Overload non accessibile '\<NomeMetodo >' può essere chiamato con questi argomenti senza una conversione verso un: \<elenco >
Un metodo di overload è stato chiamato, ma nessun metodo può essere associato all'elenco di argomenti forniti senza una conversione verso un tipo di dati più grande.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Specificare `Option Strict Off`.  
  
- Modificare gli argomenti in modo che corrispondano a una delle firme del metodo di overload.
