# <a name="security-attribute-ltattributenamegt-is-not-valid-lterrorgt"></a>Attributo di sicurezza '&lt;attributename&gt;' non è valido: &lt;errore&gt;
Common Language Runtime non accetta l'attributo in un contesto di sicurezza.

**ID errore:** BC30128

## <a name="to-correct-this-error"></a>Per correggere l'errore

1. Rimuovere l'attributo dalla dichiarazione o sostituirlo con un attributo valido per il contesto.

2. Esaminare il messaggio di errore tra virgolette e intraprendere l'azione appropriata.
