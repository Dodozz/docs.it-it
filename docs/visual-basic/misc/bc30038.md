# <a name="option-strict-on-prohibits-operands-of-type-object-for-operator-ltoperatornamegt"></a>Option Strict On non ammette operandi di tipo Object per l'operatore '&lt;nomeoperatore&gt;'
Gli unici operatori definiti per le variabili oggetto sono `Is` e `TypeOf...Is`. Quando `Option Strict` Ã¨ `On`, tutti gli operandi devono essere di tipi di dati definiti per l'operatore specificato.  
  
 **ID errore:** BC30038  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore
