 <span data-ttu-id="5bf23-104">Quando si divide la definizione di una classe in diverse dichiarazioni parziali, il compilatore considera la classe come l'unione di tutte le sue dichiarazioni parziali.</span><span class="sxs-lookup"><span data-stu-id="5bf23-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="5bf23-105">Questo riguarda non soltanto i membri, ma anche l'implementazione, l'ereditarietà e il livello di accesso.</span><span class="sxs-lookup"><span data-stu-id="5bf23-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>
