# <a name="method-ltmethodname1gt-cannot-implement-partial-method-ltmethodname2gt-because-ltmethodname3gt-already-implements-it"></a>Metodo '&lt;methodname1&gt;'non può implementare il metodo parziale'&lt;methodname2&gt;'perché'&lt;methodname3&gt;' è già implementato
Metodo '\<methodname1 >' non può implementare il metodo parziale '\<methodname2 >' perché '\<methodname3 >' è già implementato. Un solo metodo può implementare un metodo parziale.  
  
 Non è possibile avere due metodi parziali che implementano la stessa dichiarazione di metodo parziale. Il codice seguente causa questo errore.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub ValueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' First implementation of the partial method.  
    Private Sub ValueChanged()  
        MsgBox(Value was changed to " & Me.Quantity)  
    End Sub  
  
    ' Second implementation of the partial method causes this error.  
    'Private Sub ValueChanged()  
    '    Console.WriteLine("Quantity was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **ID errore:** BC31434
