# <a name="referenced-array-element-has-a-value-of-nothing"></a>L'elemento di matrice a cui si fa riferimento ha valore 'Nothing'
L'elemento di matrice a cui si fa riferimento non è stato impostato oppure in precedenza è stato impostato su `Nothing`.  
  
 **ID errore:** BC30744  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che i valori della matrice siano impostati correttamente.
