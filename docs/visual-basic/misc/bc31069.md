# <a name="enum-in-an-interface-cannot-be-declared-ltspecifiergt"></a>Enumeratore in un'interfaccia non può essere dichiarato '&lt;identificatore&gt;'
Il modificatore specificato con un'enumerazione dichiarata in un'interfaccia non è valido.  
  
 **ID errore:** BC31069  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere i modificatori come `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`o `Overridable`.
