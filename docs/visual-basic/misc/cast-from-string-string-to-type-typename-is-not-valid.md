# <a name="cast-from-string-quotltstringgtquot-to-type-lttypenamegt-is-not-valid"></a>Eseguire il cast dalla stringa &quot; &lt;stringa&gt; &quot; al tipo '&lt;typename&gt;' non è valido
Il tentativo di convertire una variabile di tipo stringa in un altro tipo non è valido.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare i tipi coinvolti nella conversione e assicurarsi che si sta tentando di convertire in un tipo significativo.
