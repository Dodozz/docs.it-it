# <a name="construct-makes-an-indirect-reference-to-assembly-ltassemblynamegt-which-contains-lttypenamegt"></a>Il costrutto contiene un riferimento indiretto all'assembly '&lt;assemblyname&gt;', che contiene'&lt;nomeTipo&gt;'
Il costrutto contiene un riferimento indiretto all'assembly '\<assemblyname >', che contiene \<typename >. Aggiungere un riferimento file a \<filename > al progetto.  
  
 Un'espressione usa un tipo, ad esempio una classe, una struttura, un'interfaccia, un'enumerazione o un delegato, ma l'assembly non ha un riferimento di progetto all'assembly che definisce il tipo.  
  
 **ID errore:** BC31528  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Nelle proprietà del progetto aggiungere un riferimento al file contenente l'assembly che definisce il tipo in uso.
