# <a name="ltargumentnamegt-is-not-a-parameter-of-ltmethoddeclarationgt"></a>«&lt;NomeArgomento&gt;'non è un parametro di'&lt;methoddeclaration&gt;»
Una chiamata di routine fornisce un argomento aggiuntivo non definito dalla routine.  
  
 **ID errore:** BC30272  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'argomento superfluo dall'elenco di argomenti.
