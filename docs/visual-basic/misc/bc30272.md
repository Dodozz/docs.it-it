# <a name="argumentname-is-not-a-parameter-of-methoddeclaration"></a>«\<NomeArgomento >' non è un parametro di '\<methoddeclaration >»
Una chiamata di routine fornisce un argomento aggiuntivo non definito dalla routine.  
  
 **ID errore:** BC30272  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'argomento superfluo dall'elenco di argomenti.
