# <a name="end-region-must-be-preceded-by-a-matching-region"></a>'#End Region' deve essere preceduto da un '#Region' corrispondente
Con `#Region` è possibile specificare un blocco di codice da espandere o comprimere durante l'uso della funzionalità di struttura dell'Editor di codice di Visual Studio. L'inizio e la fine delle istruzioni `#Region` devono essere nello stesso blocco di codice.  
  
 **ID errore:** BC30680  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Inserire `#Region` nella posizione appropriata prima dell'istruzione `#End Region` corrispondente.
