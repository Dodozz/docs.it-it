# <a name="unable-to-include-xml-fragment-ltelementpathgt-of-file-ltcommentfilegt"></a>Non è possibile includere il frammento XML '&lt;elementpath&gt;'del file'&lt;filecommento&gt;'
Un tag `<include>` in un commento XML contiene un'espressione XPath per un elemento che non esiste oppure il file contiene codice XML in formato non corretto.  
  
 **ID errore:** BC42320  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare che il codice XML nel file sia valido.  
  
2.  Verificare che il frammento XML esista nel file.
