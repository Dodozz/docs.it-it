# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>'\<identificatore >' non è valido in una dichiarazione Enum
È stato usato un identificatore che non è valido in una dichiarazione `Enum` .  
  
 **ID errore:** BC30396  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere l'identificatore.
