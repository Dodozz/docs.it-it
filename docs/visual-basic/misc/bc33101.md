# <a name="type-lttypenamegt-must-be-a-value-type-or-a-type-argument-constrained-to-structure-in-order-to-be-used-with-nullable-or-nullable-modifier-"></a>Tipo '&lt;typeName&gt;' deve essere un tipo valore o un argomento di tipo vincolato a 'Structure ' per poter essere usato con 'Nullable' o il modificatore nullable'?'
Solo i tipi valore, incluse le strutture, possono essere dichiarati nullable.  
  
```vb  
' Valid.  
Dim n? As Integer  
Dim m As Integer?  
  
' Not valid.  
' Dim p? As Object  
' Dim q As Nullable(Of Object)  
```  
  
 **ID errore:** BC33101  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere '?' o `Nullable`.  
  
-   Usare un tipo di dati valore.
