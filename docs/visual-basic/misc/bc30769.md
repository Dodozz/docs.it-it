# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-end-statement"></a>La versione di destinazione di .NET Compact Framework non supporta l'istruzione 'End'
La versione di .NET Compact Framework in uso non supporta l'istruzione `End` .  
  
 **ID errore:** BC30769  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione.
