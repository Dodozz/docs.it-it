<span data-ttu-id="42a55-103">È stato specificato un argomento di tipo in una chiamata a un metodo di estensione che è privo di parametri generici o non ha parametri generici per cui è non è già specificato un tipo.</span><span class="sxs-lookup"><span data-stu-id="42a55-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="42a55-104">Il codice seguente, ad esempio, causa questo errore.</span><span class="sxs-lookup"><span data-stu-id="42a55-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _

    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
