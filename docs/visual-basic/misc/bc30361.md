# <a name="ltproperty1gt-and-ltproperty2gt-cannot-overload-each-other-because-only-one-is-declared-default"></a>«&lt;property1&gt;'e'&lt;property2&gt;' non è in rapporto di overload perché solo uno è dichiarato come 'Default'
Se una proprietà specifica `Default`, tutte le proprietà di overload nel nome devono specificare anche `Default`.  
  
 **ID errore:** BC30361  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che tutte le proprietà di overload siano dichiarate come `Default`.
