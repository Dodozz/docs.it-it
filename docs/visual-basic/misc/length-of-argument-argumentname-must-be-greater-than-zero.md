# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>La lunghezza dell'argomento '\<NomeArgomento >' deve essere maggiore di zero
Un argomento ha un valore minore o uguale a zero.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Specificare un argomento maggiore di zero.
