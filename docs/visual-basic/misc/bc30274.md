# <a name="parameter-parametername-of-methodname-already-has-a-matching-argument"></a>Parametro '\<nomeparametro >' di '\<NomeMetodo >' ha già un argomento corrispondente
Un argomento viene passato in base al nome più di una volta nella stessa chiamata di routine.  
  
 **ID errore:** BC30274  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Controllare se uno dei nomi degli argomenti non sia stato digitato in modo errato.  
  
-   Rimuovere l'argomento duplicato dall'elenco di argomenti.
