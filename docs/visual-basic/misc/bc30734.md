# <a name="name-is-already-declared-as-a-parameter-of-this-method"></a>'\<nome >' è già dichiarato come parametro di questo metodo
Il nome del parametro specificato è già stato dichiarato per questo metodo.  
  
 **ID errore:** BC30734  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rinominare il parametro in modo che sia univoco.
