# <a name="operator-is-not-valid-for-type-lttypenamegt"></a>Operatore non è valido per il tipo '&lt;typename&gt;'
Un operatore non è appropriato per i tipi.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che l'operatore sia appropriato per i tipi usati.
