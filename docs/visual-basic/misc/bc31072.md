# <a name="warning-treated-as-error-error"></a>Avviso considerato come errore: \<errore >
Si è verificato un avviso durante l'esecuzione di un assembly compilato con l'opzione **Considera gli avvisi come errori** attivata.  
  
 **ID errore:** BC31072  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Risolvere il problema.  
  
2.  Compilare l'assembly con l'opzione **Considera gli avvisi come errori** disattivata.
