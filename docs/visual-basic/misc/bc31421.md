# <a name="ltmembergt-is-already-declared-in-this-structure"></a>«&lt;membro&gt;' è già dichiarato in questa struttura
Un membro di enumerazione è stato dichiarato più volte in questa struttura.  
  
 **ID errore:** BC31421  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la dichiarazione estranea.
