# <a name="specifiers-valid-only-at-the-beginning-of-a-declaration"></a>Gli identificatori sono validi solo all'inizio di una dichiarazione
Le parole chiave che specificano l'accessibilit√† e altre designazioni devono trovarsi prima della parola chiave di dichiarazione, ad esempio `Dim` o `Function`.  
  
 **ID errore:** BC30181  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Spostare tutti gli identificatori tra l'elenco degli attributi, se presente, e la parola chiave di dichiarazione.
