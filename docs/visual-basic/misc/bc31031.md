# <a name="project-level-conditional-compilation-constant-is-not-valid-error"></a>Costante di compilazione condizionale a livello di progetto non è valida: \<errore >
È stata specificata un'opzione di compilazione condizionale non valida per il progetto.  
  
 **ID errore:** BC31031  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare le costanti di compilazione condizionale usate con questo progetto.
