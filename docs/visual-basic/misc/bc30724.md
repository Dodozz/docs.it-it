# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>Le istruzioni 'Error' non sono valide nella finestra di controllo immediato
Le istruzioni`Error` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30724  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere le istruzioni `Error` dal codice di debug.
