# <a name="the-file-is-already-open"></a>Il file è già aperto
Si è provato ad aprire un file già aperto.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Chiudere il file prima di tentare di aprirlo.  
  
     -oppure-  
  
- Rimuovere l'istruzione `FileOpen` ridondante dal codice.
