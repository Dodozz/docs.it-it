# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>Valore di tipo '\<type1 >' non può essere convertito in '\<type2 >' perché i tipi matrice hanno numeri di dimensioni diversi
Si è provato a convertire una matrice in un'altra matrice con un numero di dimensioni diverso.  
  
 **ID errore:** BC30414  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Modificare una o entrambe le matrici in modo che il numero di dimensioni corrisponda.
