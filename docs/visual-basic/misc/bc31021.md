# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-error"></a>Compilatore Visual Basic non è in grado di risolvere l'errore seguente: \<errore >
Il compilatore Visual Basic non è in grado di risolvere l'errore seguente: <`error`>.  
  
 Alcune funzionalità IDE sono state disabilitate.  
  
 Si è verificato un errore interno nel compilatore Visual Basic.  
  
 **ID errore:** BC31021  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Salvare il lavoro e riavviare Visual Studio.  
  
2. Se l'errore si ripete, reinstallare Visual Studio.  
  
3. Se l'errore persiste dopo la reinstallazione, inviare una notifica al Servizio supporto tecnico Microsoft.
