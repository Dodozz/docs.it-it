# <a name="classes-can-inherit-only-from-other-classes"></a>Le classi possono derivare solo da altre classi
Un'istruzione `Inherits` specifica un tipo diverso da una classe come tipo di base.  
  
 **ID errore:** BC30258  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il tipo di base in una classe già definita.
