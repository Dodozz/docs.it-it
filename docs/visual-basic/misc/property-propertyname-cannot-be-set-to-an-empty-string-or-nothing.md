# <a name="property-ltpropertynamegt-cannot-be-set-to-an-empty-string-or-nothing"></a>Proprietà &lt;propertyname&gt; non può essere impostata su una stringa vuota o Nothing
La proprietà richiede un valore diverso da una stringa vuota ("") o `Nothing`. È possibile che il valore non sia stato calcolato correttamente.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare un valore valido.
