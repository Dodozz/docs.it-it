# <a name="lttypenamegt-is-an-unsupported-type"></a>«&lt;typename&gt;' è un tipo non supportato
Il tipo specificato non può essere rappresentato da Visual Studio.  
  
 **ID errore:** BC30649  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare un tipo supportato.  
  
-   Sviluppare un nuovo tipo.
