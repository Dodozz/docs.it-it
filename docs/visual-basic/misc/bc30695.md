# <a name="lttypenamegt-ltmembernamegt-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>&lt;TypeName&gt; »&lt;membername&gt;' deve essere dichiarato 'Shadows' perché un altro membro con questo nome è dichiarato come 'Shadows'
La proprietà o la routine ha lo stesso nome di un'altra proprietà o una routine contrassegnata con il modificatore `Shadows` .  
  
 **ID errore:** BC30695  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare il modificatore `Shadows` .
