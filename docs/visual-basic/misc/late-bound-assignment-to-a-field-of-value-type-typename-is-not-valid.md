# <a name="late-bound-assignment-to-a-field-of-value-type-lttypenamegt-is-not-valid-when-ltnamegt-is-the-result-of-a-late-bound-expression"></a>Assegnazione con associazione tardiva a un campo di tipo di valore '&lt;nomeTipo&gt;'non è valida quando'&lt;nome&gt;' è il risultato di un'espressione di associazione tardiva
Si è tentata un'assegnazione ad associazione tardiva non valida.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Eseguire l'assegnazione con associazione anticipata.
