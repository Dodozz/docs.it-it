# <a name="parameter-parametername-already-has-a-matching-argument"></a>Parametro '\<nomeparametro >' ha già un argomento corrispondente
Un argomento viene passato in base al nome più di una volta nella stessa chiamata di routine.  
  
 **ID errore:** BC36567  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Controllare se uno dei nomi degli argomenti sia stato digitato in modo errato.  
  
- Rimuovere l'argomento duplicato dall'elenco di argomenti.
