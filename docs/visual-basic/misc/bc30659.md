# <a name="property-or-field-39ltnamegt39-does-not-have-a-valid-attribute-type"></a>Proprietà o il campo &#39; &lt;name&gt; &#39; non dispone di un tipo di attributo valido
È possibile usare un costruttore di attributo che non accetta un tipo di attributo valido, anche se l'attributo può avere altri costruttori appropriati.  
  
 **ID errore:** BC30659  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare l'utilizzo di questo attributo.
