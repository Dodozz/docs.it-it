# <a name="method-methodname-does-not-have-a-signature-compatible-with-delegate-delegatename"></a><span data-ttu-id="3f56d-102">Metodo '\<NomeMetodo >' non ha una firma compatibile con il delegato \<'delegatename' ></span><span class="sxs-lookup"><span data-stu-id="3f56d-102">Method '\<methodname>' does not have a signature compatible with delegate \<'delegatename'></span></span>
<span data-ttu-id="3f56d-103">Questo errore si verifica quando è necessario eseguire una conversione tra un metodo e un delegato che non è possibile.</span><span class="sxs-lookup"><span data-stu-id="3f56d-103">This error occurs when a conversion is required between a method and a delegate that is not possible.</span></span> <span data-ttu-id="3f56d-104">La causa dell'errore può essere la conversione tra parametri o, quando il metodo e il delegato sono funzioni, la conversione nei valori restituiti.</span><span class="sxs-lookup"><span data-stu-id="3f56d-104">The cause of the error can be conversion between parameters or, when the method and delegate are functions, conversion in the return values.</span></span>  
  
 <span data-ttu-id="3f56d-105">Il codice seguente illustra le conversioni non riuscite.</span><span class="sxs-lookup"><span data-stu-id="3f56d-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="3f56d-106">Il delegato è `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="3f56d-106">The delegate is `FunDel`.</span></span>  
  
```vb  
Delegate Function FunDel(ByVal i As Integer, ByVal d As Double) As Integer  
```  
  
 <span data-ttu-id="3f56d-107">Ognuna delle funzioni seguenti differisce da `FunDel` in modo tale da generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="3f56d-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>  
  
```vb  
Function ExampleMethod1(ByVal m As Integer, ByVal aDate As Date) As Integer  
End Function  
  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="3f56d-108">Ognuna delle istruzioni di assegnazione seguenti genera l'errore.</span><span class="sxs-lookup"><span data-stu-id="3f56d-108">Each of the following assignment statements causes the error.</span></span>  
  
```vb  
Sub Main()  
    ' The second parameters of FunDel and ExampleMethod1, Double and Date,  
    ' are not compatible.  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2  
  
End Sub  
```
