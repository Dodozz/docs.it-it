# <a name="ltnamegt-is-declared-in-project-ltprojectname1gt-which-is-not-referenced-by-project-ltprojectname2gt"></a>«&lt;name&gt;'è dichiarato nel progetto'&lt;projectname1&gt;', che non viene fatto riferimento dal progetto'&lt;projectname2&gt;»
Un elemento di programmazione è definito in un progetto al quale il progetto corrente non fa riferimento.  
  
 **ID errore:** BC32004  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Nel progetto corrente aggiungere un riferimento al progetto che contiene l'elemento desiderato.
