# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>Si è verificata una divisione per zero durante la valutazione dell'espressione
Nella valutazione di questa espressione si è tentato di dividere un numero per zero.  
  
 **ID errore:** BC30542  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare i valori costanti e variabili per stabilire il motivo per cui nel denominatore dell'espressione è presente lo zero.
