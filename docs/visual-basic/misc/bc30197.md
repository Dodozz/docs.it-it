# <a name="as-expected"></a>Previsto 'As'
È presente una dichiarazione che non fornisce una clausola `As` per uno o più elementi dichiarati.  
  
 **ID errore:** BC30197  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Fornire la clausola `As` come richiesto o attivare `Option Strict Off`.
