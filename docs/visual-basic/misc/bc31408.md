# <a name="ltspecifier1gt-and-ltspecifier2gt-cannot-be-combined"></a>«&lt;specifier1&gt;'e'&lt;specifier2&gt;' non possono essere combinati
Due identificatori incompatibili, ad esempio `Shadows` e `Overrides` , vengono usati insieme.  
  
 **ID errore:** BC31408  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere uno degli identificatori.
