# <a name="ltspecifiergt-is-not-valid-on-a-structure-declaration"></a>«&lt;identificatore&gt;' non è valido in una dichiarazione Structure
È stato usato un identificatore, ad esempio `Dim` o `Override`, che non è valido in una dichiarazione `Structure` .  
  
 **ID errore:** BC30395  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'identificatore.
