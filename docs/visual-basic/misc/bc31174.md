# <a name="xml-processing-instruction-name-name-is-not-valid"></a>Il nome di istruzione di elaborazione XML 'name' non è valido
Il nome fornito per un'istruzione di elaborazione XML non è valido.  
  
 **ID errore:** BC31174  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Correggere gli errori nel nome dell'istruzione di elaborazione XML o rimuoverla.
