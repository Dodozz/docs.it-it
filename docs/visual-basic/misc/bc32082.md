# <a name="type-argument-lttypeargumentnamegt-is-declared-mustinherit-and-does-not-satisfy-the-new-constraint-for-type-parameter-lttypeparameternamegt"></a><span data-ttu-id="d76c1-102">Argomento di tipo '&lt;nomeargomentotipo&gt;'è dichiarato 'come MustInherit' e non soddisfa il vincolo 'New' per il parametro di tipo'&lt;nomeparametrotipo&gt;'</span><span class="sxs-lookup"><span data-stu-id="d76c1-102">Type argument '&lt;typeargumentname&gt;' is declared 'MustInherit' and does not satisfy the 'New' constraint for type parameter '&lt;typeparametername&gt;'</span></span>
<span data-ttu-id="d76c1-103">Un tipo generico viene richiamato con una classe `MustInherit` come argomento di tipo, ma il parametro di tipo corrispondente viene dichiarato con il vincolo `New` .</span><span class="sxs-lookup"><span data-stu-id="d76c1-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="d76c1-104">Il vincolo `New` richiede che il tipo passato nell'argomento di tipo corrispondente debba supportare la creazione di oggetti.</span><span class="sxs-lookup"><span data-stu-id="d76c1-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="d76c1-105">Tuttavia, una classe *astratta* , ovvero una classe dichiarata come `MustInherit`, non espone alcun costruttore perché non è possibile creare tutti gli oggetti in base ad essa.</span><span class="sxs-lookup"><span data-stu-id="d76c1-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="d76c1-106">**ID errore:** BC32082</span><span class="sxs-lookup"><span data-stu-id="d76c1-106">**Error ID:** BC32082</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d76c1-107">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="d76c1-107">To correct this error</span></span>

3.  <span data-ttu-id="d76c1-110">Se il parametro di tipo corrispondente non deve creare alcun oggetto in base al tipo passato, rimuovere il vincolo `New` dalla relativa dichiarazione.</span><span class="sxs-lookup"><span data-stu-id="d76c1-110">If the corresponding type parameter does not need to create any objects from the type passed to it, then remove the `New` constraint from its declaration.</span></span>
