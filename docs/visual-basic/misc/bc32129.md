# <a name="enum-types-cannot-be-nullable"></a>I tipi enum non possono essere nullable
Il tipo sottostante usato per dichiarare un'enumerazione non pu√≤ essere nullable. Il codice seguente, ad esempio, causa questo errore:  
  

## <a name="to-correct-this-error"></a>Per correggere l'errore
