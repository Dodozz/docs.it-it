# <a name="-requires-its-left-operand-to-have-a-type-parameter-class-or-interface-type-but-this-operand-has-the-type-type"></a>'!' richiede la sua sinistra operando per avere un parametro di tipo, classe o tipo di interfaccia, ma questo operando ha il tipo '\<tipo >'
L'accesso ai membri dizionario, che usa un punto esclamativo (`!`) anziché un punto (`.`), è valido solo in una classe o in un'interfaccia.  
  
 **ID errore:** BC30103  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Sostituire l'espressione a sinistra del punto esclamativo con una che restituisce una classe definita o un tipo di interfaccia.
