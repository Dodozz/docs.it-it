# <a name="conflicts-with-ltname1gt-which-is-implicitly-created-for-ltname2gt-in-lttypegt-ltname3gt"></a>È in conflitto con '&lt;name1&gt;', che viene creato in modo implicito per'&lt;nome2&gt;' nel &lt;tipo&gt; '&lt;name3&gt;'
Il nome di un membro di tipo è in conflitto con un altro membro creato in modo implicito.  
  
 **ID errore:** BC31058  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rinominare il membro dichiarato in modo esplicito per rimuovere il conflitto di denominazione.
