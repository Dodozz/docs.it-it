# <a name="expression-ltnamegt-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>Espressione '&lt;nome&gt;' non è una routine, ma si verifica perché la destinazione di una chiamata di routine
È stata eseguita una chiamata di routine a un'espressione che non è una routine.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare che il nome della routine sia stato digitato correttamente.  
  
2.  Fornire la routine chiamata da <`name`>.
