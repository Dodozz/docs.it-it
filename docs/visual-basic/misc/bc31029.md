# <a name="method-ltmethodnamegt-cannot-handle-event-lteventnamegt-because-they-do-not-have-the-same-signature"></a>Metodo '&lt;NomeMetodo&gt;'non è possibile gestire l'evento'&lt;eventname&gt;' perché non hanno la stessa firma
I parametri o tipi di parametro per il gestore eventi specificato non corrispondono a quelli definiti per l'evento specificato.  
  
 **ID errore:** BC31029  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Ridefinire l'evento o il gestore eventi in modo che gli elenchi di parametri corrispondano.
