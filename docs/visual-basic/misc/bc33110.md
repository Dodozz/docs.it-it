# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="08ed7-102">Non è possibile dedurre un tipo comune per il primo e il secondo operando dell'operatore 'If'</span><span class="sxs-lookup"><span data-stu-id="08ed7-102">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>
<span data-ttu-id="08ed7-103">Non è possibile dedurre un tipo comune per il primo e il secondo operando dell'operatore 'If'.</span><span class="sxs-lookup"><span data-stu-id="08ed7-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="08ed7-104">Uno deve avere una conversione Widening nell'altro.</span><span class="sxs-lookup"><span data-stu-id="08ed7-104">One must have a widening conversion to the other.</span></span>  
  
 <span data-ttu-id="08ed7-105">L'operatore binario `If` richiede che ci sia una conversione verso un tipo di dati più grande tra uno degli argomenti e l'altro.</span><span class="sxs-lookup"><span data-stu-id="08ed7-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="08ed7-106">Ad esempio, poiché non è disponibile una conversione verso un tipo di dati più grande in entrambe le direzioni tra `Integer` e `String`, il codice seguente genera questo errore.</span><span class="sxs-lookup"><span data-stu-id="08ed7-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>  
  
```vb  
Dim first? As Integer  
Dim second As String = "First is Nothing"  
'' Not valid.  
' Console.WriteLine(If(first, second))  
```  
  
 <span data-ttu-id="08ed7-107">**ID errore:** BC33110</span><span class="sxs-lookup"><span data-stu-id="08ed7-107">**Error ID:** BC33110</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="08ed7-108">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="08ed7-108">To correct this error</span></span>  
  
- <span data-ttu-id="08ed7-109">Fornire una conversione esplicita per uno degli operandi, se possibile nel codice:</span><span class="sxs-lookup"><span data-stu-id="08ed7-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>  
  
    ```  
    Console.WriteLine(If(first, CInt(second)))   
    ```  
  
- <span data-ttu-id="08ed7-110">Riscrivere il codice usando una diversa costruzione condizionale.</span><span class="sxs-lookup"><span data-stu-id="08ed7-110">Rewrite the code by using a different conditional construction.</span></span>  
  
    ```  
    If first IsNot Nothing Then  
        Console.WriteLine(first)  
    Else  
        Console.WriteLine(second)  
    End If  
    ```
