<span data-ttu-id="5fffe-103">Non è possibile dedurre un tipo comune per il primo e il secondo operando dell'operatore 'If'.</span><span class="sxs-lookup"><span data-stu-id="5fffe-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="5fffe-104">Uno deve avere una conversione Widening nell'altro.</span><span class="sxs-lookup"><span data-stu-id="5fffe-104">One must have a widening conversion to the other.</span></span>  
  
 <span data-ttu-id="5fffe-105">L'operatore binario `If` richiede che ci sia una conversione verso un tipo di dati più grande tra uno degli argomenti e l'altro.</span><span class="sxs-lookup"><span data-stu-id="5fffe-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="5fffe-106">Ad esempio, poiché non è disponibile una conversione verso un tipo di dati più grande in entrambe le direzioni tra `Integer` e `String`, il codice seguente genera questo errore.</span><span class="sxs-lookup"><span data-stu-id="5fffe-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb  
'' Not valid.

```  
  
 <span data-ttu-id="5fffe-107">**ID errore:** BC33110</span><span class="sxs-lookup"><span data-stu-id="5fffe-107">**Error ID:** BC33110</span></span>

    ```  
    Else  
        Console.WriteLine(second)
    ```  
