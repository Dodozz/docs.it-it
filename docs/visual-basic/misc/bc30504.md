# <a name="property-propertyname-cannot-be-declared-default-because-it-overrides-a-property-on-the-base-type-typename-that-is-not-default"></a>Proprietà '\<NomeProprietà >' non può essere dichiarata 'Default' perché esegue l'override di una proprietà di base \<tipo > '\<nomeTipo >' che non è predefinita
Si è provato a dichiarare una proprietà `Default`, che non è una dichiarazione valida perché esegue l'override di una proprietà che non è stata dichiarata `Default` nella classe base.  
  
 **ID errore:** BC30504  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere la dichiarazione `Default` .
