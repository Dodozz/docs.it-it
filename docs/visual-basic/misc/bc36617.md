# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Non è possibile usare il nome della funzione di aggregazione con un carattere tipo
Un carattere tipo è stato incluso alla fine del nome di una funzione di aggregazione.  
  
 **ID errore:** BC36617  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere il carattere tipo dal nome della funzione di aggregazione.
