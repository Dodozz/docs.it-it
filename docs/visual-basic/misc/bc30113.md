# <a name="namespace-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>'\<dello spazio dei nomi >' non è un nome valido e non può essere usato come nome dello spazio dei nomi radice
Uno spazio dei nomi di qualificazione non è un nome di elemento valido. Un nome deve contenere solo caratteri alfabetici, cifre decimali e caratteri di sottolineatura e deve iniziare con un carattere alfabetico o un carattere di sottolineatura (`_`).  
  
 **ID errore:** BC30113  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Se lo spazio dei nomi fa parte di un set di spazi dei nomi annidati, verificare se uno dei nomi esterni contiene caratteri non validi.
