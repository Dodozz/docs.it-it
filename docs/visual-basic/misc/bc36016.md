# <a name="implicit-variable-ltvariablenamegt-is-invalid-because-of-ltmessagegt"></a>Variabile implicita '&lt;NomeVariabile&gt;'non valido perché'&lt;messaggio&gt;'
Durante il debug è stata generata una variabile implicita non valida.  
  
 **ID errore:** BC36016  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Esaminare il messaggio per individuare il motivo e il possibile rimedio.  
  
2.  Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio supporto tecnico Microsoft.
