# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>Variabile locale '\<nomevariabile >' non pu√≤ fare riferimento alla prima che sia dichiarata
Una variabile all'interno di una routine viene usata prima di essere dichiarata.  
  
 **ID errore:** BC32000  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Spostare la dichiarazione in modo che preceda il primo riferimento alla variabile.
