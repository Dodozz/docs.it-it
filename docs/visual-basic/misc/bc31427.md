# <a name="syntax-error-in-conditional-compilation-expression"></a>Errore di sintassi nell'espressione di compilazione condizionale
L'espressione di compilazione condizionale genera un errore di sintassi non specificato.  
  
 **ID errore:** BC31427  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Consultare la documentazione per una compilazione condizionale e le parole chiave usate nell'espressione. Confrontare il codice di esempio nella documentazione con la riga di codice sorgente che genera questo errore.  
  
2. Se non si riesce a identificare la causa dell'errore, raccogliere informazioni sulla situazione contingente e informare il Servizio supporto tecnico Microsoft.
