# <a name="cannot-inherit-interface-interfacename1-because-it-could-be-identical-to-interface-interfacename2-for-some-type-arguments"></a><span data-ttu-id="efaed-102">Non è possibile ereditare l'interfaccia '\<interfacename1 >' perché potrebbe essere identica all'interfaccia '\<interfacename2 >' per alcuni argomenti tipo</span><span class="sxs-lookup"><span data-stu-id="efaed-102">Cannot inherit interface '\<interfacename1>' because it could be identical to interface '\<interfacename2>' for some type arguments</span></span>
<span data-ttu-id="efaed-103">Un'interfaccia generica eredita più volte dall'altra interfaccia generica e due ereditarietà possono entrare in conflitto per determinati valori degli argomenti di tipo.</span><span class="sxs-lookup"><span data-stu-id="efaed-103">A generic interface inherits more than once from another generic interface, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="efaed-104">Le istruzioni seguenti possono generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="efaed-104">The following statements can generate this error.</span></span>  
  
 `Public Interface interfaceA(Of u)`  
  
 `End Interface`  
  
 `Public Interface derivedInterface(Of t1, t2)`  
  
 `Inherits interfaceA(Of t1), interfaceA(Of t2)`  
  
 `End Interface`  
  
 <span data-ttu-id="efaed-105">Se l'interfaccia `derivedInterface` è costruita o implementata specificando lo stesso tipo sia in `t1` che in `t2`, deve ereditare due versioni di `interfaceA` con argomenti di tipo identici.</span><span class="sxs-lookup"><span data-stu-id="efaed-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="efaed-106">In questo modo si produrrebbe un'ambiguità sulla versione a cui accedere.</span><span class="sxs-lookup"><span data-stu-id="efaed-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="efaed-107">**ID errore:** BC32120</span><span class="sxs-lookup"><span data-stu-id="efaed-107">**Error ID:** BC32120</span></span>
