# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-to-specifier1-type1-through-specifier2-type2"></a>«\<nome >' non può esporre il tipo delegato sottostante '\<tipodelegato >' dell'evento che sta implementando in \<specifier1 > '\<type1 >' tramite \<specifier2 > '\<type2 >'
Il codice espone un tipo privato all'esterno della classe pubblica.  
  
 **ID errore:** BC30914  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Dichiarare il tipo `Public` o usare un altro tipo.
