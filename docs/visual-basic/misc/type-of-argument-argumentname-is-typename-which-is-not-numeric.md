# <a name="type-of-argument-ltargumentnamegt-is-lttypenamegt-which-is-not-numeric"></a>Tipo di argomento '&lt;NomeArgomento&gt;'is'&lt;nomeTipo&gt;', che non è numerico
Un argomento non numerico si verifica in una situazione in cui è previsto un argomento numerico.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che l'argomento sia numerico.
