# <a name="ltmathfunction1gt-is-not-declared"></a>«&lt;mathfunction1&gt;' non è dichiarato
'\<mathfunction1 >' non è dichiarato. Funzione è stato spostato alla classe System. Math'ed è denominata '\<mathfunction2 >'.  
  
 Numerose funzioni che erano intrinseche a Visual Basic nelle versioni precedenti sono state spostate i <xref:System.Math?displayProperty=nameWithType> dello spazio dei nomi. In questo modo le relative funzionalità sono rese più generalmente disponibili per tutti i linguaggi di programmazione.  
  
 **ID errore:** BC30819  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Usare i metodi dichiarati in <xref:System.Math?displayProperty=nameWithType>.
