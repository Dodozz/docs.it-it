# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>Il 'carattere character' (0x\<esadecimale >) non è consentita nel codice XML letterale
È stato usato un carattere non supportato in un valore letterale XML.  
  
 **ID errore:** BC31176  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere il carattere non supportato.
