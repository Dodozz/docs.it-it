# <a name="continue-must-be-followed-by-do-for-or-while"></a><span data-ttu-id="a8c6b-102">'Continue' deve essere seguito da 'Do', 'For' o 'While'</span><span class="sxs-lookup"><span data-stu-id="a8c6b-102">'Continue' must be followed by 'Do', 'For' or 'While'</span></span>
<span data-ttu-id="a8c6b-103">Un'istruzione `Continue` deve essere seguita da `Do`, `For`o `While`, a seconda che l'istruzione `Continue` si trovi all'interno di un ciclo `Do...Loop` , `For...Next` o `While...End While` .</span><span class="sxs-lookup"><span data-stu-id="a8c6b-103">A `Continue` statement must be followed by `Do`, `For`, or `While`, depending on if the `Continue` statement appears within a `Do...Loop` loop, `For...Next` loop, or `While...End While` loop.</span></span>
