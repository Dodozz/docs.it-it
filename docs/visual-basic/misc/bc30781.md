<span data-ttu-id="7927a-103">Un'istruzione `Continue` deve essere seguita da `Do`, `For`o `While`, a seconda che l'istruzione `Continue` si trovi all'interno di un ciclo `Do...Loop` , `For...Next` o `While...End While` .</span><span class="sxs-lookup"><span data-stu-id="7927a-103">A `Continue` statement must be followed by `Do`, `For`, or `While`, depending on if the `Continue` statement appears within a `Do...Loop` loop, `For...Next` loop, or `While...End While` loop.</span></span>  
  
 <span data-ttu-id="7927a-104">**ID errore:** BC30781</span><span class="sxs-lookup"><span data-stu-id="7927a-104">**Error ID:** BC30781</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7927a-105">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="7927a-105">To correct this error</span></span>
