# <a name="xml-attribute-version-must-be-the-first-attribute-in-xml-declaration"></a>L'attributo XML 'version' deve essere il primo attributo nella dichiarazione XML
L'attributo `version` di un valore letterale documento XML non Ã¨ il primo attributo specificato. L'attributo `version` deve essere il primo attributo specificato per un valore letterale documento XML, in base alla specifica XML 1.0.  
  
 **ID errore:** BC31156  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore
