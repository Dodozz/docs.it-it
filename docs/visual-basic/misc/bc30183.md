# <a name="keyword-is-not-valid-as-an-identifier"></a>La parola chiave non è valida come identificatore
Una parola chiave viene visualizzata nel punto in cui è richiesto il nome di un elemento. Il contesto chiama una variabile o una routine.  
  
 **ID errore:** BC30183  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Sostituire la parola chiave con un nome di elemento valido.
