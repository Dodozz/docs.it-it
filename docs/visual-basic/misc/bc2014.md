# <a name="the-value-ltvaluegt-is-invalid-for-option-ltoptionnamegt"></a>il valore '&lt;valore&gt;'non è valido per l'opzione'&lt;nomeopzione&gt;'
Per un'opzione della riga di comando è stato fornito un valore non valido.  
  
 **ID errore:** BC2014  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Controllare i valori per individuare quello non valido.
