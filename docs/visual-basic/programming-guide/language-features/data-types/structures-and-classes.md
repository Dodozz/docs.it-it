- <span data-ttu-id="9c18b-116">Entrambi possono implementare interfacce.</span><span class="sxs-lookup"><span data-stu-id="9c18b-116">Both can implement interfaces.</span></span>

## <a name="differences"></a><span data-ttu-id="9c18b-120">Differenze</span><span class="sxs-lookup"><span data-stu-id="9c18b-120">Differences</span></span>  
 <span data-ttu-id="9c18b-121">Classi e strutture sono diversi per le indicazioni seguenti:</span><span class="sxs-lookup"><span data-stu-id="9c18b-121">Structures and classes differ in the following particulars:</span></span>  
  
- <span data-ttu-id="9c18b-122">Le strutture sono *i tipi di valore*; le classi sono *fanno riferimento ai tipi*.</span><span class="sxs-lookup"><span data-stu-id="9c18b-122">Structures are *value types*; classes are *reference types*.</span></span> <span data-ttu-id="9c18b-123">Una variabile di un tipo di struttura contiene i dati della struttura, anziché che contiene un riferimento ai dati come un tipo di classe.</span><span class="sxs-lookup"><span data-stu-id="9c18b-123">A variable of a structure type contains the structure's data, rather than containing a reference to the data as a class type does.</span></span>  
  
- <span data-ttu-id="9c18b-124">Strutture di utilizzano l'allocazione dello stack; classi usano allocazione dell'heap.</span><span class="sxs-lookup"><span data-stu-id="9c18b-124">Structures use stack allocation; classes use heap allocation.</span></span>  
  
- <span data-ttu-id="9c18b-125">Tutti gli elementi di struttura sono `Public` per impostazione predefinita; classe variabili e costanti rappresentano `Private` per impostazione predefinita, mentre altri membri della classe sono `Public` per impostazione predefinita.</span><span class="sxs-lookup"><span data-stu-id="9c18b-125">All structure elements are `Public` by default; class variables and constants are `Private` by default, while other class members are `Public` by default.</span></span> <span data-ttu-id="9c18b-126">Questo comportamento per i membri di classi fornisce la compatibilità con il sistema di Visual Basic 6.0 di valori predefiniti.</span><span class="sxs-lookup"><span data-stu-id="9c18b-126">This behavior for class members provides compatibility with the Visual Basic 6.0 system of defaults.</span></span>  
  
- <span data-ttu-id="9c18b-127">Una struttura deve avere almeno un non personalizzato non condiviso di variabile o condiviso, elemento di un evento; una classe può essere completamente vuota.</span><span class="sxs-lookup"><span data-stu-id="9c18b-127">A structure must have at least one nonshared variable or nonshared, noncustom event element; a class can be completely empty.</span></span>  
  
- <span data-ttu-id="9c18b-128">Gli elementi di struttura non possono essere dichiarati come `Protected`; i membri di classe possono.</span><span class="sxs-lookup"><span data-stu-id="9c18b-128">Structure elements cannot be declared as `Protected`; class members can.</span></span>  
  
- <span data-ttu-id="9c18b-129">Una routine di struttura può gestire gli eventi solo se è un [Shared](../../../../visual-basic/language-reference/modifiers/shared.md) `Sub` routine e solo per mezzo del [istruzione AddHandler](../../../../visual-basic/language-reference/statements/addhandler-statement.md); qualsiasi routine della classe può gestire gli eventi utilizzando entrambi il [ Gestisce](../../../../visual-basic/language-reference/statements/handles-clause.md) parola chiave o il `AddHandler` istruzione.</span><span class="sxs-lookup"><span data-stu-id="9c18b-129">A structure procedure can handle events only if it is a [Shared](../../../../visual-basic/language-reference/modifiers/shared.md)`Sub` procedure, and only by means of the [AddHandler Statement](../../../../visual-basic/language-reference/statements/addhandler-statement.md); any class procedure can handle events, using either the [Handles](../../../../visual-basic/language-reference/statements/handles-clause.md) keyword or the `AddHandler` statement.</span></span> <span data-ttu-id="9c18b-130">Per ulteriori informazioni, vedi [Eventi](../../../../visual-basic/programming-guide/language-features/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="9c18b-130">For more information, see [Events](../../../../visual-basic/programming-guide/language-features/events/index.md).</span></span>  
  
- <span data-ttu-id="9c18b-131">Le dichiarazioni di variabili di struttura non possono specificare inizializzatori o dimensioni iniziali per le matrici. le dichiarazioni di variabili di classe possono.</span><span class="sxs-lookup"><span data-stu-id="9c18b-131">Structure variable declarations cannot specify initializers or initial sizes for arrays; class variable declarations can.</span></span>  
  
- <span data-ttu-id="9c18b-132">Le strutture ereditano in modo implicito dal <xref:System.ValueType?displayProperty=nameWithType> classe e non può ereditare da qualsiasi altro tipo; le classi possono ereditare da qualsiasi classe o classi diverso da <xref:System.ValueType?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="9c18b-132">Structures implicitly inherit from the <xref:System.ValueType?displayProperty=nameWithType> class and cannot inherit from any other type; classes can inherit from any class or classes other than <xref:System.ValueType?displayProperty=nameWithType>.</span></span>  
  
- <span data-ttu-id="9c18b-133">Strutture non sono ereditabili; le classi sono.</span><span class="sxs-lookup"><span data-stu-id="9c18b-133">Structures are not inheritable; classes are.</span></span>  
  
- <span data-ttu-id="9c18b-134">Le strutture non sono mai terminate, common language runtime (CLR) non chiama mai il <xref:System.Object.Finalize%2A> metodo su qualsiasi struttura; le classi vengono terminate dal garbage collector (GC), che chiama <xref:System.Object.Finalize%2A> in una classe quando rileva non sono presenti riferimenti attivi rimanenti.</span><span class="sxs-lookup"><span data-stu-id="9c18b-134">Structures are never terminated, so the common language runtime (CLR) never calls the <xref:System.Object.Finalize%2A> method on any structure; classes are terminated by the garbage collector (GC), which calls <xref:System.Object.Finalize%2A> on a class when it detects there are no active references remaining.</span></span>  
  
- <span data-ttu-id="9c18b-135">Una struttura non richiede un costruttore. esegue una classe.</span><span class="sxs-lookup"><span data-stu-id="9c18b-135">A structure does not require a constructor; a class does.</span></span>  
  
- <span data-ttu-id="9c18b-136">Le strutture possono contenere costruttori non condivisi solo se si accettano parametri; le classi possibile configurarli con o senza parametri.</span><span class="sxs-lookup"><span data-stu-id="9c18b-136">Structures can have nonshared constructors only if they take parameters; classes can have them with or without parameters.</span></span>  
  
 <span data-ttu-id="9c18b-137">Ogni struttura dispone di un costruttore pubblico senza parametri.</span><span class="sxs-lookup"><span data-stu-id="9c18b-137">Every structure has an implicit public constructor without parameters.</span></span> <span data-ttu-id="9c18b-138">Questo costruttore inizializza gli elementi di dati della struttura per i relativi valori predefiniti.</span><span class="sxs-lookup"><span data-stu-id="9c18b-138">This constructor initializes all the structure's data elements to their default values.</span></span> <span data-ttu-id="9c18b-139">Non è possibile ridefinire il problema.</span><span class="sxs-lookup"><span data-stu-id="9c18b-139">You cannot redefine this behavior.</span></span>  
  
## <a name="instances-and-variables"></a><span data-ttu-id="9c18b-140">Istanze e variabili</span><span class="sxs-lookup"><span data-stu-id="9c18b-140">Instances and Variables</span></span>  
 <span data-ttu-id="9c18b-141">Poiché le strutture sono tipi valore, ogni variabile di struttura in modo permanente è associato a un'istanza di singole strutture.</span><span class="sxs-lookup"><span data-stu-id="9c18b-141">Because structures are value types, each structure variable is permanently bound to an individual structure instance.</span></span> <span data-ttu-id="9c18b-142">Ma le classi sono tipi riferimento e una variabile oggetto può fare riferimento a diverse istanze di classi in momenti diversi.</span><span class="sxs-lookup"><span data-stu-id="9c18b-142">But classes are reference types, and an object variable can refer to various class instances at different times.</span></span> <span data-ttu-id="9c18b-143">Questa differenza influisce sull'utilizzo di strutture e classi nei modi seguenti:</span><span class="sxs-lookup"><span data-stu-id="9c18b-143">This distinction affects your usage of structures and classes in the following ways:</span></span>  
  
- <span data-ttu-id="9c18b-144">**Inizializzazione.**</span><span class="sxs-lookup"><span data-stu-id="9c18b-144">**Initialization.**</span></span> <span data-ttu-id="9c18b-145">Una variabile di struttura in modo implicito include un'inizializzazione degli elementi usando il costruttore della struttura senza parametri.</span><span class="sxs-lookup"><span data-stu-id="9c18b-145">A structure variable implicitly includes an initialization of the elements using the structure's parameterless constructor.</span></span> <span data-ttu-id="9c18b-146">Pertanto `Dim s As struct1` equivale a `Dim s As struct1 = New struct1()`.</span><span class="sxs-lookup"><span data-stu-id="9c18b-146">Therefore, `Dim s As struct1` is equivalent to `Dim s As struct1 = New struct1()`.</span></span>

     <span data-ttu-id="9c18b-157">Elementi della struttura, tuttavia, sono isolati all'interno della relativa istanza.</span><span class="sxs-lookup"><span data-stu-id="9c18b-157">Structure elements, however, are isolated within their own instance.</span></span> <span data-ttu-id="9c18b-158">I valori non vengono riflesse nelle altre variabili di struttura, anche in altre istanze dello stesso `Structure` dichiarazione.</span><span class="sxs-lookup"><span data-stu-id="9c18b-158">Changes to their values are not reflected in any other structure variables, even in other instances of the same `Structure` declaration.</span></span>
