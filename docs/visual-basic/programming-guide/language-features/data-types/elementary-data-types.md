# <a name="elementary-data-types-visual-basic"></a>Tipi di dati elementari (Visual Basic)
Visual Basic fornisce un set di tipi di dati predefiniti, che è possibile usare per molti elementi di programmazione. Questa sezione descrive questi tipi e come usarli.  
  
> [!NOTE]
>  Ogni tipo di dati elementari in Visual Basic è supportata da una struttura o una classe che si trova nel <xref:System> dello spazio dei nomi. Il compilatore Usa ogni parola chiave tipo di dati come un alias per la struttura sottostante o classe. Ad esempio, dichiarare una variabile usando la parola riservata `Byte` equivale dichiararla usando il nome della struttura completo <xref:System.Byte?displayProperty=nameWithType>.
