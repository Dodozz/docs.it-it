-   <span data-ttu-id="2e03d-115">**Protezione**.</span><span class="sxs-lookup"><span data-stu-id="2e03d-115">**Protection**.</span></span> <span data-ttu-id="2e03d-116">Nella scelta tra i due meccanismi di passaggio, il criterio più importante è l'esposizione della chiamata al metodo le variabili da modificare.</span><span class="sxs-lookup"><span data-stu-id="2e03d-116">In choosing between the two passing mechanisms, the most important criterion is the exposure of calling variables to change.</span></span> <span data-ttu-id="2e03d-117">Il vantaggio di passaggio di un argomento `ByRef` è che la procedura può restituire un valore al codice chiamante tramite l'argomento.</span><span class="sxs-lookup"><span data-stu-id="2e03d-117">The advantage of passing an argument `ByRef` is that the procedure can return a value to the calling code through that argument.</span></span> <span data-ttu-id="2e03d-118">Il vantaggio di passaggio di un argomento `ByVal` è che impedisce che la variabile viene modificato dalla procedura.</span><span class="sxs-lookup"><span data-stu-id="2e03d-118">The advantage of passing an argument `ByVal` is that it protects a variable from being changed by the procedure.</span></span>
