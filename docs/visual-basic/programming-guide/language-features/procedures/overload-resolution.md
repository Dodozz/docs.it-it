# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="70678-102">Risoluzione dell'overload (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="70678-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="70678-103">Quando il compilatore Visual Basic rileva una chiamata a una routine definito in numerose versioni di overload, il compilatore deve decidere quale overload da chiamare.</span><span class="sxs-lookup"><span data-stu-id="70678-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="70678-104">A tale scopo, seguire questa procedura:</span><span class="sxs-lookup"><span data-stu-id="70678-104">It does this by performing the following steps:</span></span>  
  
1.  <span data-ttu-id="70678-105">**Accessibilità.**</span><span class="sxs-lookup"><span data-stu-id="70678-105">**Accessibility.**</span></span> <span data-ttu-id="70678-106">Elimina tutti gli overload con un livello di accesso che impedisce che il codice chiamante viene chiamata.</span><span class="sxs-lookup"><span data-stu-id="70678-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2.  <span data-ttu-id="70678-107">**Numero di parametri.**</span><span class="sxs-lookup"><span data-stu-id="70678-107">**Number of Parameters.**</span></span> <span data-ttu-id="70678-108">Elimina qualsiasi overload che definisce un numero di parametri diverso rispetto a quello fornito nella chiamata.</span><span class="sxs-lookup"><span data-stu-id="70678-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3.  <span data-ttu-id="70678-109">**Tipi di dati di parametro.**</span><span class="sxs-lookup"><span data-stu-id="70678-109">**Parameter Data Types.**</span></span> <span data-ttu-id="70678-110">Il compilatore assegna la priorità di metodi di istanza rispetto ai metodi di estensione.</span><span class="sxs-lookup"><span data-stu-id="70678-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="70678-111">Se viene trovato alcun metodo di istanza che richiede solo di ampliamento per rispondere alla chiamata di procedura, vengono eliminati tutti i metodi di estensione e il compilatore continua con solo i candidati dei metodi di istanza.</span><span class="sxs-lookup"><span data-stu-id="70678-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="70678-112">Se non viene trovato alcun metodo di tale istanza, continua con l'istanza sia i metodi di estensione.</span><span class="sxs-lookup"><span data-stu-id="70678-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="70678-113">In questo passaggio elimina alcun overload per il quale i tipi di dati degli argomenti di chiamata non convertibili nei tipi di parametro definiti in overload.</span><span class="sxs-lookup"><span data-stu-id="70678-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4.  <span data-ttu-id="70678-114">**Conversioni di Narrowing.**</span><span class="sxs-lookup"><span data-stu-id="70678-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="70678-115">Elimina qualsiasi overload che richiede una conversione narrowing dai tipi di argomento chiamante per i tipi di parametro definiti.</span><span class="sxs-lookup"><span data-stu-id="70678-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="70678-116">Ciò è vero se il controllo del tipo di opzione ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) viene `On` o `Off`.</span><span class="sxs-lookup"><span data-stu-id="70678-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5.  <span data-ttu-id="70678-117">**Widening minimi.**</span><span class="sxs-lookup"><span data-stu-id="70678-117">**Least Widening.**</span></span> <span data-ttu-id="70678-118">Il compilatore prende in considerazione gli overload rimanenti nelle coppie.</span><span class="sxs-lookup"><span data-stu-id="70678-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="70678-119">Per ogni coppia, confronta i tipi di dati dei parametri definiti.</span><span class="sxs-lookup"><span data-stu-id="70678-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="70678-120">Se i tipi in uno degli overload tutti vengano ampliati ai tipi corrispondenti in altra, il compilatore consente di eliminare quest'ultimo.</span><span class="sxs-lookup"><span data-stu-id="70678-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="70678-121">Vale a dire, viene mantenuto l'overload che richiede una quantità minima di ampliamento.</span><span class="sxs-lookup"><span data-stu-id="70678-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6.  <span data-ttu-id="70678-122">**Singola candidato.**</span><span class="sxs-lookup"><span data-stu-id="70678-122">**Single Candidate.**</span></span> <span data-ttu-id="70678-123">Vengono presi in considerazione gli overload nelle coppie finché l'unico overload rimane impostato su e la chiamata a tale overload viene risolta.</span><span class="sxs-lookup"><span data-stu-id="70678-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="70678-124">Se il compilatore non è possibile ridurre gli overload a uno solo, viene generato un errore.</span><span class="sxs-lookup"><span data-stu-id="70678-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
