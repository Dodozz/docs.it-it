# <a name="modules-visual-basic"></a>Moduli (Visual Basic)
Visual Basic fornisce diversi moduli che consentono di semplificare le attività comuni nel codice, tra cui la modifica di stringhe, l'esecuzione di calcoli matematici, recupero di informazioni di sistema, l'esecuzione di operazioni su file e directory e così via. Nella tabella seguente elenca i moduli forniti da Visual Basic.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.Constants>|Contiene costanti varie. Queste costanti possono essere utilizzate ovunque nel codice.|  
|<xref:Microsoft.VisualBasic.ControlChars>|Contiene i caratteri di controllo costante per la stampa e visualizzazione di testo.|  
|<xref:Microsoft.VisualBasic.Conversion>|Contiene i membri che consentono di convertire i numeri decimali in altre basi, numeri in stringhe, le stringhe in numeri e i dati di un tipo a altro.|  
|<xref:Microsoft.VisualBasic.DateAndTime>|Contiene membri che ottiene la data corrente o l'ora, eseguono calcoli di data, restituiscono una data o ora, impostare la data o ora o calcolare la durata di un processo.|  
|<xref:Microsoft.VisualBasic.ErrObject>|Contiene informazioni sui metodi per generare o cancellare un errore e di errori di run-time.|  
|<xref:Microsoft.VisualBasic.FileSystem>|Contiene i membri che eseguono operazioni di file, directory o cartella e del sistema.|  
|<xref:Microsoft.VisualBasic.Financial>|Contiene procedure che consentono di eseguire i calcoli finanziari.|  
|<xref:Microsoft.VisualBasic.Globals>|Contiene informazioni sulla versione del motore di scripting corrente.|  
|<xref:Microsoft.VisualBasic.Information>|Contiene i membri che restituiscono, testano o verificare le informazioni come dimensione della matrice, i nomi dei tipi e così via.|  
|<xref:Microsoft.VisualBasic.Interaction>|Contiene membri che interagiscono con gli oggetti, applicazioni e sistemi.|  
|<xref:Microsoft.VisualBasic.Strings>|Contiene i membri che eseguono operazioni sulle stringhe, ad esempio stringhe, la ricerca di una stringa, ottenere la lunghezza di una stringa, riformattazione e così via.|  
|<xref:Microsoft.VisualBasic.VBMath>|Contiene i membri di eseguire operazioni matematiche.|
