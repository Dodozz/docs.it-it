# <a name="property-or-method-not-found"></a>Proprietà o metodo non trovato
Il metodo di oggetto di riferimento o una proprietà dell'oggetto non è definita.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Si potrebbe avere digitato correttamente il nome dell'oggetto. Per visualizzare le proprietà e metodi definiti per un oggetto, visualizzare il Visualizzatore oggetti. Selezionare la raccolta di oggetti appropriati per visualizzare un elenco di metodi e proprietà disponibili.
