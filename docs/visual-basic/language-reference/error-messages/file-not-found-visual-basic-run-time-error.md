# <a name="file-not-found-visual-basic-run-time-error"></a>Impossibile trovare il file (errore di run-time Visual Basic)
Il file non trovato in cui specificata. L'errore presenta le seguenti cause possibili:  
  
-   Un'istruzione fa riferimento a un file che non esiste.  
  
-   È stato effettuato un tentativo di chiamare una routine in una libreria di collegamento dinamico (DLL), ma la libreria specificata nel `Lib` clausola del `Declare` istruzione nebyla nalezena.  
  
-   Si è tentato di aprire un progetto o si carica un file di testo che non esiste.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Controllare l'ortografia del nome del file e la specifica del percorso.
