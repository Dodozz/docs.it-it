# <a name="subscript-out-of-range-visual-basic"></a>Indice non incluso nell'intervallo (Visual Basic)
Indice di matrice non è valido perché rientra nell'intervallo consentito. Il valore di indice più basso di una dimensione è sempre 0 e il valore di indice massimo restituito dal `GetUpperBound` metodo per la dimensione.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Modificare l'indice in modo che rientri nell'intervallo valido.
