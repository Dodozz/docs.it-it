# <a name="property-array-index-is-not-valid"></a>Indice della matrice di proprietà non valido
Il valore fornito non è valido per un indice di matrice di proprietà.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Controllare la documentazione del componente per assicurarsi che l'indice è compreso nell'intervallo valido per la proprietà specificata.
