# <a name="internal-error-happened-at-location"></a>Si è verificato un errore interno nella \<percorso >
Si è verificato un errore interno. La riga in corrispondenza del quale si è verificato è contenuta nel messaggio di errore.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Assicurarsi che questo errore non è stato generato per il `Error` istruzione o `Raise` metodo; se non era, contattare il servizio supporto tecnico clienti Microsoft per segnalare le condizioni in cui è stato visualizzato il messaggio.
