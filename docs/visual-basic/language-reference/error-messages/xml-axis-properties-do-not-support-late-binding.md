# <a name="xml-axis-properties-do-not-support-late-binding"></a>Le proprietà axis XML non supportano l'associazione tardiva
È stato fatto riferimento a una proprietà axis XML per un oggetto non tipizzato.  
  
 **ID errore:** BC31168  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Assicurarsi che l'oggetto è fortemente tipizzato <xref:System.Xml.Linq.XElement> oggetto prima di fare riferimento alla proprietà axis XML.
