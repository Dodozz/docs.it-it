# <a name="method-does-not-have-a-signature-compatible-with-the-delegate"></a>La firma del metodo non è compatibile con il delegato
È presente un problema di incompatibilità tra le firme del metodo e il delegato a cui che si sta tentando di usare. L'istruzione `Delegate` definisce i tipi di parametro e i tipi restituiti di una classe delegata. Qualsiasi routine con corrispondenti ai parametri di tipi compatibili che restituisce tipi utilizzabili per creare un'istanza di questo tipo di delegato.  
  
 **ID errore**: BC36563
