# <a name="ifthenelse-directives"></a><span data-ttu-id="5170e-102">Direttive #If...Then...#Else</span><span class="sxs-lookup"><span data-stu-id="5170e-102">#If...Then...#Else Directives</span></span>
<span data-ttu-id="5170e-103">Viene compilata in modo condizionale selezionati blocchi di codice Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="5170e-103">Conditionally compiles selected blocks of Visual Basic code.</span></span>  
  
## <a name="syntax"></a><span data-ttu-id="5170e-104">Sintassi</span><span class="sxs-lookup"><span data-stu-id="5170e-104">Syntax</span></span>  
  
```  
#If expression Then  
   statements  
[ #ElseIf expression Then  
   [ statements ]  
...  
#ElseIf expression Then  
   [ statements ] ]  
[ #Else  
   [ statements ] ]  
#End If  
```  
  
## <a name="parts"></a><span data-ttu-id="5170e-105">Parti</span><span class="sxs-lookup"><span data-stu-id="5170e-105">Parts</span></span>  
 `expression`  
 <span data-ttu-id="5170e-106">Obbligatorio per `#If` e `#ElseIf` istruzioni facoltative in un' posizione.</span><span class="sxs-lookup"><span data-stu-id="5170e-106">Required for `#If` and `#ElseIf` statements, optional elsewhere.</span></span> <span data-ttu-id="5170e-107">Qualsiasi espressione, costituito esclusivamente da uno o più costanti del compilatore condizionale, i valori letterali e gli operatori, che restituisce `True` o `False`.</span><span class="sxs-lookup"><span data-stu-id="5170e-107">Any expression, consisting exclusively of one or more conditional compiler constants, literals, and operators, that evaluates to `True` or `False`.</span></span>  
  
 `statements`  
 <span data-ttu-id="5170e-108">Obbligatorio per `#If` istruzione blocco, facoltativo in un' posizione.</span><span class="sxs-lookup"><span data-stu-id="5170e-108">Required for `#If` statement block, optional elsewhere.</span></span> <span data-ttu-id="5170e-109">Righe programma Visual Basic o le direttive del compilatore che vengono compilate se l'espressione associata viene valutata `True`.</span><span class="sxs-lookup"><span data-stu-id="5170e-109">Visual Basic program lines or compiler directives that are compiled if the associated expression evaluates to `True`.</span></span>  
  
 `#End If`

## <a name="remarks"></a><span data-ttu-id="5170e-111">Note</span><span class="sxs-lookup"><span data-stu-id="5170e-111">Remarks</span></span>  
 <span data-ttu-id="5170e-112">Nell'area di, il comportamento dei `#If...Then...#Else` direttive viene visualizzato lo stesso del `If...Then...Else` istruzioni.</span><span class="sxs-lookup"><span data-stu-id="5170e-112">On the surface, the behavior of the `#If...Then...#Else` directives appears the same as that of the `If...Then...Else` statements.</span></span> <span data-ttu-id="5170e-113">Tuttavia, il `#If...Then...#Else` direttive valutano ciò che viene compilato dal compilatore, mentre il `If...Then...Else` le istruzioni vengono valutate le condizioni in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="5170e-113">However, the `#If...Then...#Else` directives evaluate what is compiled by the compiler, whereas the `If...Then...Else` statements evaluate conditions at run time.</span></span>  
  
 <span data-ttu-id="5170e-114">Compilazione condizionale viene in genere utilizzata per la compilazione del programma stesso per le diverse piattaforme.</span><span class="sxs-lookup"><span data-stu-id="5170e-114">Conditional compilation is typically used to compile the same program for different platforms.</span></span> <span data-ttu-id="5170e-115">Viene inoltre usato per impedire il debug del codice venga visualizzato in un file eseguibile.</span><span class="sxs-lookup"><span data-stu-id="5170e-115">It is also used to prevent debugging code from appearing in an executable file.</span></span> <span data-ttu-id="5170e-116">Codice escluso durante la compilazione condizionale viene omesso completamente dal file eseguibile finale, in modo che non ha alcun effetto sulla dimensione o delle prestazioni.</span><span class="sxs-lookup"><span data-stu-id="5170e-116">Code excluded during conditional compilation is completely omitted from the final executable file, so it has no effect on size or performance.</span></span>  
  
 <span data-ttu-id="5170e-117">Indipendentemente dal risultato di qualsiasi valutazione, tutte le espressioni vengono valutate usando `Option Compare Binary`.</span><span class="sxs-lookup"><span data-stu-id="5170e-117">Regardless of the outcome of any evaluation, all expressions are evaluated using `Option Compare Binary`.</span></span> <span data-ttu-id="5170e-118">Il `Option Compare` istruzione non ha effetto sulle espressioni nelle `#If` e `#ElseIf` istruzioni.</span><span class="sxs-lookup"><span data-stu-id="5170e-118">The `Option Compare` statement does not affect expressions in `#If` and `#ElseIf` statements.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="5170e-119">Nessun formato a riga singola del `#If`, `#Else`, `#ElseIf`, e `#End If` direttive esiste.</span><span class="sxs-lookup"><span data-stu-id="5170e-119">No single-line form of the `#If`, `#Else`, `#ElseIf`, and `#End If` directives exists.</span></span> <span data-ttu-id="5170e-120">Nessun altro codice può apparire sulla stessa riga come una delle direttive.</span><span class="sxs-lookup"><span data-stu-id="5170e-120">No other code can appear on the same line as any of the directives.</span></span> 

<span data-ttu-id="5170e-121">Le istruzioni all'interno di un blocco di compilazione condizionale devono includere istruzioni logiche completate.</span><span class="sxs-lookup"><span data-stu-id="5170e-121">The statements within a conditional compilation block must be complete logical statements.</span></span> <span data-ttu-id="5170e-122">Ad esempio, non è possibile compilare in modo condizionale solo gli attributi di una funzione, ma è possibile dichiarare in modo condizionale la funzione insieme ai relativi attributi:</span><span class="sxs-lookup"><span data-stu-id="5170e-122">For example, you cannot conditionally compile only the attributes of a function, but you can conditionally declare the function along with its attributes:</span></span>

```vb
   #If DEBUG Then
   <WebMethod()>
   Public Function SomeFunction() As String
   #Else
   <WebMethod(CacheDuration:=86400)>
   Public Function SomeFunction() As String
   #End If
```
