# <a name="mycomputerregistry-object"></a>Oggetto My.Computer.Registry
Specifica proprietà e metodi per la modifica del Registro di sistema.  
  
## <a name="remarks"></a>Note  
 Per informazioni sui metodi e le proprietà dell'oggetto `My.Computer.Registry` , vedere <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>.  
  
 Per altre informazioni, vedere [leggere e scrivere nel Registro di sistema](../../../visual-basic/developing-apps/programming/computer-resources/reading-from-and-writing-to-the-registry.md).  
  
> [!NOTE]
>  È inoltre possibile modificare il Registro di sistema usando i metodi del <xref:Microsoft.Win32.Registry?displayProperty=nameWithType> classe.  
  
## <a name="requirements"></a>Requisiti  
 **Spazio dei nomi:** <xref:Microsoft.VisualBasic.MyServices>  
  
 **Classe:** <xref:Microsoft.VisualBasic.MyServices.RegistryProxy> (fornisce l'accesso a <xref:Microsoft.Win32.Registry>)
