# <a name="else-statement-visual-basic"></a>Istruzione Else (Visual Basic)
Introduce un gruppo di istruzioni da eseguire o compilate se nessun altro gruppo di istruzioni condizionali è stato eseguito o compilato.  
  
## <a name="remarks"></a>Note  
 È possibile usare la parola chiave `Else` nei contesti seguenti:  
  
 [Istruzione If...Then...Else](../../../visual-basic/language-reference/statements/if-then-else-statement.md)  
  
 [Istruzione Select...Case](../../../visual-basic/language-reference/statements/select-case-statement.md)  
  
 [#If... Then... #Else (direttiva)](../../../visual-basic/language-reference/directives/if-then-else-directives.md)
