## <a name="lambda-expression-syntax"></a><span data-ttu-id="378e6-122">Sintassi delle espressioni lambda</span><span class="sxs-lookup"><span data-stu-id="378e6-122">Lambda Expression Syntax</span></span>  
 <span data-ttu-id="378e6-123">La sintassi di un'espressione lambda è simile a quello di una funzione standard.</span><span class="sxs-lookup"><span data-stu-id="378e6-123">The syntax of a lambda expression resembles that of a standard function.</span></span> <span data-ttu-id="378e6-124">Le differenze sono i seguenti:</span><span class="sxs-lookup"><span data-stu-id="378e6-124">The differences are as follows:</span></span>

-   <span data-ttu-id="378e6-130">Il corpo di un'espressione lambda a riga singola deve essere un'espressione, non un'istruzione.</span><span class="sxs-lookup"><span data-stu-id="378e6-130">The body of a single-line lambda expression must be an expression, not a statement.</span></span> <span data-ttu-id="378e6-131">Il corpo può essere costituito da una chiamata a una routine di funzione, ma non una chiamata a una routine sub.</span><span class="sxs-lookup"><span data-stu-id="378e6-131">The body can consist of a call to a function procedure, but not a call to a sub procedure.</span></span>

-   <span data-ttu-id="378e6-133">I parametri Paramarray e Optional non sono consentiti.</span><span class="sxs-lookup"><span data-stu-id="378e6-133">Optional and Paramarray parameters are not permitted.</span></span>  
  
-   <span data-ttu-id="378e6-134">I parametri generici non sono consentiti.</span><span class="sxs-lookup"><span data-stu-id="378e6-134">Generic parameters are not permitted.</span></span>
