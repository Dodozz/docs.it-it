## <a name="implementing-validations-in-the-domain-model-layer"></a><span data-ttu-id="c6773-118">Implementazione di convalide nel livello del modello di dominio</span><span class="sxs-lookup"><span data-stu-id="c6773-118">Implementing validations in the domain model layer</span></span>

### <a name="validating-conditions-and-throwing-exceptions"></a><span data-ttu-id="c6773-122">Convalida delle condizioni e generazione delle eccezioni</span><span class="sxs-lookup"><span data-stu-id="c6773-122">Validating conditions and throwing exceptions</span></span>
