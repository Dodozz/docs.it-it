# <a name="azure-serverless-platform"></a>Piattaforma senza server di Azure

La piattaforma senza server di Azure include funzioni di Azure, App per la logica e griglia di eventi. Questi servizi funzionano insieme e connettersi con numerose altre risorse. La piattaforma senza server funziona con tutti gli elementi da un archivio di database di analitica e machine learning/artificiali intelligence.

È anche possibile usare Application Insights, una piattaforma senza server per l'acquisizione di tracce di diagnostica e telemetria. Application Insights sono disponibili per le applicazioni di tutti i tipi (desktop, dispositivi mobili o web), nonché implementazioni senza server.
