4.  <span data-ttu-id="f54df-116">Impostare l'applicazione in modo da consentire la lettura e scrittura di dati da e verso una vasta gamma di codifiche, utilizzando le classi di codifica nello spazio dei nomi <xref:System.Text>.</span><span class="sxs-lookup"><span data-stu-id="f54df-116">Enable your application to read and write data to and from a variety of encodings by using the encoding classes in the <xref:System.Text> namespace.</span></span> <span data-ttu-id="f54df-117">Non utilizzare dati ASCII.</span><span class="sxs-lookup"><span data-stu-id="f54df-117">Do not assume ASCII data.</span></span> <span data-ttu-id="f54df-118">Partire dal presupposto che verranno forniti caratteri internazionali in qualsiasi punto in cui un utente può immettere testo.</span><span class="sxs-lookup"><span data-stu-id="f54df-118">Assume that international characters will be supplied anywhere a user can enter text.</span></span> <span data-ttu-id="f54df-119">L'applicazione deve, ad esempio, accettare i caratteri internazionali in nomi di server, directory, nomi file, nomi utente e URL.</span><span class="sxs-lookup"><span data-stu-id="f54df-119">For example, the application should accept international characters in server names, directories, file names, user names, and URLs.</span></span>  
  
5.  <span data-ttu-id="f54df-120">Quando si utilizza la classe <xref:System.Text.UTF8Encoding>, per motivi di sicurezza utilizzare la funzionalità di rilevamento degli errori offerta da questa classe.</span><span class="sxs-lookup"><span data-stu-id="f54df-120">When using the <xref:System.Text.UTF8Encoding> class, for security reasons, use the error detection feature offered by this class.</span></span> <span data-ttu-id="f54df-121">Per attivare la funzionalità di rilevamento degli errori, l'applicazione crea un'istanza della classe utilizzando il costruttore che accetta un parametro `throwOnInvalidBytes` e imposta il valore di questo parametro su `true`.</span><span class="sxs-lookup"><span data-stu-id="f54df-121">To turn on the error detection feature, the application creates an instance of the class using the constructor that takes a `throwOnInvalidBytes` parameter and sets the value of this parameter to `true`.</span></span>  
  
6.  <span data-ttu-id="f54df-122">Quando possibile, gestire le stringhe come unità intere e non come serie di caratteri singoli.</span><span class="sxs-lookup"><span data-stu-id="f54df-122">Whenever possible, handle strings as entire strings instead of as a series of individual characters.</span></span> <span data-ttu-id="f54df-123">Questo metodo di gestione è particolarmente importante quando si ordinano o si cercano sottostringhe.</span><span class="sxs-lookup"><span data-stu-id="f54df-123">This is especially important when sorting or searching for substrings.</span></span> <span data-ttu-id="f54df-124">In questo modo non si verificheranno problemi associati all'analisi dei caratteri combinati.</span><span class="sxs-lookup"><span data-stu-id="f54df-124">This will prevent problems associated with parsing combined characters.</span></span>  
  
7.  <span data-ttu-id="f54df-125">Visualizzare il testo utilizzando le classi fornite dallo spazio dei nomi <xref:System.Drawing>.</span><span class="sxs-lookup"><span data-stu-id="f54df-125">Display text using the classes provided by the <xref:System.Drawing> namespace.</span></span>  
  
8.  <span data-ttu-id="f54df-126">Per garantire l'uniformità tra i sistemi operativi, non consentire l'override di <xref:System.Globalization.CultureInfo> da parte delle impostazioni utente.</span><span class="sxs-lookup"><span data-stu-id="f54df-126">For consistency across operating systems, do not allow user settings to override <xref:System.Globalization.CultureInfo>.</span></span> <span data-ttu-id="f54df-127">Utilizzare il costruttore di `CultureInfo` che accetta un parametro `useUserOverride` e impostarlo su `false`.</span><span class="sxs-lookup"><span data-stu-id="f54df-127">Use the `CultureInfo` constructor that accepts a `useUserOverride` parameter and set it to `false`.</span></span>

10. <span data-ttu-id="f54df-129">Se una decisione relativa alla sicurezza è basata sul risultato di un confronto di stringhe o di un'operazione di modifica delle lettere maiuscole e minuscole, eseguire un'operazione indipendente dalle impostazioni cultura.</span><span class="sxs-lookup"><span data-stu-id="f54df-129">If a security decision is based on the result of a string comparison or case change operation, have the application perform a culture-insensitive operation.</span></span> <span data-ttu-id="f54df-130">In questo modo è possibile verificare che il valore di `CultureInfo.CurrentCulture` non influisca sul risultato.</span><span class="sxs-lookup"><span data-stu-id="f54df-130">This practice ensures that the result is not affected by the value of `CultureInfo.CurrentCulture`.</span></span> <span data-ttu-id="f54df-131">Per un esempio in cui viene illustrato il modo in cui i confronti tra stringhe dipendenti dalle impostazioni cultura possono comportare risultati non coerenti, vedere la sezione "Confronti tra stringhe che usano le impostazioni cultura correnti" in [Procedure consigliate per l'uso delle stringhe](../../../docs/standard/base-types/best-practices-strings.md).</span><span class="sxs-lookup"><span data-stu-id="f54df-131">See the "String Comparisons that Use the Current Culture" section of [Best Practices for Using Strings](../../../docs/standard/base-types/best-practices-strings.md) for an example that demonstrates how culture-sensitive string comparisons can produce inconsistent results.</span></span>  
  
## <a name="localization-best-practices"></a><span data-ttu-id="f54df-132">Suggerimenti per la localizzazione</span><span class="sxs-lookup"><span data-stu-id="f54df-132">Localization Best Practices</span></span>  
  
1.  <span data-ttu-id="f54df-133">Spostare tutte le risorse localizzabili in DLL distinte di sole risorse.</span><span class="sxs-lookup"><span data-stu-id="f54df-133">Move all localizable resources to separate resource-only DLLs.</span></span> <span data-ttu-id="f54df-134">Le risorse localizzabili includono gli elementi dell'interfaccia utente quali stringhe, messaggi di errore, finestre di dialogo, menu e risorse di oggetti incorporati.</span><span class="sxs-lookup"><span data-stu-id="f54df-134">Localizable resources include user interface elements, such as strings, error messages, dialog boxes, menus, and embedded object resources.</span></span>

9. <span data-ttu-id="f54df-147">Usare Visual Studio per creare finestre di dialogo Windows Form, in modo che possano essere localizzate tramite l'[Editor risorse di Windows Form (Winres.exe)](../../../docs/framework/tools/winres-exe-windows-forms-resource-editor.md).</span><span class="sxs-lookup"><span data-stu-id="f54df-147">Use Visual Studio to create Windows Forms dialog boxes, so that they can be localized using the [Windows Forms Resource Editor (Winres.exe)](../../../docs/framework/tools/winres-exe-windows-forms-resource-editor.md).</span></span> <span data-ttu-id="f54df-148">Non codificare manualmente le finestre di dialogo di Windows Form.</span><span class="sxs-lookup"><span data-stu-id="f54df-148">Do not code Windows Forms dialog boxes by hand.</span></span>  
  
10. <span data-ttu-id="f54df-149">Assegnare la localizzazione (traduzione) a professionisti.</span><span class="sxs-lookup"><span data-stu-id="f54df-149">Arrange for professional localization (translation).</span></span>
