# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Modernizza le app .NET esistenti per le applicazioni ottimizzato per il Cloud

> Visione artificiale: Modernizzare le applicazioni esistenti di .NET Framework ottimizzato per il Cloud le applicazioni per migliorare concretamente la flessibilità di distribuzione, in modo da velocizzare la distribuzione e ridurre i costi di distribuzione dell'applicazione.

Per sfruttare i vantaggi del cloud e le nuove tecnologie come i contenitori, è consigliabile modernizzare almeno parzialmente le applicazioni .NET esistenti. In definitiva, la modernizzazione delle applicazioni aziendali consentirà una riduzione del costo totale di proprietà.

La modernizzazione parziale di un'app non significa necessariamente una migrazione completa e rielaborazione. È possibile modernizzare inizialmente le applicazioni esistenti con un'operazione di modernizzazione semplice ma importante. È possibile mantenere il codice corrente, scritto in versioni di .NET Framework esistenti, con tutte le dipendenze di Windows e IIS. Figura 4-1 illustra come ottimizzato per il Cloud le app vengono posizionati in modelli di maturità modernizzazione delle applicazioni Azure.
