# <a name="type-providers"></a><span data-ttu-id="3c98c-103">Provider di tipi</span><span class="sxs-lookup"><span data-stu-id="3c98c-103">Type Providers</span></span>

<span data-ttu-id="3c98c-104">Un provider di tipi F# è un componente che fornisce tipi, proprietà e metodi da utilizzare in un programma.</span><span class="sxs-lookup"><span data-stu-id="3c98c-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="3c98c-105">Provider di tipi generano i cosiddetti **i tipi forniti**, che vengono generati dal F# del compilatore e sono basati su un'origine dati esterna.</span><span class="sxs-lookup"><span data-stu-id="3c98c-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="3c98c-106">Ad esempio, un F# Provider di tipi per SQL può generare tipi che rappresentano le tabelle e colonne in un database relazionale.</span><span class="sxs-lookup"><span data-stu-id="3c98c-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="3c98c-107">In effetti, questo è ciò che il [SQLProvider](https://fsprojects.github.io/SQLProvider/) provider di tipo.</span><span class="sxs-lookup"><span data-stu-id="3c98c-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="3c98c-108">Forniti che tipi dipendono dai parametri di input per un Provider di tipi.</span><span class="sxs-lookup"><span data-stu-id="3c98c-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="3c98c-109">Input di questo tipo può essere un'origine dati di esempio (ad esempio un file di schema JSON), un URL che punta direttamente a un servizio esterno o a una stringa di connessione a un'origine dati.</span><span class="sxs-lookup"><span data-stu-id="3c98c-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="3c98c-110">Un Provider di tipi possono anche garantire che i gruppi di tipi siano espandibili solamente su richiesta. vale a dire, vengono espansi se i tipi vengono effettivamente fatto riferimento dal programma.</span><span class="sxs-lookup"><span data-stu-id="3c98c-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="3c98c-111">Questo consente di utilizzare l'integrazione diretta su richiesta di spazi di informazioni su larga scala fortemente tipizzati, come i mercati di dati online.</span><span class="sxs-lookup"><span data-stu-id="3c98c-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="3c98c-112">Provider di tipi propria e cancellati</span><span class="sxs-lookup"><span data-stu-id="3c98c-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="3c98c-113">Provider di tipi sono disponibili due tipi: Propria e cancellati.</span><span class="sxs-lookup"><span data-stu-id="3c98c-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="3c98c-114">Provider di tipi propria producono i tipi che possono essere scritti come tipi .NET nell'assembly in cui essi vengono generati.</span><span class="sxs-lookup"><span data-stu-id="3c98c-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="3c98c-115">In questo modo possono essere usati dal codice in altri assembly.</span><span class="sxs-lookup"><span data-stu-id="3c98c-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="3c98c-116">Ciò significa che la rappresentazione in forma tipizzata dell'origine dati deve essere in genere che è possibile rappresentare con i tipi .NET.</span><span class="sxs-lookup"><span data-stu-id="3c98c-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="3c98c-117">Provider di tipi cancellazione produrre tipi che possono essere utilizzati solo nell'assembly o nel progetto da che vengono generati.</span><span class="sxs-lookup"><span data-stu-id="3c98c-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="3c98c-118">I tipi sono temporanei; vale a dire, essi non vengono scritti in un assembly e non può essere utilizzati dal codice in altri assembly.</span><span class="sxs-lookup"><span data-stu-id="3c98c-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="3c98c-119">Possono contenere *ritardato* membri, consentendo ai tipi di uso fornito da uno spazio di informazioni potenzialmente infinito.</span><span class="sxs-lookup"><span data-stu-id="3c98c-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="3c98c-120">Sono utili per l'uso di un piccolo subset di un'origine dei dati di grandi dimensioni e interconnesse.</span><span class="sxs-lookup"><span data-stu-id="3c98c-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
