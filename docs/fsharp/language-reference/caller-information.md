## <a name="example"></a><span data-ttu-id="2756a-120">Esempio</span><span class="sxs-lookup"><span data-stu-id="2756a-120">Example</span></span>

<span data-ttu-id="2756a-121">Nell'esempio seguente viene illustrato come è possibile usare questi attributi per tracciare un chiamante.</span><span class="sxs-lookup"><span data-stu-id="2756a-121">The following example shows how you might use these attributes to trace a caller.</span></span>

```fsharp
open System.Diagnostics
open System.Runtime.CompilerServices

type Tracer() =
    member __.DoTrace(message: string,
                      [<CallerMemberName>] ?memberName: string,
                      [<CallerFilePath>] ?path: string,
                      [<CallerLineNumber>] ?line: int) =
        Trace.WriteLine(sprintf "Message: %s" message)
        match (memberName, path, line) with
        | Some m, Some p, Some l ->
            Trace.WriteLine(sprintf "Member name: %s" m)
            Trace.WriteLine(sprintf "Source file path: %s" p)
            Trace.WriteLine(sprintf "Source line number: %d" l)
        | _,_,_ -> ()
```
