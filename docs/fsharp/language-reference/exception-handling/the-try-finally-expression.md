    expression2

<span data-ttu-id="d7b37-108">Il tipo della *expression2* non viene utilizzato il valore dell'espressione intera; il tipo restituito quando non si verifica un'eccezione è l'ultimo valore nella *expression1*.</span><span class="sxs-lookup"><span data-stu-id="d7b37-108">The type of *expression2* does not contribute to the value of the whole expression; the type returned when an exception does not occur is the last value in *expression1*.</span></span> <span data-ttu-id="d7b37-109">Quando si verifica un'eccezione, viene restituito alcun valore e trasferisce il flusso di controllo per il gestore di eccezioni corrispondente successivo lo stack di chiamate.</span><span class="sxs-lookup"><span data-stu-id="d7b37-109">When an exception does occur, no value is returned and the flow of control transfers to the next matching exception handler up the call stack.</span></span> <span data-ttu-id="d7b37-110">Se non viene trovato alcun gestore di eccezioni, il programma termina.</span><span class="sxs-lookup"><span data-stu-id="d7b37-110">If no exception handler is found, the program terminates.</span></span> <span data-ttu-id="d7b37-111">Prima viene eseguito il codice in un gestore corrispondente o il programma termina, il codice nel `finally` ramo viene eseguito.</span><span class="sxs-lookup"><span data-stu-id="d7b37-111">Before the code in a matching handler is executed or the program terminates, the code in the `finally` branch is executed.</span></span>
