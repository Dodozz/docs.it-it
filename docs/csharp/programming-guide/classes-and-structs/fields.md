<span data-ttu-id="0298d-103">Un *campo* è una variabile di qualsiasi tipo che viene dichiarata direttamente in una [classe](../../../csharp/language-reference/keywords/class.md) o [struct](../../../csharp/language-reference/keywords/struct.md).</span><span class="sxs-lookup"><span data-stu-id="0298d-103">A *field* is a variable of any type that is declared directly in a [class](../../../csharp/language-reference/keywords/class.md) or [struct](../../../csharp/language-reference/keywords/struct.md).</span></span> <span data-ttu-id="0298d-104">I campi sono *membri* del rispettivo tipo contenitore.</span><span class="sxs-lookup"><span data-stu-id="0298d-104">Fields are *members* of their containing type.</span></span>  
  
 <span data-ttu-id="0298d-105">Una classe o struct può includere campi di istanza, campi statici o entrambi.</span><span class="sxs-lookup"><span data-stu-id="0298d-105">A class or struct may have instance fields or static fields or both.</span></span> <span data-ttu-id="0298d-106">I campi di istanza sono specifici di un'istanza di tipo.</span><span class="sxs-lookup"><span data-stu-id="0298d-106">Instance fields are specific to an instance of a type.</span></span> <span data-ttu-id="0298d-107">Se si ha una classe T con un campo di istanza F, è possibile creare due oggetti di tipo T e modificare il valore di F in ciascun oggetto senza modificare il valore nell'altro oggetto.</span><span class="sxs-lookup"><span data-stu-id="0298d-107">If you have a class T, with an instance field F, you can create two objects of type T, and modify the value of F in each object without affecting the value in the other object.</span></span> <span data-ttu-id="0298d-108">Al contrario, un campo statico appartiene alla classe stessa ed è condiviso tra tutte le istanze della classe.</span><span class="sxs-lookup"><span data-stu-id="0298d-108">By contrast, a static field belongs to the class itself, and is shared among all instances of that class.</span></span> <span data-ttu-id="0298d-109">Le modifiche apportate dall'istanza A saranno visibili immediatamente alle istanze B e C se accedono al campo.</span><span class="sxs-lookup"><span data-stu-id="0298d-109">Changes made from instance A will be visibly immediately to instances B and C if they access the field.</span></span>
