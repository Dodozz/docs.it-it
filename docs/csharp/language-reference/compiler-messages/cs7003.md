# <a name="compiler-error-cs7003"></a>Errore del compilatore CS7003

Uso imprevisto di un nome generico non associato

Questo errore si verifica se si usa un tipo generico che richiede un tipo di parametro senza passare nomi di tipo di parametro generico tra parentesi acute. Questo uso pu√≤ essere una dichiarazione di variabile o una creazione dell'istanza di un oggetto.

## <a name="to-correct-this-error"></a>Per correggere l'errore  

Specificare un nome di tipo di parametro tra parentesi acute quando si usa un tipo generico.

## <a name="example"></a>Esempio

L'esempio seguente genera l'errore CS7003:

```csharp
// CS7003.cs  
    class Program
    {
        static void Main(string[] args)
        {
            var myVar1 = new MyGenericClass<>();  //CS7003

            MyGenericClass<> var2;                //CS7003
        }
    }

    public class MyGenericClass<T> { }
```
