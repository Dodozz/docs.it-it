# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="26596-102">Avviso del compilatore (livello 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="26596-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="26596-103">'membro1' nasconde il membro ereditato 'membro2'.</span><span class="sxs-lookup"><span data-stu-id="26596-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="26596-104">Se questo comportamento è intenzionale, usare la parola chiave new.</span><span class="sxs-lookup"><span data-stu-id="26596-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="26596-105">Una variabile è stata dichiarata con lo stesso nome di una variabile in una classe di base.</span><span class="sxs-lookup"><span data-stu-id="26596-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="26596-106">Non è stata tuttavia usata la parola chiave [new](../../../csharp/language-reference/keywords/new.md).</span><span class="sxs-lookup"><span data-stu-id="26596-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="26596-107">Questo avviso informa che è necessario usare la parola chiave **new**. La variabile è dichiarata come se la parola chiave **new** sia stata usata nella dichiarazione.</span><span class="sxs-lookup"><span data-stu-id="26596-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="26596-108">L'esempio seguente genera l'errore CS0108:</span><span class="sxs-lookup"><span data-stu-id="26596-108">The following sample generates CS0108:</span></span>  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
