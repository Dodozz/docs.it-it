## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="a6c7c-106">Esempio di conversione ambigua</span><span class="sxs-lookup"><span data-stu-id="a6c7c-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="a6c7c-107">Questo errore può verificarsi quando non viene eseguita alcuna conversione o se le conversioni implicite disponibili generano una situazione ambigua.</span><span class="sxs-lookup"><span data-stu-id="a6c7c-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="a6c7c-108">L'esempio seguente genera l'errore CS0151:</span><span class="sxs-lookup"><span data-stu-id="a6c7c-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```
