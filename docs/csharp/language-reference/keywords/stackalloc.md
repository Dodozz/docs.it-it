## <a name="remarks"></a><span data-ttu-id="ec075-104">Note</span><span class="sxs-lookup"><span data-stu-id="ec075-104">Remarks</span></span>

<span data-ttu-id="ec075-105">La parola chiave è valida solo per gli inizializzatori di variabili locali.</span><span class="sxs-lookup"><span data-stu-id="ec075-105">The keyword is valid only in local variable initializers.</span></span> <span data-ttu-id="ec075-106">Il codice seguente causa errori di compilazione.</span><span class="sxs-lookup"><span data-stu-id="ec075-106">The following code causes compiler errors.</span></span>

```csharp
int* block;
// The following assignment statement causes compiler errors. You
// can use stackalloc only when declaring and initializing a local
// variable.
block = stackalloc int[100];
```

<span data-ttu-id="ec075-107">A partire da C# 7.3, è possibile usare la sintassi dell'inizializzatore di matrice per le matrici `stackalloc`.</span><span class="sxs-lookup"><span data-stu-id="ec075-107">Beginning with C# 7.3, you can use array initializer syntax for `stackalloc` arrays.</span></span> <span data-ttu-id="ec075-108">Tutte le dichiarazioni seguenti dichiarano una matrice con tre elementi i cui valori sono i numeri interi `1`, `2` e `3`:</span><span class="sxs-lookup"><span data-stu-id="ec075-108">All the following declarations declare an array with three elements whose values are the integers `1`, `2`, and `3`:</span></span>

```csharp

int* first = stackalloc int[3] { 1, 2, 3 };
int* second = stackalloc int[] { 1, 2, 3 };
int* third = stackalloc[] { 1, 2, 3 };
```

<span data-ttu-id="ec075-109">Poiché vengono usati tipi puntatore, `stackalloc` richiede un contesto [unsafe](unsafe.md).</span><span class="sxs-lookup"><span data-stu-id="ec075-109">Because pointer types are involved, `stackalloc` requires an [unsafe](unsafe.md) context.</span></span> <span data-ttu-id="ec075-110">Per altre informazioni, vedere [Codice unsafe e puntatori](../../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="ec075-110">For more information, see [Unsafe Code and Pointers](../../programming-guide/unsafe-code-pointers/index.md).</span></span>

<span data-ttu-id="ec075-111">La parola chiave `stackalloc` è come [_alloca](/cpp/c-runtime-library/reference/alloca) nella libreria di runtime del linguaggio C.</span><span class="sxs-lookup"><span data-stu-id="ec075-111">`stackalloc` is like [_alloca](/cpp/c-runtime-library/reference/alloca) in the C run-time library.</span></span>
