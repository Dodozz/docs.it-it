# <a name="operator-keywords-c-reference"></a>Parole chiave per operatori (Riferimenti per C#)

Consentono di eseguire diversi tipi di operazioni, come creare oggetti, controllare il tipo di runtime di un oggetto, ottenere la dimensione di un tipo e altre azioni. In questa sezione vengono presentate le parole chiave seguenti:

- [as](as.md) Converte un oggetto in un tipo compatibile.

- [await](await.md) Sospende un metodo [async](async.md) finché non viene completata un'attività attesa.

- [is](is.md) Controlla il tipo di runtime di un oggetto, oppure (a partire da C# 7.0) controlla un'espressione rispetto a un criterio.

- [new](new.md)

  - [Operatore new](new-operator.md) Crea oggetti.

  - [Modificatore new](new-modifier.md) Nasconde un membro ereditato.

  - [Vincolo new](new-constraint.md) Qualifica un parametro di tipo.

- [nameof](nameof.md) Ottiene il nome di stringa semplice (non qualificato) di una variabile, un tipo o un membro.

- [sizeof](sizeof.md) Ottiene le dimensioni di un tipo non gestito.  

- [typeof](typeof.md) Ottiene l'oggetto <xref:System.Type?displayProperty=nameWithType> per un tipo.  

- [true](true.md)  

  - [Operatore true](true-false-operators.md) Restituisce il valore [bool](bool.md) `true` per indicare che un operando è indubbiamente true.

  - [Valore letterale true](true-literal.md) Rappresenta il valore [bool](bool.md) `true`.

- [false](false.md)  

  - [Operatore false](true-false-operators.md) Restituisce il valore [bool](bool.md) `true` per indicare che un operando è indubbiamente false.

  - [Valore letterale false](false-literal.md) Rappresenta il valore [bool](bool.md) `false`.

- [stackalloc](stackalloc.md) Alloca un blocco di memoria nello stack.  

Le parole chiave seguenti, che possono essere usate come operatori e istruzioni, sono descritte nella sezione [Istruzioni](statement-keywords.md):

- [checked](checked.md) Specifica il contesto verificato.  

- [unchecked](unchecked.md) Specifica il contesto non verificato.
