# <a name="out-c-reference"></a>out (Riferimenti per C#)

È possibile usare la parola chiave `out` in due contesti:

- Come [modificatore di parametro](out-parameter-modifier.md), che consente di passare un argomento a un metodo per riferimento e non per valore.

- In [dichiarazioni di parametri di tipo generico](out-generic-modifier.md) per interfacce e delegati e indica che un parametro tipo è covariante.
