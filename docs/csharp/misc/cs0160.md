<span data-ttu-id="842c1-104">Una serie di istruzioni [catch](../../csharp/language-reference/keywords/try-catch.md) deve essere in ordine di derivazione decrescente.</span><span class="sxs-lookup"><span data-stu-id="842c1-104">A series of [catch](../../csharp/language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="842c1-105">Ad esempio, gli oggetti più derivati devono essere visualizzati per prima.</span><span class="sxs-lookup"><span data-stu-id="842c1-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="842c1-106">Per altre informazioni, vedere [eccezioni e gestione delle eccezioni](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="842c1-106">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="842c1-107">L'esempio seguente genera l'errore CS0160:</span><span class="sxs-lookup"><span data-stu-id="842c1-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
