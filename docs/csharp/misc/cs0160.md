 <span data-ttu-id="4ab13-104">Una serie di istruzioni **catch** deve essere in ordine di derivazione decrescente.</span><span class="sxs-lookup"><span data-stu-id="4ab13-104">A series of **catch** statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="4ab13-105">Ad esempio, gli oggetti più derivati devono essere visualizzati per prima.</span><span class="sxs-lookup"><span data-stu-id="4ab13-105">For example, the most derived objects must appear first.</span></span>  
  
 <span data-ttu-id="4ab13-106">Per altre informazioni, vedere [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) e [eccezioni e gestione delle eccezioni](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="4ab13-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="4ab13-107">L'esempio seguente genera l'errore CS0160:</span><span class="sxs-lookup"><span data-stu-id="4ab13-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
