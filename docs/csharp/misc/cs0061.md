 <span data-ttu-id="27cd6-104">Un costrutto [pubblico](../../csharp/language-reference/keywords/public.md) deve restituire un oggetto accessibile pubblicamente.</span><span class="sxs-lookup"><span data-stu-id="27cd6-104">A [public](../../csharp/language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="27cd6-107">L'esempio seguente genera l'errore CS0061.</span><span class="sxs-lookup"><span data-stu-id="27cd6-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
