## <a name="example"></a><span data-ttu-id="d6ddc-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="d6ddc-110">Example</span></span>  
 <span data-ttu-id="d6ddc-111">Il codice seguente genera l'errore CS0736 perché `Program.testMethod` è dichiarato come statico:</span><span class="sxs-lookup"><span data-stu-id="d6ddc-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>

```csharp  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
