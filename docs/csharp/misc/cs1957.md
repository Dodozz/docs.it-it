## <a name="example"></a><span data-ttu-id="c3aa8-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="c3aa8-109">Example</span></span>  
 <span data-ttu-id="c3aa8-110">Il codice seguente genera l'errore CS1957:</span><span class="sxs-lookup"><span data-stu-id="c3aa8-110">The following code generates CS1957:</span></span>  
  
```csharp  
// cs1957.cs  
class Base<T, S>  
{  
    public virtual string Test(out T x) // CS1957  
    {  
        x = default(T);  
        return "Base.Test";  
    }  
    public virtual void Test(ref S x) { }  
}  
  
class Derived : Base<int, int>  
{  
    public override string Test(out int x)  
    {  
        x = 0;  
        return "Derived.Test";  
    }  
  
    static int Main()  
    {  
        int x;  
        if (new Derived().Test(out x) == "Derived.Test")  
            return 0;  
        return 1;  
    }  
}  
```
