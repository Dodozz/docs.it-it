# <a name="compiler-error-cs1662"></a><span data-ttu-id="01b54-102">Errore del compilatore CS1662</span><span class="sxs-lookup"><span data-stu-id="01b54-102">Compiler Error CS1662</span></span>

<span data-ttu-id="01b54-103">Non è possibile convertire il blocco di metodi anonimi nel tipo delegato 'delegate type' perché alcuni dei tipi restituiti nel blocco non sono convertibili in modo implicito nel tipo restituito del delegato</span><span class="sxs-lookup"><span data-stu-id="01b54-103">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>

<span data-ttu-id="01b54-104">Questo errore si verifica se l'istruzione restituita del blocco di metodi anonimi ha un tipo che non è convertibile in modo implicito nel tipo restituito del delegato.</span><span class="sxs-lookup"><span data-stu-id="01b54-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

<span data-ttu-id="01b54-105">L'esempio seguente genera l'errore CS1662:</span><span class="sxs-lookup"><span data-stu-id="01b54-105">The following sample generates CS1662:</span></span>

```csharp
// CS1662.cs

delegate int MyDelegate(int i);

class C
{

  public static void Main()
  {
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662
     // Try this instead:
     // MyDelegate d = delegate(int i) { return (int)1.0; };
  }
}
```
