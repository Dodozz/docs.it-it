# <a name="compiler-warning-level-1-cs0684"></a>Avviso del compilatore (livello 1) CS0684
Interfaccia 'interface' contrassegnata con 'CoClassAttribute' non contrassegnata con 'ComImportAttribute'  
  
 Se si specifica **CoClassAttribute** in un'interfaccia, Ã¨ necessario specificare anche **ComImportAttribute**.  
  
 L'esempio seguente genera l'errore CS0684:  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
