# <a name="compiler-error-cs1606"></a>Errore del compilatore CS1606
Firma dell'assembly non riuscita. È possibile che l'output non sia firmato: reason  
  
 L'assembly è stato generato, ma quando il compilatore ha tentato di completarne la firma, si è verificato un errore.
