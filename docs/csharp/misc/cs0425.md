## <a name="example"></a><span data-ttu-id="9227d-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="9227d-107">Example</span></span>

```csharp

{  
}  
  
class C2  
{  
}  
  
interface IBase  
{  
    void F<ItemType>(ItemType item) where ItemType : C1;  
}

{  
    public void F<ItemType>(ItemType item) where ItemType : C2  // CS0425

    }

{  
    public static void Main()  
    {  
    }  
}  
```
