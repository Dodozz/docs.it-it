## <a name="example"></a><span data-ttu-id="f524c-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="f524c-108">Example</span></span>  
 <span data-ttu-id="f524c-109">L'esempio seguente genera l'errore CS3018.</span><span class="sxs-lookup"><span data-stu-id="f524c-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
