## <a name="example"></a><span data-ttu-id="c7f22-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="c7f22-108">Example</span></span>  
 <span data-ttu-id="c7f22-109">L'esempio seguente genera l'errore CS0131.</span><span class="sxs-lookup"><span data-stu-id="c7f22-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="c7f22-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="c7f22-110">Example</span></span>  
 <span data-ttu-id="c7f22-111">Questo errore può verificarsi anche se si tenta di eseguire operazioni aritmetiche nella parte sinistra dell'operatore di assegnazione, come nell'esempio seguente.</span><span class="sxs-lookup"><span data-stu-id="c7f22-111">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>

```csharp  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)

        return 1;  
    }

```
