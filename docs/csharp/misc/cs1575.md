# <a name="compiler-error-cs1575"></a><span data-ttu-id="34d02-102">Errore del compilatore CS1575</span><span class="sxs-lookup"><span data-stu-id="34d02-102">Compiler Error CS1575</span></span>
<span data-ttu-id="34d02-103">In un'espressione stackalloc occorre specificare [] dopo il tipo</span><span class="sxs-lookup"><span data-stu-id="34d02-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="34d02-104">Le dimensioni dell'allocazione richiesta, con [stackalloc](../language-reference/operators/stackalloc.md), devono essere specificate tra parentesi quadre.</span><span class="sxs-lookup"><span data-stu-id="34d02-104">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="34d02-105">L'esempio seguente genera l'errore CS1575:</span><span class="sxs-lookup"><span data-stu-id="34d02-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
