# <a name="compiler-error-cs1575"></a>Errore del compilatore CS1575
In un'espressione stackalloc occorre specificare [] dopo il tipo  
  
 Le dimensioni dell'allocazione richiesta, con [stackalloc](../language-reference/operators/stackalloc.md), devono essere specificate tra parentesi quadre.  
  
 L'esempio seguente genera l'errore CS1575:  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
