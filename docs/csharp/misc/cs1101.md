# <a name="compiler-error-cs1101"></a><span data-ttu-id="a1a44-102">Errore del compilatore CS1101</span><span class="sxs-lookup"><span data-stu-id="a1a44-102">Compiler Error CS1101</span></span>
<span data-ttu-id="a1a44-103">Non è possibile usare il modificatore di parametro 'ref' con 'this'.</span><span class="sxs-lookup"><span data-stu-id="a1a44-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>

## <a name="example"></a><span data-ttu-id="a1a44-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="a1a44-106">Example</span></span>  
 <span data-ttu-id="a1a44-107">L'esempio seguente genera l'errore CS1101:</span><span class="sxs-lookup"><span data-stu-id="a1a44-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
