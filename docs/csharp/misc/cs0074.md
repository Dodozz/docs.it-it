# <a name="compiler-error-cs0074"></a>Errore del compilatore CS0074
'event': l'evento astratto non può avere un inizializzatore  
  
 Se un [evento](../../csharp/language-reference/keywords/event.md) è contrassegnato come **abstract**, non può essere inizializzato. Per ulteriori informazioni, vedi [Eventi](../../csharp/programming-guide/events/index.md).  
  
 L'esempio seguente genera l'errore CS0074:  
  
```csharp  
// CS0074.cs  
delegate void D();  
  
abstract class Test  
{  
   public abstract event D e = null;   // CS0074  
   // try the following line instead  
   // public abstract event D e;  
  
   public static void Main()  
   {  
   }  
}  
```
