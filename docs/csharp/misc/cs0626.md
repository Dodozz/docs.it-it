# <a name="compiler-warning-level-1-cs0626"></a><span data-ttu-id="226c6-102">Avviso del compilatore (livello 1) CS0626</span><span class="sxs-lookup"><span data-stu-id="226c6-102">Compiler Warning (level 1) CS0626</span></span>

<span data-ttu-id="226c6-103">Il metodo, la funzione di accesso o l'operatore 'method' è contrassegnato come esterno e non include attributi.</span><span class="sxs-lookup"><span data-stu-id="226c6-103">Method, operator, or accessor 'method' is marked external and has no attributes on it.</span></span> <span data-ttu-id="226c6-104">È consigliabile aggiungere un attributo DllImport per specificare l'implementazione esterna.</span><span class="sxs-lookup"><span data-stu-id="226c6-104">Consider adding a DllImport attribute to specify the external implementation.</span></span>
  
 <span data-ttu-id="226c6-105">Un metodo contrassegnato come `extern` deve essere contrassegnato anche con un attributo, ad esempio l'attributo [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) .</span><span class="sxs-lookup"><span data-stu-id="226c6-105">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>
  
 <span data-ttu-id="226c6-106">L'attributo specifica dove viene implementato il metodo.</span><span class="sxs-lookup"><span data-stu-id="226c6-106">The attribute specifies where the method is implemented.</span></span> <span data-ttu-id="226c6-107">Il programma avrà bisogno di queste informazioni in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="226c6-107">At run time, the program will need this information.</span></span>  
  
 <span data-ttu-id="226c6-108">L'esempio seguente genera l'errore CS0626:</span><span class="sxs-lookup"><span data-stu-id="226c6-108">The following sample generates CS0626:</span></span>  
  
```csharp
// CS0626.cs  
// compile with: /warnaserror  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   static extern public void M(); // CS0626  
   // try the following line  
   // [DllImport("mydll.dll")] static extern public void M();  
  
   public static void Main()  
   {  
   }  
}  
```
