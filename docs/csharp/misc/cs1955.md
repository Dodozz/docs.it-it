## <a name="example"></a><span data-ttu-id="99520-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="99520-108">Example</span></span>

<span data-ttu-id="99520-109">Il codice seguente genera l'errore CS1955 poiché il codice sta tentando di richiamare un campo e una proprietà usando la [operatore di chiamata `()` ](../language-reference/operators/member-access-operators.md#invocation-operator-).</span><span class="sxs-lookup"><span data-stu-id="99520-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="99520-110">Non è possibile chiamare un campo o una proprietà.</span><span class="sxs-lookup"><span data-stu-id="99520-110">You cannot call a field or a property.</span></span> <span data-ttu-id="99520-111">Usare la [operatore di accesso al membro `.` ](../language-reference/operators/member-access-operators.md#member-access-operator-) per accedere al valore archiviato.</span><span class="sxs-lookup"><span data-stu-id="99520-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
