<span data-ttu-id="b6ad5-103">Un membro statico 'function' non può essere contrassegnato come override, virtual o abstract</span><span class="sxs-lookup"><span data-stu-id="b6ad5-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="b6ad5-104">Una dichiarazione di metodo in cui viene usata la parola chiave `override`, **virtual**o **abstract** non può usare anche la parola chiave **static** .</span><span class="sxs-lookup"><span data-stu-id="b6ad5-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="b6ad5-105">Per altre informazioni, vedere [Metodi](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="b6ad5-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="b6ad5-106">L'esempio seguente genera l'errore CS0112:</span><span class="sxs-lookup"><span data-stu-id="b6ad5-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
