<span data-ttu-id="de60f-103">L'interfaccia ereditata 'interface1' causa un ciclo nella gerarchia delle interfacce di 'interface2'</span><span class="sxs-lookup"><span data-stu-id="de60f-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="de60f-104">L'elenco di ereditarietà di un ['interfaccia](../../csharp/language-reference/keywords/interface.md) include un riferimento diretto o indiretto a se stessa.</span><span class="sxs-lookup"><span data-stu-id="de60f-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="de60f-105">Un'interfaccia non può ereditare da se stessa.</span><span class="sxs-lookup"><span data-stu-id="de60f-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="de60f-106">L'esempio seguente genera l'errore CS0529:</span><span class="sxs-lookup"><span data-stu-id="de60f-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
