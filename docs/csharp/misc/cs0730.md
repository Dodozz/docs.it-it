## <a name="example"></a><span data-ttu-id="c61d5-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="c61d5-105">Example</span></span>  
 <span data-ttu-id="c61d5-106">L'esempio seguente genera l'errore CS0730.</span><span class="sxs-lookup"><span data-stu-id="c61d5-106">The following sample generates CS0730.</span></span> <span data-ttu-id="c61d5-107">È costituito da due file di origine.</span><span class="sxs-lookup"><span data-stu-id="c61d5-107">It consists of two source files.</span></span> <span data-ttu-id="c61d5-108">Compilare prima di tutto il file di libreria `CS0730a.cs`, quindi compilare il file `CS0730.cs` che fa riferimento al file di libreria.</span><span class="sxs-lookup"><span data-stu-id="c61d5-108">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
