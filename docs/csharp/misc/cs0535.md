## <a name="example"></a><span data-ttu-id="354a0-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="354a0-106">Example</span></span>  
 <span data-ttu-id="354a0-107">L'esempio seguente genera l'errore CS0535.</span><span class="sxs-lookup"><span data-stu-id="354a0-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
