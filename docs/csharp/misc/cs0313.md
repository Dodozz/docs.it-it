## <a name="to-correct-this-error"></a><span data-ttu-id="cfe3a-108">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="cfe3a-108">To correct this error</span></span>

  
## <a name="example"></a><span data-ttu-id="cfe3a-111">Esempio</span><span class="sxs-lookup"><span data-stu-id="cfe3a-111">Example</span></span>  
 <span data-ttu-id="cfe3a-112">Il codice seguente genera l'errore CS0313:</span><span class="sxs-lookup"><span data-stu-id="cfe3a-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
