# <a name="compiler-error-cs0025"></a>Errore del compilatore CS0025
Impossibile trovare il file di libreria standard 'file'  
  
 Non è stato trovato un file che è richiesto dal compilatore. Verificare che il percorso sia corretto e che il file esista.  
  
 Se il file è un file di sistema di Visual Studio, si potrebbe essere necessario ripristinare l'installazione di Visual Studio o reinstallare completamente l'applicazione.
