# <a name="compiler-warning-level-3-cs0665"></a>Avviso del compilatore (livello 3) CS0665
L'assegnazione nell'espressione condizionale Ã¨ sempre costante. Si intendeva utilizzare == invece di = ?  
  
 Un'espressione condizionale ha usato [= operator](../language-reference/operators/assignment-operator.md) e non [== operator](../language-reference/operators/equality-operators.md#equality-operator-).
  
 L'esempio seguente genera l'errore CS0665:  
  
```csharp  
// CS0665.cs  
// compile with: /W:3  
class Test  
{  
   public static void Main()  
   {  
      bool i = false;  
  
      if (i = true)   // CS0665  
      // try the following line instead  
      // if (i == true)  
      {  
      }  
  
      System.Console.WriteLine(i);  
   }  
}  
```
