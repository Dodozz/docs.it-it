# <a name="compiler-error-cs0077"></a><span data-ttu-id="dbe97-102">Errore del compilatore CS0077</span><span class="sxs-lookup"><span data-stu-id="dbe97-102">Compiler Error CS0077</span></span>
<span data-ttu-id="dbe97-103">L'operatore as deve essere usato con un tipo riferimento o con un tipo nullable ('int' è un tipo valore non nullable).</span><span class="sxs-lookup"><span data-stu-id="dbe97-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="dbe97-104">All'operatore [as](../../csharp/language-reference/keywords/as.md) operatore è stato passato un [tipo valore](../../csharp/language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="dbe97-104">The [as](../../csharp/language-reference/keywords/as.md) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="dbe97-105">Poiché `as` può restituire [null](../../csharp/language-reference/keywords/null.md), quindi è possibile passare a tale operatore solo [tipi riferimento](../../csharp/language-reference/keywords/reference-types.md) o un tipo nullable.</span><span class="sxs-lookup"><span data-stu-id="dbe97-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="dbe97-106">Per altre informazioni sui tipi nullable, vedere [tipi Nullable](../../csharp/programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="dbe97-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="dbe97-107">L'esempio seguente genera l'errore CS0077:</span><span class="sxs-lookup"><span data-stu-id="dbe97-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
