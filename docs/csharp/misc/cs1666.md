<span data-ttu-id="b760f-103">Impossibile utilizzare buffer a dimensione fissa contenuti in espressioni unfixed.</span><span class="sxs-lookup"><span data-stu-id="b760f-103">You cannot use fixed size buffers contained in unfixed expressions.</span></span> <span data-ttu-id="b760f-104">Provare a utilizzare l'istruzione fixed.</span><span class="sxs-lookup"><span data-stu-id="b760f-104">Try using the fixed statement.</span></span>  
  
 <span data-ttu-id="b760f-105">Questo errore si verifica se si usa il buffer a dimensione fissa in un'espressione che coinvolge una classe che a sua volta non è fixed.</span><span class="sxs-lookup"><span data-stu-id="b760f-105">This error occurs if you use the fixed sized buffer in an expression involving a class that is not itself fixed.</span></span> <span data-ttu-id="b760f-106">In fase di esecuzione la classe unfixed può essere spostata per ottimizzare l'accesso alla memoria, causando errori durante l'uso del buffer a dimensione fissa.</span><span class="sxs-lookup"><span data-stu-id="b760f-106">The runtime is free to move the unfixed class around to optimize memory access, which could lead to errors when using the fixed sized buffer.</span></span> <span data-ttu-id="b760f-107">Per evitare questo errore, usare la parola chiave `fixed` nell'istruzione.</span><span class="sxs-lookup"><span data-stu-id="b760f-107">To avoid this error, use the `fixed` keyword on the statement.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b760f-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="b760f-108">Example</span></span>  
 <span data-ttu-id="b760f-109">L'esempio seguente genera l'errore CS1666.</span><span class="sxs-lookup"><span data-stu-id="b760f-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);   
   }   
}  
```
