## <a name="example"></a><span data-ttu-id="99ede-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="99ede-106">Example</span></span>  
 <span data-ttu-id="99ede-107">L'esempio seguente genera l'errore CS0559:</span><span class="sxs-lookup"><span data-stu-id="99ede-107">The following sample generates CS0559:</span></span>

```csharp  
{  
   public static implicit operator int(iii x)  
   {  
      return 0;  
   }  
  
   public static implicit operator iii(int x)  
   {  
      return null;  
   }  
  
   public static int operator ++(int aa)   // CS0559

   // public static iii operator ++(iii aa)

      return (iii)0;  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="99ede-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="99ede-108">Example</span></span>  
 <span data-ttu-id="99ede-109">L'esempio seguente genera l'errore CS0559.</span><span class="sxs-lookup"><span data-stu-id="99ede-109">The following sample generates CS0559.</span></span>  
  
```csharp  
// CS0559_b.cs  
// compile with: /target:library  
public struct S  
{  
   public static S operator ++(S? s) { return new S(); }   // CS0559

}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
}  
```
