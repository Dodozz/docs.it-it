## <a name="example"></a><span data-ttu-id="bb883-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="bb883-105">Example</span></span>  
 <span data-ttu-id="bb883-106">L'esempio seguente genera l'errore CS1529:</span><span class="sxs-lookup"><span data-stu-id="bb883-106">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs

{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition

}

```
