<span data-ttu-id="9318f-103">Riferimento ambiguo nell'attributo cref: 'Method Name1'.</span><span class="sxs-lookup"><span data-stu-id="9318f-103">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="9318f-104">Verrà usato 'Method Name2', ma è anche possibile che corrisponda ad altri overload, tra cui 'Method Name3'.</span><span class="sxs-lookup"><span data-stu-id="9318f-104">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="9318f-105">Non è stato possibile risolvere un riferimento in un commento della documentazione XML nel codice.</span><span class="sxs-lookup"><span data-stu-id="9318f-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="9318f-106">Questa situazione può verificarsi se il metodo viene sottoposto a overload o se vengono trovati due identificatori diversi con lo stesso nome.</span><span class="sxs-lookup"><span data-stu-id="9318f-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="9318f-107">Per risolvere il problema, usare un nome completo per evitare ambiguità nel riferimento oppure racchiudere l'overload specifico tra parentesi.</span><span class="sxs-lookup"><span data-stu-id="9318f-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="9318f-108">L'esempio seguente genera l'errore CS0419.</span><span class="sxs-lookup"><span data-stu-id="9318f-108">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
