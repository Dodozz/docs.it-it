# <a name="compiler-error-cs0206"></a>Errore del compilatore CS0206
Una proprietà o un indicizzatore non può essere passato come parametro out o ref  
  
 Non sono disponibili oggetti [property](../../csharp/programming-guide/classes-and-structs/properties.md) da passare come parametro [ref](../../csharp/language-reference/keywords/ref.md) o [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) . Per altre informazioni, vedere [Passaggio di parametri](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS0206:  
  
```csharp  
// CS0206.cs  
public class MyClass  
{  
    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set  
        {  
        }  
    }  
  
    public static void MyMeth(ref int i)  
    // public static void MyMeth(int i)  
    {  
    }  
  
    public static void Main()  
    {  
        MyMeth(ref P);   // CS0206  
        // try the following line instead  
        // MyMeth(P);   // CS0206  
    }  
}  
```
