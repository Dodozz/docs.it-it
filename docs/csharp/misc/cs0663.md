# <a name="compiler-error-cs0663"></a>Errore del compilatore CS0663

 I metodi che differiscono solo per l'uso del [nelle](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) e [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) su un parametro non sono consentiti.  
  
 L'esempio seguente genera l'errore CS0663:  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
