# <a name="compiler-warning-level-3-cs0660"></a><span data-ttu-id="abc12-102">Avviso del compilatore (livello 3) CS0660</span><span class="sxs-lookup"><span data-stu-id="abc12-102">Compiler Warning (level 3) CS0660</span></span>

<span data-ttu-id="abc12-103">'class' definisce l'operatore == o l'operatore != ma non esegue l'override di Object.Equals(object o)</span><span class="sxs-lookup"><span data-stu-id="abc12-103">'class' defines operator == or operator != but does not override Object.Equals(object o)</span></span>  
  
<span data-ttu-id="abc12-104">Il compilatore ha rilevato l'uguaglianza definito dall'utente o operatore di disuguaglianza, ma non l'override per il <xref:System.Object.Equals%2A?displayProperty=nameWithType> (metodo).</span><span class="sxs-lookup"><span data-stu-id="abc12-104">The compiler detected the user-defined equality or inequality operator, but no override for the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="abc12-105">Un operatore di uguaglianza o disuguaglianza definito dall'utente implica che si vuole anche eseguire l'override di <xref:System.Object.Equals%2A> (metodo).</span><span class="sxs-lookup"><span data-stu-id="abc12-105">A user-defined equality or inequality operator implies that you also want to override the <xref:System.Object.Equals%2A> method.</span></span> <span data-ttu-id="abc12-106">Per altre informazioni, vedere [Procedura: Definire l'uguaglianza di valori per un tipo](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span><span class="sxs-lookup"><span data-stu-id="abc12-106">For more information, see [How to: Define Value Equality for a Type](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span></span>
  
<span data-ttu-id="abc12-107">L'esempio seguente genera l'errore CS0660:</span><span class="sxs-lookup"><span data-stu-id="abc12-107">The following sample generates CS0660:</span></span>  

```csharp
// CS0660.cs  
// compile with: /W:3 /warnaserror  
class Test   // CS0660  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   // uncomment the Equals function to resolve  
   // public override bool Equals(object o)  
   // {  
   //    return true;  
   // }  
  
   public override int GetHashCode()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
