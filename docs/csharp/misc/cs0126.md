# <a name="compiler-error-cs0126"></a>Errore del compilatore CS0126
È necessario un oggetto di un tipo convertibile in 'type'  
  
 Un'istruzione return è presente, ma non restituisce un valore del tipo previsto. Per ulteriori informazioni, vedi [Proprietà](../../csharp/programming-guide/classes-and-structs/properties.md).  
  
 L'esempio seguente genera l'errore CS0126:  
  
```csharp  
// CS0126.cs  
public class a  
{  
   public int i  
   {  
      set  
      {  
      }  
      get  
      {  
         return;   // CS0126, specify a value to return  
      }  
   }  
}  
```
