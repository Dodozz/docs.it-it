# <a name="compiler-error-cs1639"></a>Errore del compilatore CS1639
La firma della classe wrapper 'signature' della coclasse gestita per l'interfaccia 'interface' non è valida per il nome della classe  
  
 Questo errore si verifica se i metadati per l'interfaccia non sono validi. Non dovrebbe mai verificarsi se l'interfaccia è stata generata utilizzando C# o un linguaggio .NET supportato. Contattare il fornitore dell'interfaccia o verificare che l'assembly dell'interfaccia sia stato generato correttamente.
