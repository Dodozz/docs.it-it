 <span data-ttu-id="cc6c1-108">L'esempio seguente genera l'errore CS3010:</span><span class="sxs-lookup"><span data-stu-id="cc6c1-108">The following example generates CS3010:</span></span>

```csharp  
    [CLSCompliant(false)]

{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
