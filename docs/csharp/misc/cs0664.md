Usare un suffisso 'suffix' per creare un valore letterale di questo tipo</span><span class="sxs-lookup"><span data-stu-id="7e656-103">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="7e656-104">Non è stato possibile completare un'assegnazione; usare un suffisso per correggere l'istruzione.</span><span class="sxs-lookup"><span data-stu-id="7e656-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="7e656-105">La documentazione per ogni tipo identifica il suffisso corrispondente per il tipo.</span><span class="sxs-lookup"><span data-stu-id="7e656-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="7e656-106">Per altre informazioni sulle conversioni, vedere [cast e conversioni tipo](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="7e656-106">For more information on conversions, see [Casting and Type Conversions](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="7e656-107">L'esempio seguente genera l'errore CS0664:</span><span class="sxs-lookup"><span data-stu-id="7e656-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
