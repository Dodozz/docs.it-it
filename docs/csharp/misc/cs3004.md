# <a name="compiler-warning-level-1-cs3004"></a>Avviso del compilatore (livello 1) CS3004
I caratteri Unicode misti e scomposti non sono conformi a CLS  
  
 Per la conformità a Common Language Specification (CLS), negli identificatori [public](../../csharp/language-reference/keywords/public.md), [protected](../../csharp/language-reference/keywords/protected.md)o `protected internal` sono consentiti solo caratteri UNICODE composti. Per altre informazioni sulla conformità a CLS, vedere [indipendenza del linguaggio e componenti indipendenti dal linguaggio](../../standard/language-independence.md).
