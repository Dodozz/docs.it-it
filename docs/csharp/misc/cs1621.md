## <a name="example"></a><span data-ttu-id="e90a2-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="e90a2-105">Example</span></span>  
 <span data-ttu-id="e90a2-106">L'esempio seguente genera l'errore CS1621:</span><span class="sxs-lookup"><span data-stu-id="e90a2-106">The following sample generates CS1621:</span></span>

delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()

        MyDelegate d = delegate  
        {  
            yield return this; // CS1621

        };

    {  
    }  
}
