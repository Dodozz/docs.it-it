# <a name="compiler-error-cs0157"></a>Errore del compilatore CS0157
Il controllo non pu√≤ lasciare il corpo di una clausola finally  
  
 Tutte le istruzioni in una clausola [finally](../../csharp/language-reference/keywords/try-catch-finally.md) devono essere eseguite. Per altre informazioni, vedere [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) e [eccezioni e gestione delle eccezioni](../../csharp/programming-guide/exceptions/index.md).  
  
 L'esempio seguente genera l'errore CS0157:  
  
```csharp  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
