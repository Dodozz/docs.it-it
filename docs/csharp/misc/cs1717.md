<span data-ttu-id="d2a1a-103">Assegnazione fatta alla stessa variabile. Si intendeva assegnare qualcos'altro?</span><span class="sxs-lookup"><span data-stu-id="d2a1a-103">Assignment made to same variable; did you mean to assign something else?</span></span>

 <span data-ttu-id="d2a1a-105">L'avviso può essere generato da diversi errori comuni:</span><span class="sxs-lookup"><span data-stu-id="d2a1a-105">Several common mistakes can generate this warning:</span></span>

-   <span data-ttu-id="d2a1a-108">Errori di digitazione.</span><span class="sxs-lookup"><span data-stu-id="d2a1a-108">Mistyping.</span></span> <span data-ttu-id="d2a1a-109">È probabile che si volesse scrivere `a = b`.</span><span class="sxs-lookup"><span data-stu-id="d2a1a-109">You probably meant to say `a = b`.</span></span>
