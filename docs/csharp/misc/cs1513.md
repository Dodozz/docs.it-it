 Il compilatore prevedeva una parentesi graffa chiusa (`}`) non trovata.  
  
 L'esempio seguente genera l'errore CS1513:  
  
```csharp  
// CS1513  
namespace y   // CS1513, no close curly brace  
{  
   class x  
   {  
      public static void Main()  
      {  
      }  
   }  
```
