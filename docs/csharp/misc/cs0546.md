## <a name="example"></a><span data-ttu-id="03f5c-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="03f5c-110">Example</span></span>  
 <span data-ttu-id="03f5c-111">L'esempio seguente genera l'errore CS0546 perché la classe base non dichiara una funzione di accesso set per la proprietà.</span><span class="sxs-lookup"><span data-stu-id="03f5c-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
