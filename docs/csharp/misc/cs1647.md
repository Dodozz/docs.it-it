# <a name="compiler-error-cs1647"></a>Errore del compilatore CS1647
Espressione troppo lunga o complessa per essere compilata accanto a 'code'  
  
 Si è verificato un overflow dello stack nel compilatore che sta elaborando il codice. Per risolvere questo errore, semplificare il codice. Se il codice è valido, consultare la Guida nel [Community degli sviluppatori](https://developercommunity.visualstudio.com/spaces/61/index.html) sito oppure [StackOverflow](https://stackoverflow.com/).
