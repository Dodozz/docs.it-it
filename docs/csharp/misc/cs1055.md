# <a name="compiler-error-cs1055"></a><span data-ttu-id="fb1b5-102">Errore del compilatore CS1055</span><span class="sxs-lookup"><span data-stu-id="fb1b5-102">Compiler Error CS1055</span></span>
<span data-ttu-id="fb1b5-103">È prevista una funzione di accesso add o remove</span><span class="sxs-lookup"><span data-stu-id="fb1b5-103">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="fb1b5-104">Se [event](../../csharp/language-reference/keywords/event.md) non è dichiarato come campo, deve definire le funzioni di accesso **add** e **remove** .</span><span class="sxs-lookup"><span data-stu-id="fb1b5-104">If your [event](../../csharp/language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="fb1b5-105">L'esempio seguente genera l'errore CS1055:</span><span class="sxs-lookup"><span data-stu-id="fb1b5-105">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
