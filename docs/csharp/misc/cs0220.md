# <a name="compiler-error-cs0220"></a>Errore del compilatore CS0220
Operazione in overflow in fase di compilazione in modalità checked  
  
 In modalità [checked](../../csharp/language-reference/keywords/checked.md), ovvero l'impostazione predefinita, è stata rilevata un'operazione che ha provocato una perdita di dati. Correggere gli input dell'assegnazione oppure usare [unchecked](../../csharp/language-reference/keywords/unchecked.md) per risolvere l'errore. Per altre informazioni, vedere [Checked e Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).  
  
 L'esempio seguente genera l'errore CS0220:  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
