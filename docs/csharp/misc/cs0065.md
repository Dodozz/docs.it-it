# <a name="compiler-error-cs0065"></a><span data-ttu-id="f477d-102">Errore del compilatore CS0065</span><span class="sxs-lookup"><span data-stu-id="f477d-102">Compiler Error CS0065</span></span>
<span data-ttu-id="f477d-103">'event': la proprietà dell'evento deve avere entrambe le funzioni di accesso add e remove</span><span class="sxs-lookup"><span data-stu-id="f477d-103">'event' : event property must have both add and remove accessors</span></span>  
  
 <span data-ttu-id="f477d-104">Un evento che non è un campo deve avere entrambi i metodi di accesso.</span><span class="sxs-lookup"><span data-stu-id="f477d-104">An event that is not a field must have both access methods.</span></span>  
  
 <span data-ttu-id="f477d-105">L'esempio seguente genera l'errore CS0065:</span><span class="sxs-lookup"><span data-stu-id="f477d-105">The following sample generates CS0065:</span></span>  
  
```csharp  
// CS0065.cs  
using System;  
public delegate void Eventhandler(object sender, int e);  
public class MyClass  
{  
   public event EventHandler Click   // CS0065,  
   {  
      // to fix, uncomment the add and remove definitions  
      /*  
      add  
      {  
         Click += value;  
      }  
      remove  
      {  
         Click -= value;  
      }  
      */  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
