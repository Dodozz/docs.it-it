# <a name="compiler-error-cs0619"></a><span data-ttu-id="001d7-102">Errore del compilatore CS0619</span><span class="sxs-lookup"><span data-stu-id="001d7-102">Compiler Error CS0619</span></span>
<span data-ttu-id="001d7-103">'member' è obsoleto: 'text'</span><span class="sxs-lookup"><span data-stu-id="001d7-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="001d7-104">Un membro di classe è stato contrassegnato con [obsoleto] ((.. /.. / csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) dell'attributo, tale che verrà generato un errore quando si fa riferimento il membro della classe.</span><span class="sxs-lookup"><span data-stu-id="001d7-104">A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="001d7-105">L'esempio seguente genera l'errore CS0619:</span><span class="sxs-lookup"><span data-stu-id="001d7-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
