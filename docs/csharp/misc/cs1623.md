# <a name="compiler-error-cs1623"></a><span data-ttu-id="1264a-102">Errore del compilatore CS1623</span><span class="sxs-lookup"><span data-stu-id="1264a-102">Compiler Error CS1623</span></span>
<span data-ttu-id="1264a-103">Gli iteratori non possono avere parametri out o ref</span><span class="sxs-lookup"><span data-stu-id="1264a-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="1264a-104">Questo errore si verifica se un metodo iteratore accetta un `in`, `ref`, o `out` parametro.</span><span class="sxs-lookup"><span data-stu-id="1264a-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="1264a-105">Per evitare questo errore, rimuovere il `in`, `ref`, o `out` parola chiave di firma del metodo.</span><span class="sxs-lookup"><span data-stu-id="1264a-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1264a-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="1264a-106">Example</span></span>  
 <span data-ttu-id="1264a-107">L'esempio seguente genera l'errore CS1623:</span><span class="sxs-lookup"><span data-stu-id="1264a-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
