# <a name="compiler-error-cs0682"></a>Errore del compilatore CS0682
'type1' non può implementare 'type2'perché non è supportato dal linguaggio  
  
 Questo errore si verifica quando si tenta di implementare un'interfaccia scritta in un altro linguaggio e il compilatore non supporta tale interfaccia.
