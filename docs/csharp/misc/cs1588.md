# <a name="compiler-error-cs1588"></a>Errore del compilatore CS1588
Impossibile determinare la directory di Common Language Runtime: 'reason'  
  
 Il compilatore non può recuperare la directory in cui è installato Common Language Runtime di .NET Framework. Questo errore indica che Common Language Runtime non è installato correttamente.
