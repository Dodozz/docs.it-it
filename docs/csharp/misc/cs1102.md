## <a name="example"></a><span data-ttu-id="1c8ed-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="1c8ed-108">Example</span></span>  
 <span data-ttu-id="1c8ed-109">L'esempio seguente genera l'errore CS1102:</span><span class="sxs-lookup"><span data-stu-id="1c8ed-109">The following example generates CS1102:</span></span>  
  
```csharp  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
