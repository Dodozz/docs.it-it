## <a name="example"></a><span data-ttu-id="ff289-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="ff289-109">Example</span></span>  
 <span data-ttu-id="ff289-110">Il codice seguente genera l'errore CS1914 perché l'inizializzatore tenta di inizializzare il campo `TestClass.Number` , che è `static`:</span><span class="sxs-lookup"><span data-stu-id="ff289-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }      
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
