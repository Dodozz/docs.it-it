 <span data-ttu-id="06ca4-107">L'esempio seguente genera l'errore CS0438:</span><span class="sxs-lookup"><span data-stu-id="06ca4-107">The following example generates CS0438:</span></span>

 <span data-ttu-id="06ca4-109">Compilare quindi il file:</span><span class="sxs-lookup"><span data-stu-id="06ca4-109">Then compile this file:</span></span>

   public class A { }  
}

 <span data-ttu-id="06ca4-110">Infine compilare il file:</span><span class="sxs-lookup"><span data-stu-id="06ca4-110">And then compile this file:</span></span>

{

      Console.WriteLine(typeof(Util.A));   // CS0438  
   }
