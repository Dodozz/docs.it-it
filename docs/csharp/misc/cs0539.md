<span data-ttu-id="4b375-103">'member' nella dichiarazione esplicita dell'interfaccia non è un membro dell'interfaccia</span><span class="sxs-lookup"><span data-stu-id="4b375-103">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="4b375-104">Si è provato a dichiarare esplicitamente un membro di [interfaccia](../../csharp/language-reference/keywords/interface.md) che non esiste.</span><span class="sxs-lookup"><span data-stu-id="4b375-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="4b375-105">Eliminare la dichiarazione o modificarla in modo che faccia riferimento a un membro di interfaccia valido.</span><span class="sxs-lookup"><span data-stu-id="4b375-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="4b375-106">L'esempio seguente genera l'errore CS0539:</span><span class="sxs-lookup"><span data-stu-id="4b375-106">The following sample generates CS0539:</span></span>

```csharp  
   {  
      void m();  
   }

      void I.x()   // CS0539

      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
