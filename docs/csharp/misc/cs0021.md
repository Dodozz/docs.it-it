## <a name="example"></a><span data-ttu-id="40947-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="40947-108">Example</span></span>  
 <span data-ttu-id="40947-109">Il seguente file viene compilato in un file DLL, con l'attributo `DefaultMember` impostato come commento, in modo da generare l'errore.</span><span class="sxs-lookup"><span data-stu-id="40947-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```
