# <a name="compiler-error-cs0076"></a>Errore del compilatore CS0076
Il nome dell'enumeratore 'value__' è riservato e non può essere usato  
  
 Un elemento di [enumerazione](../../csharp/language-reference/keywords/enum.md) non può avere un identificatore denominato **value__**.
