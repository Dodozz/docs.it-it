# <a name="compiler-error-cs1593"></a><span data-ttu-id="a6ed8-102">Errore del compilatore CS1593</span><span class="sxs-lookup"><span data-stu-id="a6ed8-102">Compiler Error CS1593</span></span>
<span data-ttu-id="a6ed8-103">Il delegato 'del' non accetta argomenti 'number'</span><span class="sxs-lookup"><span data-stu-id="a6ed8-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="a6ed8-104">Il numero di argomenti passati alla chiamata di un [delegato](../../csharp/language-reference/keywords/delegate.md) non corrisponde al numero di parametri presenti nella dichiarazione del delegato.</span><span class="sxs-lookup"><span data-stu-id="a6ed8-104">The number of arguments passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="a6ed8-105">L'esempio seguente genera l'errore CS1593:</span><span class="sxs-lookup"><span data-stu-id="a6ed8-105">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
