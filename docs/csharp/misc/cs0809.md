# <a name="compiler-warning-level-1-cs0809"></a>Avviso del compilatore (livello 1) CS0809

Il membro obsoleto 'memberA' esegue l'override del membro non obsoleto 'memberB'.

In genere, un membro contrassegnato come obsoleto non deve eseguire l'override di un membro che non Ã¨ contrassegnato come obsoleto. Questo avviso viene generato in Visual Studio 2008 ma non in Visual Studio 2005.

## <a name="to-correct-this-error"></a>Per correggere l'errore

Rimuovere l'attributo `Obsolete` dal membro che esegue l'override oppure aggiungerlo al membro di classe base.

## <a name="example"></a>Esempio

```csharp
// CS0809.cs
public class Base
{
    public virtual void Test1()
    {
    }
}
public class C : Base
{
    [System.Obsolete()]
    public override void Test1() // CS0809
    {
    }
}
```
