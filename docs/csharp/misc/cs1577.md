# <a name="compiler-error-cs1577"></a>Errore del compilatore CS1577
Generazione assembly non riuscita - motivo  
  
 La parte della compilazione relativa alla generazione dell'assembly non è riuscita dopo che il compilatore C# ha compilato il codice sorgente. Per altre informazioni, vedere la documentazione sugli errori relativi all'utilità ALink ([Errori e avvisi di Al.exe](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)).
